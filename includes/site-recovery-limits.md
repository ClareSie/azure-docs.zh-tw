| 限制識別碼 | 限制 |
| --- | --- |
| 每個訂用帳戶的保存庫數目 |500 |     
| 每個復原服務保存庫的伺服器數目 |250 |
| 每個復原服務保存庫的保護群組數目 |沒有限制 |
| 每個復原服務保存庫的復原計畫數目 |沒有限制 |
| 每個保護群組的伺服器數目 |沒有限制 |
| 每個復原計畫的伺服器數目 |100 |
