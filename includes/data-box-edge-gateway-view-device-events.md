請在 Azure 入口網站中採取下列步驟，以檢視裝置事件。 

1. 在 Azure 入口網站中，移至您的 Azure Stack Edge/資料箱閘道資源，然後移至 [**監視] > [裝置事件**]。
2. 選取事件並檢視警示詳細資料。 採取適當動作來解決警示條件。

    ![選取事件並檢視詳細資料](media/data-box-edge-gateway-view-device-events/view-device-events.png)
