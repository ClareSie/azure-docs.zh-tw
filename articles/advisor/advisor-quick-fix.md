# <a name="quick-fix-remediation-for-advisor"></a>Advisor 的快速修正補救措施
**快速修正** 可讓您更快速且更輕鬆地進行補救，以在多個資源上提供建議。 它提供大量補救資源的功能，並可協助您更快速地透過針對您的資源進行補救來更快地將訂閱優化。
這項功能僅適用于特定的建議，可透過 Azure 入口網站取得。


## <a name="steps-to-use-quick-fix"></a>使用「快速修正」的步驟

1. 從具有 **快速修正** 標籤的建議清單中，按一下建議。

   :::image type="content" source="./media/quick-fix-1.png" alt-text="{Azure Advisor 的螢幕擷取畫面，其中顯示建議中的快速修正標籤。}":::
   
   *映射中的價格僅供範例之用*

2. 在 [建議詳細資料] 頁面上，您會看到有此建議的資源清單。 選取您要補救的所有資源以取得建議。

   :::image type="content" source="./media/quick-fix-2.png" alt-text="[受影響的資源] 視窗的螢幕擷取畫面，其中已醒目提示清單專案和 [快速修正] 按鈕。":::
   
   *映射中的價格僅供範例之用*

3. 選取資源後，請按一下 [ **快速修正** ] 按鈕以進行大量補救。

   > [!NOTE]
   > 某些列出的資源可能已停用，因為您沒有適當的許可權可以修改它們。
   
   > [!NOTE]
   > 如果有其他的含意，除了建議程式中提及的優點之外，您將會在體驗中傳達，以協助您採取明智的補救決策。
   
4. 您將會收到補救完成的通知。 如果資源清單視圖中有未補救的資源和所選模式的資源，您將會看到錯誤。
