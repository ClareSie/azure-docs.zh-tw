下表列出 Azure 服務匯流排目前支援和不支援的**JAVA Message Service （JMS）** 功能。


| 功能 | 狀態 |
|---|---|
| 佇列   | 支援 |
| 主題   | 支援 |
| 暫存佇列 | 支援 |
| 暫存主題 | 支援 |
| 訊息選取器 | 支援 |
| 佇列瀏覽器 | 支援 |
| 共用的持久訂閱 | 支援|
| 未共用的持久訂閱 | 支援 |
| 共用的非持久訂閱 | 支援 |
| 不共用的非持久性訂閱 | 支援 |
| 分散式交易 | **不支援** |
| 持久終點 | **不支援** |
