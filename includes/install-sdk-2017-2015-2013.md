## <a name="set-up-the-development-environment"></a><a name="setupdevenv"></a>設定開發環境
首先，使用 Visual Studio 和 Azure SDK 設定您的開發環境。

* Visual Studio 2019 包括 Azure SDK。 如果您使用的是 Visual Studio 2019，則開發環境不需要其他設置。
* 若為 Visual Studio 2015，請按一下下列連結以安裝 [Azure SDK for Visual Studio 2015](https://go.microsoft.com/fwlink/?linkid=518003)。
* 若為 Visual Studio 2013，按一下下列連結以安裝 [Azure SDK for Visual Studio 2013](https://go.microsoft.com/fwlink/?LinkID=324322)。

* 如果未安裝 Visual Studio，請使用以下內容使用 Azure SDK 安裝[Visual Studio 2019。](https://www.visualstudio.com/)

> [!NOTE]
> 視您的電腦上已有的 SDK 相依性數目而定，安裝 SDK 可能需要很長的時間 (從數分鐘到半小時以上不等)。
