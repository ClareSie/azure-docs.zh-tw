# <a name="azure-maps-weather-coverage"></a>Azure 地圖天氣覆蓋範圍

本文提供 Azure 地圖[天氣服務的](https://aka.ms/AzureMapsWeatherService)覆蓋範圍資訊。 Azure 地圖天氣資料服務返回詳細資訊，如雷達磁貼、當前天氣狀況、天氣預報和沿途的天氣。

Azure 地圖對於所有國家和地區的資訊和準確性水準不同。

下表提供有關可以從每個國家/地區請求的天氣資訊類型的資訊。

| 符號 | 意義 |
|--------|---------|
|*       |涵蓋沿線天氣、當前狀況、每小時預報、季度預報和每日預報 API |
