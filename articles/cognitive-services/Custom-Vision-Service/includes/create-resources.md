若要使用自訂視覺服務，您必須在 Azure 中建立自訂視覺定型和預測資源。 若要在 Azure 入口網站中這麼做，請在[建立自訂視覺](https://portal.azure.com/?microsoft_azure_marketplace_ItemHideKey=microsoft_azure_cognitiveservices_customvision#create/Microsoft.CognitiveServicesCustomVision)頁面上填寫對話方塊視窗，以建立定型和預測資源。
