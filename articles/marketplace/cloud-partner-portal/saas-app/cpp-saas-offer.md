# <a name="azure-saas-application-offer"></a>Azure SaaS 應用程式供應項目

[雲合作夥伴門戶](https://cloudpartner.azure.com/)不再支援創建和管理 SaaS 應用程式產品/  相反，此功能已移動到 Microsoft[合作夥伴中心門戶](https://partner.microsoft.com/pcv/)。  有關詳細資訊，請參閱[創建新的 SaaS 產品/服務](../../partner-center-portal/create-new-saas-offer.md)。 


## <a name="next-steps"></a>後續步驟

在創建新的 SaaS 應用產品/服務之前，應閱讀[SaaS 應用程式產品/服務發佈指南](../../marketplace-saas-applications-technical-publishing-guide.md)，並查看["產品/服務創建"檢查表](../../partner-center-portal/offer-creation-checklist.md)中列出的合作夥伴中心門戶步驟。
