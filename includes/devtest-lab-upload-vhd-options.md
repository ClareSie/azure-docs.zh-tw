## <a name="prerequisites"></a>Prerequisites 
要完成此教學課程，您必須將 VHD 檔案上傳到與您建立自訂映像相關聯之實驗室的儲存體帳戶中。 選取下列任何連結，上傳 VHD 檔案：

- [使用 AzCopy 命令列公用程式上傳](../articles/lab-services/devtest-lab-upload-vhd-using-azcopy.md)
- [使用 Microsoft Azure 儲存體總管上傳](../articles/lab-services/devtest-lab-upload-vhd-using-storage-explorer.md)
- [使用 PowerShell 主控台上傳](../articles/lab-services/devtest-lab-upload-vhd-using-powershell.md)
