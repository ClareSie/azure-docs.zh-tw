對於靜止資料：

- 對存儲在共用中的資料的訪問受到限制。

    - 訪問共用資料的 SMB 用戶端需要與共享關聯的使用者憑據。 創建共用時定義這些憑據。
    - 創建共用時需要添加訪問共用的 NFS 用戶端的 IP 位址。
