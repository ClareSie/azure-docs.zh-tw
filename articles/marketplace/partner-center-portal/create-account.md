# <a name="create-a-commercial-marketplace-account-in-partner-center"></a>在合作夥伴中心創建商業市場帳戶

要將優惠發佈到[Azure 應用商店](https://azuremarketplace.microsoft.com/)或[Microsoft AppSource,](https://appsource.microsoft.com/)您需要在合作夥伴中心的商業市場程式中創建一個帳戶。 在本文中,我們將介紹如何創建合作夥伴中心帳戶以及如何向商業市場計劃添加新發佈商。

>[!IMPORTANT]
>如果您在[雲合作夥伴門戶 (CPP)](https://cloudpartner.azure.com)中具有已移動到合作夥伴中心的帳戶,則無需創建新帳戶。 有關詳細資訊[,請參閱從 CPP 移動的發行者](#publishers-moving-from-cpp)。

## <a name="before-you-begin"></a>開始之前

要在合作夥伴中心創建帳戶,請確保您具有:

- 代表貴公司簽署法律協議的許可權。
- 您公司的合法業務名稱、位址和主要連絡人(可以是您)。

我們將會在帳戶建立程序期間驗證這項資訊。

建立帳號的方法有兩種:

- 如果您是合作夥伴中心的新客戶,並且沒有 Microsoft 網路帳戶,請使用[合作夥伴中心註冊頁建立帳戶](#create-an-account-using-the-partner-center-enrollment-page)。
- 如果您已註冊 Microsoft 合作夥伴網路,請使用[現有的 Microsoft 合作夥伴中心註冊創建帳戶](#create-an-account-using-existing-microsoft-partner-center-enrollments)直接從合作夥伴中心創建帳戶。

## <a name="create-an-account-using-the-partner-center-enrollment-page"></a>使用合作夥伴中心註冊頁面建立帳戶

如果您是合作夥伴中心的新客戶,並且未註冊到 Microsoft 合作夥伴網路,請使用此方法。 本節介紹如何創建新的合作夥伴中心帳戶,包括:

- [在合作夥伴中心註冊頁面上註冊](#register-on-the-partner-center-enrollment-page)
- [使用工作帳戶登入](#sign-in-with-a-work-account)
- [同意條款和條件](#agree-to-terms-and-conditions)
- [提供發行者設定檔](#provide-your-publisher-profile)

>[!IMPORTANT]
>如果您在[雲合作夥伴門戶 (CPP)](https://cloudpartner.azure.com)中具有已移動到合作夥伴中心的帳戶,則無需創建新帳戶。 有關詳細資訊[,請參閱從 CPP 移動的發行者](#publishers-moving-from-cpp)。

### <a name="register-on-the-partner-center-enrollment-page"></a>在合作夥伴中心註冊頁面上註冊

查看「[**歡迎加入 Microsoft 合作夥伴中心**](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/partnership)」註冊頁面上的資訊,然後註冊帳戶。

### <a name="sign-in-with-a-work-account"></a>使用工作帳戶登入

使用工作帳戶登錄,以便將公司的工作電子郵件帳戶域連結到新的合作夥伴中心帳戶。 通過關聯這些帳戶,公司員工可以使用其工作帳戶使用者名和密碼登錄到合作夥伴中心。

>[!NOTE]
>要檢查公司是否已擁有工作帳戶、如何創建新工作帳戶,以及如何設置多個工作帳戶以與合作夥伴中心一起使用,請造訪[您的公司工作帳戶和合作夥伴中心](./company-work-accounts.md)。

### <a name="agree-to-terms-and-conditions"></a>同意條款和條件

您需要同意在[Microsoft 市場發佈商協定中](https://go.microsoft.com/fwlink/?LinkID=699560)的條款和條件。

### <a name="provide-your-publisher-profile"></a>提供發行者設定檔

您的發行者設定檔包括您的公司名稱和 MPN ID。 如果您尚未加入,則需要加入[Microsoft 合作夥伴網路](https://partner.microsoft.com/commercial)。 加入 Microsoft 合作夥伴網路後,您將獲得 MPN ID。

創建發行者 ID。 您的發佈者 ID 在應用商店和 AppSource 中唯一標識您的公司和產品/服務。

確認發佈商個人資料資訊後,同意條款和條件,並通過選擇 **"接受"並繼續**創建合作夥伴中心帳戶。

>[!IMPORTANT]
>*您必須獲得代表公司行事的授權,才能接受這些條款。*

註冊后,您將進入"商業市場概述"頁面。 **商業市場**顯示在左側導航欄中。 如果選擇 **「設定** > **合作夥伴設定** > **程式」**, 您將看到商業市場已列為已註冊市場。

感謝您在合作夥伴中心創建商業市場帳戶! 有關説明,請參閱[商業市場概述](./commercial-marketplace-overview.md)。 有關新增發行商的説明,請參閱[向商業市場添加新發佈商](#add-new-publishers-to-the-commercial-marketplace-program)。

## <a name="create-an-account-using-existing-microsoft-partner-center-enrollments"></a>使用現有的 Microsoft 合作夥伴中心註冊建立帳戶

如果您已在 Microsoft 合作夥伴中心註冊,請使用此部分創建商業市場帳戶。

有兩種類型的現有註冊可用於設置商業市場帳戶。 選擇適用於您的:

- 如果您有現有的 Microsoft 合作夥伴網路註冊,請使用[「在 Microsoft 合作夥伴網路註冊」](#enrolled-in-the-microsoft-partner-network)創建您的帳戶。
- 如果您有現有的開發人員程式註冊,請使用[註冊開發人員程式](#enrolled-in-a-developer-program)創建您的帳戶。

對於這兩種方法,您將使用以下步驟:

- 使用現有認證登入合作夥伴中心
- 輸入您的發佈個人資料資訊

確保帳戶資訊方便。

>[!IMPORTANT]
>如果您在[雲合作夥伴門戶 (CPP)](https://cloudpartner.azure.com/)中具有已移動到合作夥伴中心的帳戶,則無需創建新帳戶。 有關詳細資訊[,請參閱從 CPP 移動的發行者](#publishers-moving-from-cpp)。

### <a name="enrolled-in-the-microsoft-partner-network"></a>註冊加入微軟合作夥伴網路

如果您已註冊 Microsoft 合作夥伴網路,請使用此部分直接從合作夥伴中心創建您的帳戶。

使用 Microsoft 合作夥伴網路帳戶登入[合作夥伴中心](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/partnership),以便將公司的工作電子郵件帳戶域連結到新的合作夥伴中心帳戶。 通過關聯這些帳戶,您的公司員工可以使用其工作帳戶使用者名和密碼登錄合作夥伴中心。

>[!NOTE]
>要檢查您的公司是否已具有合作夥伴中心工作帳戶、如何建立新的工作帳戶,以及如何設置多個工作帳戶以與合作夥伴中心一起使用,請造訪[您的公司工作帳戶和合作夥伴中心](./company-work-accounts.md)。

登錄后,請使用以下步驟完成帳戶設置。

>[!NOTE]
> 您必須具有**帳戶管理員**或**全域管理員**角色才能登錄到 Microsoft 合作夥伴網路。

1. 從[合作夥伴中心](https://partner.microsoft.com/),選擇 **「設置** > **合作夥伴設置** > **程式** > **開始**」。
2. Microsoft 合作夥伴網路偵測您的訂閱並顯示發行者設定檔頁,以便您輸入公司名稱和 MPN ID。
3. 閱讀[Microsoft 應用商店發佈商協定中](https://go.microsoft.com/fwlink/?LinkID=699560)的條款和條件,然後選擇 **"接受並繼續**完成註冊"。

    >[!IMPORTANT]
    >*您必須獲得代表公司行事的授權,才能接受這些條款。*

註冊后,您將進入"商業市場概述"頁面。 **商業市場**顯示在左側導航欄中。 如果選擇 **「設定** > **合作夥伴設定** > **程式」**, 您將看到商業市場已列為已註冊市場。

感謝您在合作夥伴中心創建帳戶! 有關説明,請參閱[商業市場概述](./commercial-marketplace-overview.md)。 有關新增發行商的説明,請參閱[向商業市場添加新發佈商](#add-new-publishers-to-the-commercial-marketplace-program)。

### <a name="enrolled-in-a-developer-program"></a>註冊了開發人員計劃

如果您已註冊了合作夥伴中心開發人員計劃,請使用此部分直接從合作夥伴中心創建您的帳戶。

>[!NOTE]
>要在您登錄的同一帳戶下註冊商業市場,您需要使用公司帳戶在合作夥伴中心註冊。 如果您使用個人帳戶註冊,您的商業市場註冊將顯示在新帳戶下。
>
>個人帳戶是為獨立工作的單個開發人員。 公司帳戶適用於組織和企業。 公司帳戶允許您使用附加功能提交應用。 在您註冊后,我們需要對公司帳戶進行更大的驗證,以確認您已獲得為公司設置帳戶的授權。 驗證可能需要花費數天至數個星期，且通常會電話連絡您的公司。 這兩種類型的帳戶都允許您提交應用、載入項和服務。 有關詳細資訊,請參閱 [帳號類型、位置和費用](https://docs.microsoft.com/windows/uwp/publish/account-types-locations-and-fees)。

使用 Microsoft[合作夥伴中心帳戶登錄合作夥伴中心](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/partnership),以在同一帳戶下註冊商業市場計畫。 通過使用同一帳戶註冊商業市場,您可以在合作夥伴中心的左側導航功能表中查看現有計劃註冊和商業市場。

登入後,請使用以下步驟完成帳戶設置:

1. 從合作夥伴中心,選擇 **「設置** > **開發人員設定** > **程式** > **入門」,** 然後輸入您的工作電子郵件位址。
2. 在發佈者配置檔頁上完成詳細資訊。
3. 閱讀[Microsoft 應用商店發佈商協定中](https://go.microsoft.com/fwlink/?LinkID=699560)的條款和條件,然後選擇 **"接受並繼續**完成註冊"。

    >[!Important]
    >*您必須獲得代表公司行事的授權,才能接受這些條款。*

註冊后,您將進入"商業市場概述"頁面。 **商業市場**顯示在左側導航欄中。 如果選擇 **「設定** > **開發人員設定** > **程式」**, 您將看到商業市場已列為已註冊市場。

感謝您在合作夥伴中心創建帳戶! 有關説明,請參閱[商業市場概述](./commercial-marketplace-overview.md)。 有關新增發行商的説明,請參閱[向商業市場添加新發佈商](#add-new-publishers-to-the-commercial-marketplace-program)。

## <a name="publishers-moving-from-cpp"></a>從 CPP 轉移的發行者

如果您的帳戶已經從[雲合作夥伴門戶 (CPP)](https://cloudpartner.azure.com)遷移,則無需創建新的合作夥伴中心帳戶。 您應該在登錄到現有 CPP 帳戶後透過電子郵件和橫幅通知中收到指向新合作夥伴中心帳戶的自訂連結。

訪問此自定義鏈接啟用新合作夥伴中心帳戶後,可以通過訪問合作夥伴中心[中的商業市場儀錶板](https://partner.microsoft.com/dashboard/commercial-marketplace/overview)返回到您的帳戶。

發佈協定和公司配置檔資訊將遷移到您的新合作夥伴中心帳戶,以及以前設置的任何帳戶支出配置檔資訊、使用者帳戶和許可權以及與您的 CPP 帳戶關聯的活動優惠。

將您的帳戶資訊從 CPP 移動到合作夥伴中心後,您將不再使用 CPP 進行帳戶更新或管理使用者、許可權和計費。 在有限的時間內,您在合作夥伴中心進行的任何帳戶更新將自動在唯讀 CPP 帳戶中更新,直到 CPP 門戶最終被棄用為止。

## <a name="add-new-publishers-to-the-commercial-marketplace-program"></a>新增商業市場計劃新增新發行者

組織可以有多個發佈商與商業市場帳戶關聯。 現有用戶可以在登錄到合作夥伴中心後通過選擇**帳戶設置** > **發佈商** > **添加發行者**,添加更多發行者。

>[!NOTE]
>在添加新發佈商之前,請造訪合作夥伴中心並選擇 **「帳戶設置** > **發行者**」以查看現有發行者的清單,查看現有發行者。

來自同一 Azure 活動目錄租戶的另一個使用者可以按照以下步驟添加新發行者:

1. 啟動[微軟合作夥伴中心的](https://partner.microsoft.com/en-us/dashboard/account/v3/enrollment/introduction/azureisv)註冊流程。
2. 選擇**使用工作帳戶登錄**並輸入您的工作電子郵件位址。
3. 選擇「**新增發行者」** 按鈕。
4. 選擇要關聯到發布者的 MPN ID。
5. 更新表單的**資訊**型資訊 。

   1. **發行者名稱**:將在商業市場隨產品/服務顯示在其中的名稱。
   1. **發行者 ID**:合作夥伴中心用於唯一標識發佈者識別碼。 此欄位的預設值映射到系統中現有的唯一且唯`PublisherID`一的,無法重複使用,因此需要更新此欄位。
   1. **聯繫資訊**:必要時更新聯繫資訊。

完成此過程後,您可以通過訪問左側導航功能表中列出的**商業應用商店**程式來管理新創建的發行者。 如果您沒有看到**商業市場**計劃,請刷新頁面。 新發行者將顯示在 **「發佈者」** 清單中。
