|如果您的系統管理員已為您的組織開啟新的**我的應用程式**預覽體驗，當您選取帳戶圖示以在 [**我的應用程式**] 頁面頂端查看帳戶資訊時，您現在會看到連結。 如果您選取 **[試用新的我的應用程式體驗！**]，就會立即帶您前往新的https://myapplications.microsoft.com) URL （和更新的體驗）。 之後，當您返回 [**我的應用程式**] 頁面時，就會自動移至新的體驗。<br><br>若要停止使用新的體驗，您可以在選取帳戶圖示時選取 [**離開預覽體驗**]，以在 [**我的應用程式**] 頁面頂端查看您的帳戶資訊。 您會立即回到舊的 URL (myapps.microsoft.com) 和體驗。 之後，每當您返回 [我的應用程式]**** 頁面時，就會自動前往舊的體驗。|
