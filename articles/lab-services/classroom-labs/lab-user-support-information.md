# <a name="view-support-information-lab-user-in-azure-lab-services"></a>查看支援資訊（Azure 實驗室服務中的實驗室使用者）
本文介紹您（作為實驗室使用者）如何查看以下支援資訊：

- URL
- 電子郵件
- 電話
- 其他指示

在實驗室帳戶中使用實驗室時遇到任何技術問題時，可以使用此資訊獲取説明。

 
## <a name="view-support-information"></a>檢視支援資訊
1. 登錄到[Azure 實驗室服務門戶](https://labs.azure.com)。
2. 選擇需要幫助的**實驗室或虛擬機器**，然後選擇 **"實驗室**"或"虛擬機器"。 在頁面的右上角。 
3. 確認您看到指向**視圖支援網站**、**電子郵件支援****和支援電話號碼的連結**。

    ![檢視支援資訊](../media/lab-user-support-information/support-information.png)
4. 您可以通過切換到下拉清單中的該實驗室來查看其他實驗室的支援聯繫資訊。 

    ![切換到其他實驗室](../media/lab-user-support-information/switch-another-lab.png)
5. 現在，您將看到其他實驗室的支援聯繫資訊。 

    ![其他實驗室的支援資訊](../media/lab-user-support-information/second-lab-support-information.png)
