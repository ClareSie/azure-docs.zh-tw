#### <a name="c-nuget-package"></a>C# NuGet 包裝

.NET 語音 SDK 可以從 **.NET 核心 CLI**安裝`dotnet add`,並具有以下命令。

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

.NET 語音 SDK 可以從**包管理器**`Install-Package`安裝,並具有以下 命令。

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
