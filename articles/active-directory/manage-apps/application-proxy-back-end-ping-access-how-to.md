# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>如何設定應用程式 Proxy 應用程式以使用 PingAccess

我們與 PingAccess 的合作，使您現在可以使用標頭形式驗證，將應用程式 Proxy 的優點延伸到應用程式。 如果您的應用程式不使用標頭，請參閱我們的[單一登入文件](application-proxy-configure-single-sign-on-with-kcd.md)以取得其他選項的詳細資料。

## <a name="overview-of-steps-and-recommended-documents"></a>步驟和建議文件的概觀

若要搭配 PingAccess 設定應用程式，請執行四個步驟：

1.  設定應用程式 Proxy 連接器

2.  建立 Azure AD 應用程式 Proxy 應用程式

3.  下載及設定 PingAccess

4.  在 PingAccess 中設定應用程式

如需上述每個步驟的詳細資訊，請參閱我們的[搭配標頭的單一登入文件](application-proxy-configure-single-sign-on-with-ping-access.md)。
