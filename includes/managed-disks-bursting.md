在 Azure 上，我們提供將磁片儲存體 IOPS 和 MB/s 效能提升為虛擬機器和磁片上的高載的能力。 高載在許多案例中很有用，例如處理非預期的磁片流量或處理批次作業。 您可以有效地利用 VM 和磁片層級的負載平衡，在您的 VM 和磁片上達成絕佳的基準和高載效能。 如此一來，您可以在 vm 和磁片上達成絕佳的基準效能和高載效能。 

請注意，磁片和 Vm 上的高載是彼此獨立的。 如果您有負載平衡磁片，則不需要高載 VM 以允許磁片高載。 如果您有高載 VM，您不需要高載磁片來允許 VM 高載。
