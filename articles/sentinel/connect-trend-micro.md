# <a name="connect-trend-micro-deep-security"></a>連接趨勢微深安全

趨勢微深度安全連線器可讓您輕鬆地將深層安全性記錄檔與 Azure Sentinel 連接，以查看儀表板、創建自訂警報並改進調查。 這使您可以更深入地瞭解組織的網路和系統，並改進您的安全操作功能。


## <a name="prerequisites"></a>Prerequisites

- 讀取和寫入 Azure Sentinel 工作區的許可權。

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>將 Azure 哨兵連接到趨勢微深度安全

1. 在 Azure 門戶中，導航到**Azure 哨兵** > **資料連線器**，然後選擇**趨勢微深層安全**連接器。

2. 選擇 **"打開連接器"頁**。

3. 按照說明和後續步驟操作。
