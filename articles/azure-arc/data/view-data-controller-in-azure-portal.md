# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>在 Azure 入口網站中查看 Azure Arc 的資料控制器資源

當您完成第一個 [度量或記錄上傳至 azure](upload-metrics-and-logs-to-azure-monitor.md) 或 [上傳的使用](view-billing-data-in-azure.md)方式之後，您就可以看到在 azure 中建立的 Azure Arc 資料控制器資源在 Azure 入口網站中。

使用特殊 URL 開啟 Azure 入口網站：  [https://portal.azure.com](https://portal.azure.com) 。

依名稱在搜尋列中搜尋您的資料控制器，然後按一下它。
