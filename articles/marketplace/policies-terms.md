# <a name="commercial-marketplace-policies-and-terms"></a>商業市場政策和條款

>[!Note]
>"微軟商業市場"是指 Microsoft 擁有或運營的平臺，無論該平臺如何命名，通過該平臺，產品/服務可能會向客戶展示或獲得。 除非另有說明，商業市場包括微軟 Azure 應用商店和微軟 AppSource。

感謝您對商業市場上發佈優惠的興趣。 我們致力於與您合作，打造豐富的雲解決方案和業務系列產品，讓全球客戶滿意，並説明您建立業務。

市場上的報價必須符合我們的政策和條款。 我們不時更新這些政策和條款，以説明確保良好的客戶體驗，並為客戶提供成功的機會。 如果您對我們的政策或條款有回饋，請在[Azure 應用商店和 AppSource 論壇](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222)中發表評論，告知我們。
