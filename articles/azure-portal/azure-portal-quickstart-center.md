# <a name="get-started-with-the-azure-quickstart-center"></a>開始使用 Azure 快速啟動中心

Azure 快速入門中心是 Azure 門戶中的一種指導性體驗，可供任何想要提高 Azure 知識的任何人使用。 對於新加入 Azure 的組織，這是註冊和設置雲環境的最快方法。

## <a name="overview"></a>總覽

Azure 快速入門中心有三個入門選項：

* **設置指南**：專為 IT 管理員和雲架構師設計，我們的指南介紹了 Azure 採用的關鍵概念。 結構化步驟可説明您在學習時採取行動，應用 Microsoft 推薦的最佳實踐。 遷移指南可説明您在準備遷移到 Azure 時評估就緒情況並進行規劃。

* **啟動專案**：如果您已準備好創建資源，則此部分允許您在提交服務選項之前瞭解有關您的選擇的更多資訊。 您將瞭解有關服務以及使用服務的原因、探索成本和確定先決條件的更多。 做出選擇後，您可以直接進行創建。

* **線上學習**：Azure 快速入門中心的這一部分突出顯示了 Microsoft 學習的免費入門課程模組。 選擇磁貼以啟動課程並瞭解有關 Azure 中的雲概念和管理資源詳細資訊。

## <a name="how-to-use-azure-quickstart-center"></a>如何使用 Azure 快速啟動中心

1. 登錄到 Azure[門戶](https://portal.azure.com)。

1. 從 Azure 門戶功能表中選擇**所有服務**。

1. 選擇**常規** > **快速啟動中心**。
