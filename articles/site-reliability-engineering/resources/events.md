# <a name="participate-in-the-global-site-reliability-engineering-community"></a>參與全球網站可靠性工程社區

與您的同儕彼此討論就和研讀 SRE 一樣重要，而且往往更為重要。 談論您關於 SRE 的挑戰、成功與挫折，對於理解這個主題有著關鍵影響。

以下有一些主打 SRE 內容的聚會與會議。 也許最直接相關的是 USENIX 舉辦的全球分佈[的 SREcon 會議](https://www.usenix.org/conferences)。

越來越多的SRE內容正在通過會議，如[LISA，](https://www.usenix.org/conferences/byname/5)[全天DevOps，DevOps](https://www.alldaydevops.com)[企業峰會](https://events.itrevolution.com)和本地DevOps會議，如[DevOps天](https://www.devopsdays.org)。 無論在哪裡找到這個主題，都可以查找此內容以及對主題感興趣的其他人。
