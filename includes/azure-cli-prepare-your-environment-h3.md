### <a name="prepare-your-environment-for-the-azure-cli"></a>準備您的環境以進行 Azure CLI

- 若要執行 Azure CLI 命令，請執行 Azure CLI 的[本機安裝](/cli/azure/install-azure-cli)，或使用 bash 環境啟動 [Azure Cloud Shell](/azure/cloud-shell/quickstart)。

- 如果您使用的是本機安裝，請使用 [az login](/cli/azure/reference-index#az-login) 命令，透過 Azure CLI 來登入。  請遵循您終端機上顯示的步驟，來完成驗證程序。  請參閱 [使用 Azure CLI 登入](/cli/azure/authenticate-azure-cli) ，以取得其他登入選項。
- 出現提示時，請在第一次使用時安裝 Azure CLI 擴充功能。  在[使用擴充功能搭配 Azure CLI](/cli/azure/azure-cli-extensions-overview) 中深入了解擴充功能。
