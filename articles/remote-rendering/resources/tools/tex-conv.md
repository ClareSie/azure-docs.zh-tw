# <a name="texconv---texture-conversion-tool"></a>TexConv-紋理轉換工具

TexConv 工具是用來執行批次紋理轉換的外部命令列工具。
您可以在此 git 存放庫中找到此工具： [適用于 TexConv 工具的 GitHub 存放庫](https://github.com/microsoft/DirectXTex/wiki/Texconv)。

針對遠端轉譯，紋理不需要轉換，因為轉換步驟可以處理常見的影像檔案格式，並將材質轉換為適當的執行時間格式。 不過，此工具有助於在本機使用自訂設定來轉換紋理。
