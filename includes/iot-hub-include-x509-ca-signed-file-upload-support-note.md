> [!IMPORTANT]
> 使用 x.509 憑證授權單位單位 (CA) 驗證的裝置不支援檔案上傳功能。 使用 x.509 指紋驗證的裝置支援此功能。 若要深入瞭解使用 IoT 中樞的 x.509 authentication，請參閱 [支援的 x.509 憑證](../articles/iot-hub/iot-hub-devguide-security.md#supported-x509-certificates)。
