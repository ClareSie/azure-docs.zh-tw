# <a name="office-365-dynamics-365-power-apps-and-power-bi-offer-publishing-guide"></a>Office 365、動態 365、電源應用和 Power BI 提供發佈指南

Microsoft AppSource 包括基於動態 365、Office 365、Power BI 和電源應用構建或擴展的優惠。 這些產品中的每一個都有特定的文檔,可指導您進行發佈過程。 有關發佈每個產品/服務類型的發佈流程和指南的連結,請參閱下文。

AppSource 已付費、列出和試用優惠,並調用"立即獲取"、"立即聯繫"和"立即試用"。

## <a name="product-specific-guides"></a>特定於產品的指南

| **產品**    | **發佈指南**  |
| :------------------- | :-------------------|
| Office 365 | [檢視 發行流程和指南](/office/dev/store/submit-to-the-office-store) |
| Dynamics 365 Business Central | 要在合作夥伴中心發佈產品/服務,請參閱[創建新的 Dynamics 365 商業中心產品/服務](./partner-center-portal/create-new-business-central-offer.md)。 雲合作夥伴門戶不再支援創建和管理 Dynamics 365 業務中心產品/ |
| Dynamics 365 for Finance and Operations | 構建企業版時,請查看[發佈過程和指南](/dynamics365/unified-operations/dev-itpro/lcs-solutions/lcs-solutions-app-source)。<br/>要在合作夥伴中心發佈產品/服務,請參閱[為運營產品/服務創建新的 Dynamics 365。](./partner-center-portal/create-new-operations-offer.md) 雲合作夥伴門戶不再支援創建和管理 Dynamics 365,用於財務和運營產品。 |
| Dynamics 365 for Customer Engagement | 檢閱[發佈程序和指導方針](/dynamics365/customer-engagement/developer/publish-app-appsource)。<br/>要在合作夥伴中心發佈產品/服務,請參閱[為客戶互動創建新的動態 365 & PowerApps 產品/](./partner-center-portal/create-new-customer-engagement-offer.md) 雲合作夥伴門戶不再支援創建和管理 Dynamics 365,用於財務和運營產品。 |
| Power Apps | 瞭解如何[讓客戶在 AppSource 上測試您的應用程式](https://powerapps.microsoft.com/blog/appsource-test-drive/)。 |
| Power BI | 檢閱[發佈程序和指導方針](/power-bi/developer/office-store)。 |
