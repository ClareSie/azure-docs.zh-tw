# <a name="cloudsimple-node-quota-limits"></a>雲簡單節點配額限制

在為 CloudSimple 服務啟用訂閱時，四個節點是可供購買的預設數量。  可以從 Azure 門戶購買任何[節點類型](cloudsimple-node.md)。  創建私有雲至少需要三個同一 SKU 的節點。  如果您購買了節點，則當您嘗試購買其他節點時，您可能會看到錯誤。

## <a name="quota-increase"></a>配額增加

您可以通過提交支援請求來增加節點配額。 服務操作團隊將評估請求，並與您合作增加節點配額。  打開新票證時選擇以下選項：

* 問題類型：**技術**
* 訂閱：**您的訂閱 ID**
* 服務類型 **：VMware 解決方案（按雲簡單）**
* 問題類型：**專用節點配額**
* 問題子類型：**增加專用節點的配額**
* 主題：**配額增加**

在支援票證的詳細資訊中，提供所需的節點數和節點 SKU。

* 節點 SKU
* 請求增加配額的其他節點數
