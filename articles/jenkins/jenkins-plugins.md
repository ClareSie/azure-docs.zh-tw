# <a name="jenkins-plugins-for-azure"></a>Azure 的 Jenkins 外掛程式

以下 Jenkins 外掛程式支援各種功能，以便與 Azure 一起使用。
  
| 詹金斯外掛程式 | 描述                                   |
|------------------------------------------------------------------------------|
| [Azure 應用程式服務外掛程式](https://plugins.jenkins.io/azure-app-service)     | Jenkins 外掛程式用於部署 Azure 應用服務（目前僅支援 Web 應用）。 | 
| [Azure AD 外掛程式](https://plugins.jenkins.io/azure-ad)                       | 通過 Azure 活動目錄支援身份驗證&授權的 Jenkins 外掛程式。 | 
| [Azure 專案管理器外掛程式](https://plugins.jenkins.io/azure-artifact-manager) | Azure 專案管理器外掛程式是一個專案管理器，允許您將專案存儲在 Azure Blob 存儲中。 Azure 專案管理器外掛程式可透明地工作到 Jenkins 和作業，它類似于預設專案管理器。 | 
| [Azure 容器代理程式外掛程式](https://plugins.jenkins.io/azure-container-agents) | Azure 容器代理外掛程式可以説明您在 Jenkins 中作為代理運行容器 | 
| [Azure 容器註冊表任務外掛程式](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins 外掛程式將 Docker 生成要求傳送到[Azure 容器註冊表](/azure/container-registry/container-registry-tasks-overview)。 |
| [Azure Container Service 外掛程式](https://plugins.jenkins.io/azure-acs)       | Jenkins 外掛程式將配置部署到 Azure 容器服務 （AKS）。 | 
| [Azure 認證外掛程式](https://plugins.jenkins.io/azure-credentials)      | Jenkins 外掛程式用於管理 Azure 憑據。 | 
| [Azure Function 外掛程式](https://plugins.jenkins.io/azure-function)           | Jenkins 外掛程式以部署 Azure 函數。 | 
| [Azure IoT 邊緣外掛程式](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge 外掛程式可輕鬆設置 CI/CD 管道，以便在 Jenkins 上開發 IoT Edge 解決方案。 | 
| [Azure Service Fabric 外掛程式](https://plugins.jenkins.io/service-fabric)     | 為 Linux Azure 服務結構專案提供 Jenkins 外掛程式。 |
| [Azure 儲存體外掛程式](https://plugins.jenkins.io/windows-azure-storage)     | 用於將生成專案上載到 Microsoft Azure Blob 存儲或從下載生成依賴項的外掛程式。 | 
| [Azure VM 代理外掛程式](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins 外掛程式在 Azure 虛擬機器中創建 Jenkins 代理（通過 Azure 資源管理器範本）。 | 
| [Azure 虛擬機器縮放集外掛程式](https://plugins.jenkins.io/azure-vmss)           | Jenkins 外掛程式將 VM 映射部署到 Azure 虛擬機器縮放集。 |
