## <a name="prerequisites"></a>必要條件

您必須先具備下列專案，才能繼續進行本文中的工作：

- [安裝的用戶端工具](../articles/azure-arc/data/install-client-tools.md)

   工具組括 `azdata` 、Azure Data Studio 和 Azure CLI。

   針對 Azure Data Studio，視資料服務而定，安裝適用于 **Azure DATA CLI**、 **Azure Arc**和 **于 postgresql** (的延伸模組) 

- [已安裝 Azure Arc 資料控制器](../articles/azure-arc/data/create-data-controller.md)
