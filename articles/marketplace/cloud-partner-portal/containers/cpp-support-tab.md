# <a name="container-support-tab"></a>容器支援索引標籤

> [!IMPORTANT]
> 從2020年4月13日開始，我們會開始將您的 Azure 容器供應專案管理移至合作夥伴中心。 在遷移之後，您將在合作夥伴中心建立和管理您的供應專案。 請依照[建立 Azure 容器供應](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-azure-container-offer)專案中的指示來管理您的已遷移優惠。

使用 [新增供應項目]**** 頁面中的 [支援]**** 索引標籤來協助您為供應項目提供技術及使用者支援資源。  支援表單上包括下列需要填寫的欄位：**工程連絡人**、**客戶支援**和**支援 URL**。 欄位名稱上附加的星號 (*) 表示此為必填欄位。


## <a name="support-form-fields"></a>支援表單欄位

提供 [工程連絡人] 和 [客戶支援] 欄位所需的資訊。


### <a name="engineering-contact"></a>工程連絡人

工程連絡人是您的組織與 Microsoft 之間的技術連絡人。 必要欄位由星號 (*) 標示。

- **名稱\* ** -做為技術/工程支援的人員或組名。
- **電子\*郵件**-此技術連絡人的電子郵件地址。
- **電話\* ** -技術支援的電話號碼。


### <a name="customer-support"></a>客戶支援

客戶支援連絡人會收到客戶在 Azure 中開啟的支援票證。  必要欄位由星號 (*) 標示。

- **名稱\* ** -做為客戶支援的人員或組名。
- **電子\*郵件**-此支援連絡人的電子郵件地址。
- **電話\* ** -客戶支援的電話號碼。


## <a name="next-steps"></a>後續步驟

提供支援資訊之後，您即可[發佈供應項目](./cpp-publish-offer.md)。
