# <a name="open-a-support-ticket-for-azure-data-box-edge-and-azure-data-box-gateway"></a>打開 Azure 資料框邊緣和 Azure 資料框閘道的支援票證

本文適用于 Azure 資料框邊緣和 Azure 資料框閘道，它們都由 Azure 資料框邊緣/Azure 資料框閘道服務管理。 如果服務遇到任何問題，可以創建技術支援服務請求。 本文將引導您：

* 如何建立支援要求。
* 如何在入口網站上管理支援要求的生命週期。

## <a name="create-a-support-request"></a>建立支援要求

執行下列步驟來建立支援要求：

1. 轉到資料框邊緣或資料框閘道訂單。 導航到**支援 + 故障排除**部分，然後選擇 **"新建支援請求**"。
   
2. 在 **"基本"** 選項卡上**的新支援請求**中，執行以下步驟：
    
    1. 從 [問題類型]**** 下拉式清單中，選取 [技術]****。
    2. 選擇您的**訂閱**。
    3. 在 [服務]**** 底下，勾選 [我的服務]****。 從下拉清單中，選擇 **"資料框邊緣"和"資料框閘道**"。
    4. 選擇您的**資源**。 這對應于您的訂單名稱。
    5. 簡要**概述**您遇到的問題。 
    6. 選擇您的**問題類型**。
    7. 根據您選擇的問題類型，選擇相應的**問題子類型**。
    8. 選擇 **"下一步"：解決方案 >>**。

        ![基本概念](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)

3. 在"**詳細資訊"** 選項卡上，執行以下步驟：
    
    1. 提供問題開始日期與時間。
    2. 提供您的問題**描述**。
    3. 在 **"檔上傳**"中，選擇資料夾圖示以流覽要上載的任何其他檔。
    4. 勾選 [共用診斷資訊]****。
    5. 根據您的訂閱，將自動填滿**支援計畫**。
    6. 從下拉清單中，選擇**嚴重性**。
    7. 指定**首選連絡人方法**。
    8. **回應時間**會根據您的訂閱計畫自動選擇。
    9. 提供您喜歡支援的語言。
    10. 在**聯繫資訊**中，提供您的姓名、電子郵件、電話、可選連絡人、國家/地區。 Microsoft 支援服務會利用此資訊來連絡您，以取得進一步資訊、進行診斷及解決。 
    11. 選擇 **"下一步"：查看 = 創建 >>**。

        ![問題](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-2.png)

4. 在"**審核 + 創建**"選項卡上，查看與支援票證相關的資訊。 選取 [建立]****。 

    ![問題](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-3.png)

    創建支援票證後，支援工程師將儘快與您聯繫以繼續您的請求。

## <a name="get-hardware-support"></a>獲取硬體支援

此資訊僅適用于資料盒邊緣設備。 報告硬體問題的過程如下：

1. 為硬體問題從 Azure 門戶打開支援票證。 在 **"問題類型**"下，選擇 **"資料框邊緣硬體**"。 選擇 **"問題子類型**"作為**硬體故障**。 

    ![硬體問題](./media/data-box-edge-contact-microsoft-support/data-box-edge-hardware-issue-1.png)

    創建支援票證後，支援工程師將儘快與您聯繫以繼續您的請求。 

2. 如果 Microsoft 支援確定這是一個硬體問題，則發生以下操作之一： 

    - 將發送故障硬體部件的現場更換單元 （FRU）。 目前，電源單元是唯一受支援的 FRU。 
    - 對於任何其他部件故障，Microsoft 會執行完整的系統更換 （FSR） 或設備交換。

3. 如果在當地時間下午 4：30（週一至週五）之前升起支援票證，則現場技術人員將派遣至您所在的位置執行 FRU 或完整設備更換。

## <a name="manage-a-support-request"></a>管理支援要求

建立支援票證之後，您便可以從入口網站中管理票證的生命週期。

#### <a name="to-manage-your-support-requests"></a>若要管理支援要求

1. 若要移至說明和支援頁面，請瀏覽至 [瀏覽] > [說明 + 支援]****。

    ![管理支援要求](./media/data-box-edge-contact-microsoft-support/data-box-edge-manage-support-request-1.png)   

2. [說明 + 支援]**** 中會以表格式清單列出 [最近的支援要求]****。

    <!--[Manage support requests](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)--> 

3. 選取並按一下支援要求。 您可以檢視此要求的狀態與詳細資料。 如果您想要追蹤此要求，請按一下 [+ New message]\(+ 新訊息\)****。
