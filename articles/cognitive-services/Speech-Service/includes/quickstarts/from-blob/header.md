在本快速入門中，您將使用 REST API 於批次程序中辨識檔案內的語音。 批次程序不需要任何使用者互動就會執行語音轉譯。 其會提供您簡單的程式設計模型，讓您不必管理並行、自訂語音模型或其他詳細資料。 其需要進階的控制選項，才能有效率地使用 Azure 語音服務資源。

如需可用選項和組態詳細資料的詳細資訊，請參閱[批次轉譯](../../../batch-transcription.md)。
