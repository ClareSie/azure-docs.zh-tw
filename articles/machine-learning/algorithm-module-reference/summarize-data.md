# <a name="summarize-data"></a>資料總結

本文介紹 Azure 機器學習設計器（預覽）模組。

使用匯總資料模組創建一組標準統計度量值，用於描述輸入表中的每一列。

當您想要瞭解完整資料集的特徵時，摘要統計資訊非常有用。 例如，您可能需要知道：

- 每個資料行中遺漏多少值？
- 要素列中有多少個唯一值？
- 每列的平均偏差和標準差是多少？

模組計算每列的重要分數，並返回作為輸入提供的每個變數（資料列）的匯總統計資訊行。

## <a name="how-to-configure-summarize-data"></a>如何配置匯總資料  

1. 將**匯總資料**模組添加到管道中。 您可以在設計器中的 **"統計函數**"類別中找到此模組。

1. 連接要為其生成報表的資料集。

    如果只想報告某些列，請使用["資料集"模組中的"選擇列](select-columns-in-dataset.md)"投影要使用的列子集。

1. 不需要其他參數。 預設情況下，模組分析作為輸入提供的所有列，並根據列中的數值型別輸出一組相關的統計資訊，如["結果"](#results)部分所述。

1. 提交管道。

## <a name="technical-notes"></a>技術說明

- 針對非數值資料行，只會計算計數、唯一值計數和遺漏值計數。 至於其他統計資料，則傳回 null 值。

- 包含布林值的列使用以下規則進行處理：

    - 計算 Min 時，套用邏輯 AND。
    
    - 計算最大值時，將應用邏輯 OR
    
    - 計算「範圍」時，模組會先檢查資料行中的唯一值數目是否等於 2。
    
    - 在計算需要浮點計算的任何統計資料，True 值視為 1.0，False 值視為 0.0。
