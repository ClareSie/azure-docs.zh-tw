# <a name="integrated-solutions-in-microsoft-appsource-and-azure-marketplace"></a>微軟應用程式來源和 Azure 應用商店中的整合解決方案

整合解決方案彙集了Microsoft合作夥伴提供的同類最佳技術,以及專家服務合作夥伴提供的諮詢、部署和支援。

兩個或多個 Microsoft 合作夥伴整合了他們的技術和服務,以創建統一、無縫和可重複的部署體驗,以便快速啟動和運行客戶。  解決方案中的領先合作夥伴(通常是服務合作夥伴)為客戶提供單一聯繫點,圍繞解決方案提供相關的自訂、諮詢和支援。

在其他方案中,合作夥伴共同銷售或共同銷售其解決方案,而集成解決方案是一種共同構建的新解決方案,用於跨合作夥伴集成技術和服務。

## <a name="advantages"></a>優點

綜合方法的優點是:

* *行業專業知識*:服務合作夥伴提供行業特定解決方案和知識
* *價值時間*:整合部署方法,幫助您實現眼前的好處
* *同類最佳的解決方案*:瞭解集成購買和實施體驗的簡便性

有關詳細資訊,請參閱[發佈整合解決方案](https://docs.microsoft.com/azure/marketplace/integrated-solutions-for-publishers)。

## <a name="submit-a-nomination"></a>提交提名

希望參與該計劃的合作夥伴應確定要使用的其他合作夥伴。 他們應填寫[綜合解決方案提名表](https://assetsprod.microsoft.com/mpn/integrated-solutions-partner-nomination.docx)。 集成解決方案計劃團隊審核提名並在 10 個工作日內跟進。 團隊提供批准和後續步驟。
