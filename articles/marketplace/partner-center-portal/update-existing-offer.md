# <a name="update-an-existing-offer-in-the-commercial-marketplace"></a>更新商業市集中的現有供應項目

您可以在合作夥伴中心[商業市場門戶](https://partner.microsoft.com/dashboard/commercial-marketplace/offers)的 **"概述"** 選項卡上查看現有優惠。

要更新當前市場上現有的產品/服務：：

1. 選擇要更新的優惠名稱。 報價的狀態可能被列為**即時**，**發佈在進行**中，**草稿**，**需要注意**，或**不在市場上出售**（如果你以前選擇停止銷售報價）。 選中後，該優惠的 **"產品/服務"概覽**頁面將打開。
2. 選擇"從概覽卡片**中更新**"，或從左側導航中選擇功能表項目，以選擇要更新的區域。 您可能需要更新**產品/服務設置**、**屬性**、**優惠清單**、**預覽**、**技術配置**、**計畫概述**或**試駕**。 
3. 進行更改，然後選擇 **"保存**"。 重複此過程，直到所有更改完成。

## <a name="review-and-publish-an-updated-offer"></a>查看併發布更新的報價

準備好發佈更新的優惠時，從任何頁面選擇 **"發佈**"。 將打開 **"審閱和發佈**"頁。 在此頁面上，您可以：

- 請參閱您更新的報價部分的完成狀態： 
    - **未發佈的更改**：該部分已更新並已完成。 提供了所有必需的資料，並且更新中沒有引入任何錯誤。
    - **不完整**：對該部分所做的更新引入了需要修復或需要提供更多資訊的錯誤。
- 向認證測試團隊提供其他資訊，以確保測試順利進行。
- 通過選擇 **"提交**"提交更新的發佈產品/服務。  當您有更新優惠的預覽版本可供您查看和登出時，我們會向您發送電子郵件。

> [!IMPORTANT]
> 一旦優惠預覽可用，您必須查看該優惠預覽，並選擇 **"Go-live"** 將更新的優惠發佈到預期受眾（公共或私人）。

## <a name="update-a-plan-within-an-existing-offer"></a>在現有產品/服務範圍內更新計畫

要更改已發佈的現有產品/服務範圍內的計畫，：

- 打開現有**優惠的"產品/服務概覽**"頁後，請選擇要更改的計畫。 如果計畫無法從 **"計畫概覽"清單中**訪問，請選擇"**查看所有計劃**"。
- 您可以選擇計畫**名稱**、**定價模型**或**可用性**。 *目前計畫僅提供英文（美國）版本*。
- 對計畫名稱、說明、定價或訪問群體可用性進行任何必要的更改後選擇 **"保存**"。 
- 準備好發佈更新後選擇 **"發佈**"。 "**[審閱"和"發佈](#review-and-publish-an-updated-offer)**"頁將打開，並為您的更新提供完成狀態。

## <a name="compare-changes-to-marketplace-offers"></a>將更改與市場優惠進行比較

您可以使用 比較 對[已發佈](#compare-changes-to-published-offer)或[預覽](#compare-changes-to-a-preview-offer)產品/服務進行即時更改，然後對其進行**審核**。

>[!NOTE]
> 已發佈的產品/服務是已成功發佈到預覽版或即時狀態的優惠。

有關一般審核資訊，請參閱下文：

- 您可以在編輯過程中的任何點使用**比較**。
- 在 **"比較**"頁上選擇一個欄位以導航到要修改的值。
- 如果您已準備好發佈更新，請選擇 **"審閱"併發布**。
- 要查看所有欄位的值，即使欄位未更新，請選擇"**所有欄位**"篩選器。 您可以通過選擇 **"修改"欄位**來修改這些欄位中的篩選器，然後選擇以下篩選器之一：
    - **刪除的值**篩選器顯示您發佈的欄位，現在您正在完全刪除。
    - **添加的值**篩選器顯示您最初未發佈且正在添加的欄位。
    - **編輯的值**篩選器顯示已發佈但現已更新內容的欄位。

      >[!NOTE]
      > 如果這些篩選器之一不可用，則表示您沒有更新該類型。

- 要僅查看尚未更新的值，請選擇"**未更改欄位**"篩選器。 為已發佈版本和草稿版本顯示的欄位值將相同。

  ![用於將更新與已發佈或預覽產品/服務進行比較的篩選器](./media/compare-changes-marketplace.png)

>[!NOTE]
> 以下頁面當前不支援**比較**：
>- CSP 轉銷商受眾
>- 試駕技術配置
>- 試駕市場清單
>- 共同銷售
>- 補充檔

### <a name="compare-changes-to-published-offer"></a>將更改與已發佈產品/服務進行比較

Follow the instructions below to compare your changes from the published offer:

1. 在頁面的命令列中選擇 **"比較**"。 **比較**頁面提供此產品/服務已保存的更改和已發佈的市場產品/服務並行版本。
2. 當您從產品/服務的特定頁面訪問**比較時**，預設值僅顯示該頁面上所做的更改。
    - 如果要比較所有頁面上的更改，請從 **"選擇要比較的頁面"更改頁面**。  
    - 如果要比較所有頁面對產品/服務所做的更改，請選擇 **"所有頁面**"。

預設情況下，**比較**將您對即時市場產品/服務的新更改進行比較。

請記住，在更新更改以生效後重新發佈產品/服務。

### <a name="compare-changes-to-a-preview-offer"></a>將更改與預覽優惠進行比較

如果預覽版中的更改不即時，則可以將新更改與預覽市場產品/服務進行比較。

按照以下說明將新更改與您的預覽市場產品/服務進行比較：

1. 在頁面的命令列中選擇 **"比較**"。
2. 選擇 **"帶下**拉清單"並將其從**即時優惠**更改為**預覽優惠**。
3. "**比較**"頁提供顯示更改的並行版本。

>[!NOTE]
>如果您的報價尚未變為即時，但您已發佈預覽版，則您無法選擇與即時優惠進行比較。

請記住，在更新更改以生效後重新發佈產品/服務。

## <a name="delete-a-draft-offer"></a>刪除要約草稿

您可以通過從 **"優惠概覽**"頁中選擇 **"刪除"草稿**來刪除草稿產品/服務（尚未發佈）。 如果您以前發佈過產品/服務，此選項將不可用。

確認要刪除優惠草稿後，優惠將不再在合作夥伴中心可見或訪問，**並且"所有優惠"** 頁面將打開。

## <a name="delete-a-draft-plan"></a>刪除計畫草稿

要刪除尚未發佈的計畫，請從 **"計畫概述"** 頁**中選擇"刪除草稿**"。 如果您以前發佈過產品/服務，此選項將不可用。

確認要刪除計畫草稿後，該計畫將不再在合作夥伴中心可見或訪問。

## <a name="cancel-publishing"></a>取消發佈

要取消處於 **"發佈"** 狀態的優惠：

- 選擇產品/服務名稱以打開 **"產品/服務概覽**"頁。
- 選擇 **"從頁面右上角取消發佈**"。
- 確認要停止發佈產品/服務。

如果要在以後發佈產品/服務，則需要重新開始發佈過程。

> [!NOTE]
> 僅當產品/服務尚未向發行者簽署步驟時，才能阻止發佈產品/服務。 選擇 **"上線"** 後，您將無法選擇取消發佈。

## <a name="stop-selling-an-offer-or-plan"></a>停止銷售要約或計畫

要在發佈產品/服務後停止銷售產品/服務，請從 **"優惠概覽**"頁面選擇 **"停止銷售**"。 要停止銷售計畫，請從 **"計畫概覽"** 頁中選擇 **"停止銷售**"。 （只有在產品/服務單中有多個計畫時，才提供停止銷售計畫的選項。 您可以選擇停止銷售一個計畫，而不影響產品/服務範圍內的其他計畫。

一旦您確認要停止銷售產品/服務或計畫，在幾個小時內，它將不再在市場上可見，並且沒有新客戶可以下載它。

任何以前獲得產品/服務或計畫的客戶仍可以使用。 他們可以再次下載它，但如果您稍後更新並重新發佈產品/服務或計畫，他們將不會獲得更新。 

完成停止銷售產品/計畫的請求後，您仍將在合作夥伴中心的商業市場門戶中看到它。 要刪除草稿，請從 **"優惠概覽**"或"**計畫概覽"** 頁中選擇 **"刪除草稿**"。 

如果您決定列出或出售此要約或再次計畫，請按照說明[更新現有報價](#update-an-existing-offer-in-the-commercial-marketplace)。 不要忘記，在進行任何更改後，您需要**發佈**產品/服務或再次計畫。

## <a name="remove-offers-from-existing-customers"></a>刪除現有客戶的產品/服務

要刪除現有客戶的產品/服務，[請記錄支援請求](https://support.microsoft.com/supportforbusiness/productselection?sapId=48734891-ee9a-5d77-bf29-82bf8d8111ff)。 從 **"問題類型"** 清單中選擇您的特定產品/服務類型，然後在 **"類別"** 清單中選擇 **"刪除已發佈的產品/** 服務"。 提交要求。 支援小組將指導您完成產品/服務刪除過程。

## <a name="sync-private-plan-audiences"></a>同步私人計畫訪問群體

如果您的優惠包含一個或多個計畫，這些計畫配置為僅對私人受限受眾可用，則可以僅更新可以訪問該私人計畫的受眾，而無需發佈對產品/服務的其他更改。 

要更新和同步計畫的私人訪問群體：

- 使用 +**添加 ID**或**導入客戶 （csv）** 按鈕修改一個或多個私人計畫中的受眾，然後保存更改。
- 從 **"計畫概述"** 頁面選擇 **"同步私人受眾**"。

**同步私人訪問群體**將僅向您的私人訪問群體發佈更改，而不發佈您可能對草稿產品/服務所做的任何其他更新。
