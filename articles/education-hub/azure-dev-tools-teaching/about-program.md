# <a name="what-is-azure-dev-tools-for-teaching"></a>什麼是 Azure Dev Tools for Teaching？

Microsoft Azure Dev Tools for Teaching 讓教職員和學生能夠以低成本訂用帳戶享用 Microsoft 的專業開發人員工具、軟體及服務。 學生可以免費取得開發人員工具 (建立應用程式、遊戲和網站所需的所有工具)，讓他們能夠追逐夢想、創造技術的下一個重大突破，或讓他們開啟事業。

身為 Azure Dev Tools for Teaching 訂用帳戶的系統管理員，您將能夠：

- 透過集中式線上位置來管理及存取軟體，免於維護 WebStore 或內部網站的必要。

- 透過相同的線上入口網站，讓學術使用者可以存取軟體、免費訓練/學習材料，以及 Azure 雲端平台。

## <a name="program-details"></a>計畫詳細資料

我們針對以 STEM 為重點的指示，設計 Azure Dev Tools for Teaching。 任何以科學、技術、工程或數學為焦點的課程都有資格使用 Azure Dev Tools for Teaching，協助教授教導以及讓學生更有效率地學習。 

您的 Microsoft Azure Dev Tools for Teaching 訂用帳戶可讓您存取特定軟體開發人員工具。 這些工具可以免費下載。 如果您是已在核准課程中註冊的教職員成員，就有資格在個人電腦上針對非商業用途安裝 Azure Dev Tools for Teaching 軟體。

Azure Dev Tools for Teaching 在 Azure 上市的 140 個以上國家/地區皆有提供。 Education Hub 市集有下列語言版本︰阿拉伯文、簡體中文、繁體中文、丹麥文、荷蘭文、英文、法文、德文、希伯來文、義大利文、日文、韓文、葡萄牙文、俄文、西班牙文、瑞典文和土耳其文。

## <a name="student-benefits"></a>學生權益

所有軟體都可以透過 [Education Hub](https://azureforeducation.microsoft.com/devtools) (內建至 [Azure 入口網站](https://portal.azure.com/)) 存取，這裡放置了可供學術使用者使用的所有產品和服務。 學生只要使用他們用來存取其 Imagine 訂用帳戶的相同認證登入 Azure 入口網站，就能存取軟體。

學生不需要註冊 Azure 訂用帳戶，即可存取其軟體權益。 不過，Microsoft 提供 $ 100 美元的 Azure 點數，以及免費的 Azure 學生版服務存取權：學生不需要信用卡，就可以註冊 [Azure 學生版供應項目](azure-students-program.md)。
