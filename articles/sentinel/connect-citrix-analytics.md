# <a name="connect-citrix-analytics-security"></a>連線 Citrix Analytics (Security) 

Citrix 分析（安全）與 Azure Sentinel 集成可説明您將針對風險事件分析的資料從 Citrix 分析（安全）匯出到 Azure Sentinel。 您可以創建自訂儀表板、分析來自其他來源的資料以及 Citrix 分析（安全性）的資料，並使用邏輯應用創建自訂工作流來監視和緩解安全事件。 



## <a name="prerequisites"></a>Prerequisites

- 讀取和寫入 Azure Sentinel 工作區的許可權。

- 在 Citrix 雲中授權 Citrix 安全分析。


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>將 Azure 哨兵連接到 Citrix 分析（安全性）

1. 在 Azure 門戶中，導航到**Azure 哨兵** > **資料連線器**，然後選擇**Citrix 分析（安全）** 連接器。

2. 選擇 **"打開連接器"頁**。

3. 按照**Citrix 分析（安全）** 頁面上的說明進行操作。
