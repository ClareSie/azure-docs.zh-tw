# <a name="connect-with-the-web-client"></a>與 Web 用戶端連線

Web 用戶端允許您從 Web 瀏覽器訪問 Windows 虛擬桌面資源，而無需冗長的安裝過程。

>[!NOTE]
>Web 用戶端當前不支援移動作業系統。

## <a name="supported-operating-systems-and-browsers"></a>支援的作業系統與瀏覽器

雖然任何支援 HTML5 的瀏覽器都應該工作，但我們正式支援以下作業系統和瀏覽器。

| 瀏覽器           | 支援的 OS                     | 注意               |
|-------------------|----------------------------------|---------------------|
| Microsoft Edge    | Windows                          |                     |
| Internet Explorer | Windows                          |                     |
| Apple Safari      | macOS                            |                     |
| Mozilla Firefox   | Windows、macOS、Linux            | 版本 55 或更高版本 |
| Google Chrome     | 視窗、macOS、Linux、Chrome作業系統 |                     |

## <a name="access-remote-resources-feed"></a>訪問遠端資源源

在瀏覽器中，導航到[Windows 虛擬桌面 Web 用戶端](https://rdweb.wvd.microsoft.com/webclient)，然後使用使用者帳戶登錄。

>[!NOTE]
>如果您已經使用與要用於 Windows 虛擬桌面的帳戶不同的 Azure 活動目錄帳戶登錄，則應登出或使用專用瀏覽器視窗。

登錄後，現在應看到資源清單。 您可以通過在"**所有資源"** 選項卡中選擇資源來啟動資源，就像選擇普通應用一樣。
