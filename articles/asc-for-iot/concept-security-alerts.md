# <a name="azure-security-center-for-iot-security-alerts"></a>適用於 IoT 的 Azure 資訊安全中心的安全性警示

IoT Azure 安全中心使用高級分析和威脅情報持續分析 IoT 解決方案，以提醒您注意惡意活動。
此外，您可以根據您對預期設備行為的瞭解創建自訂警報。
警報充當潛在危害的指示器，應進行調查和修正。

在本文中，您將找到可在 IoT 中心和/或 IoT 設備上觸發的內置警報的清單。
除了內置警報外，IoT Azure 安全中心還允許您根據預期的 IoT 中心和/或設備行為定義自訂警報。
有關詳細資訊，請參閱[可自訂的警報](concept-customizable-security-alerts.md)。
