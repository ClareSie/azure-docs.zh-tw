- **[使用 Azure Data Studio 儀表板監視](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> Azure Data Studio 是跨平台資料庫工具，適合在 Windows、macOS 和 Linux 上使用 Microsoft 系列內部部署和雲端資料平台的資料專業人員。 Azure Data Studio 提供新式編輯器體驗，其中包含 IntelliSense、程式碼片段、原始檔控制整合及整合式終端。 在工程設計時，考量到資料平台使用者，並內建查詢結果集的圖表和可自訂的儀表板。

**[使用 Grafana 進行監視](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> Grafana 是多平臺開放原始碼分析和互動式視覺效果 web 應用程式。 它會在連線到支援的資料來源時，為 web 提供圖表、圖形和警示。 它可以透過外掛程式系統來展開。 終端使用者可以使用互動式查詢來建立複雜的監視儀表板。

**[使用 Kibana 的 Log analytics](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>Kibana 是 Elasticsearch 的開放原始碼資料視覺效果儀表板。 它會在 Elasticsearch 叢集上索引的內容之上提供視覺效果功能。 使用者可以在大量資料上方建立橫條圖、折線圖和散佈圖，或是圓形圖和地圖。
