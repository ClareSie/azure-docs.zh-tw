## <a name="personal-information-entity-types"></a>個人資訊實體類型：

### <a name="person"></a>個人
在文本中識別人員姓名。

語言：
* 公共預覽：`English`

| 子類型名稱 | 描述                                               | 可從模型版本開始 |
|--------------|-----------------------------------------------------------|----------------------------------------|
| N/A          | 識別的名稱，例如`Bill Gates`，`Marie Curie` | `2020-02-01`                           |

### <a name="organization"></a>組織  

認可組織、公司、機構、公司、俱樂部和其他團體。

語言： 

* 公共預覽：`English`

| 子類型名稱 | 描述                                                                                       | 可從模型版本開始|
|--------------|---------------------------------------------------------------------------------------------------|--------------|
| N/A          | 例如`Microsoft`， `NASA``National Oceanic and Atmospheric Administration` | `2020-02-01` |

### <a name="phone-number"></a>電話號碼

電話號碼（僅限美國電話號碼）。 

語言：

* 公共預覽：`English`

| 子類型名稱 | 描述                                    | 可從模型版本開始 |
|--------------|------------------------------------------------|----------------------------------------|
| N/A          | 美國電話號碼，例如`(312) 555-0176` | `2020-02-01`                           |

### <a name="email"></a>電子郵件

電子郵件地址。 

語言：

* 公共預覽：`English`

| 子類型名稱 | 描述                                      | 可從模型版本開始 |
|--------------|--------------------------------------------------|----------------------------------------|
| N/A          | 電子郵件地址，例如`support@contoso.com` | `2020-02-01`                           |

### <a name="url"></a>URL

互聯網網址。

語言：

* 公共預覽：`English`

| 子類型名稱 | 描述                                          | 可從模型版本開始 |

| Azure IoT 連接字串           | Azure 事物 Internet （IoT） 的連接字串。                        |
| Azure 發佈設定密碼        | Azure 發佈設置的密碼。                                        |
| Azure Redis 快取連接字串   | 用於雷瑞斯的 Azure 緩存的連接字串。                             |
| Azure SAS                             | Azure 軟體作為服務 （SAS） 的連接字串。                     |
| Azure 服務匯流排連接字串   | Azure 服務匯流排的連接字串。                                |
| Azure 存儲帳戶金鑰             | Azure 存儲帳戶的帳戶金鑰。                                   |

| SQL Server 連接字串          | SQL 伺服器的連接字串。                                         |
