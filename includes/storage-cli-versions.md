## <a name="versions-of-the-azure-cli"></a>Azure CLI 的版本

目前可提供兩個版本的 Azure 命令列介面 (CLI)：

* [Azure CLI](../articles/storage/common/storage-azure-cli.md)：Azure CLI 與 Resource Manager 部署模型搭配使用目前的版本。
* [Azure 傳統 CLI](../articles/storage/common/storage-azure-cli-nodejs.md):較舊版本的 Azure CLI，適用於僅搭配傳統部署模型。
