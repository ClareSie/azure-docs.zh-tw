**應用程式範本**

| [範本]                 | 名稱        | 說明 |
| ------------------------ | ----------- | ----------- |
| iotc-explorer-pnp-預覽         | 自訂應用程式 | 為您建立空的應用程式，以填入您自己的裝置範本和裝置。 |
| iotc-explorer-條件           | 店內分析-條件監視 | 建立應用程式來連接及監視存放區環境。 |
| iotc-explorer-耗用量         | 水用量監視 | 建立應用程式來監視和控制水流程。 |
| iotc-explorer-散發        | 數位配送中心 | 建立應用程式，藉由將重要資產和動作數位化，來改善倉儲的輸出效率。 |
| iotc-explorer-清查           | 智慧庫存管理 | 建立應用程式，以自動化接收、產品移動、週期計數及追蹤。 |
| iotc-explorer-物流           | 聯網物流 | 建立應用程式，以使用位置與條件監視，在空中、水和 land 之間即時追蹤您的運送。 |
| iotc-explorer-計量               | 智慧型計量分析 | 建立應用程式來監視能源耗用量、網路狀態，並找出趨勢來改善客戶支援和智慧型計量管理。  |
| iotc-explorer-mfc                 | 微型物流中心 | 建立應用程式，以數位方式連接及管理完全自動化的履行中心。 |
| iotc-explorer-影片-分析-om  | 影片分析 - 物件與動態偵測 | 使用攝影機作為智慧型影片分析解決方案中的感應器，由 Azure IoT Edge、AI 和 Azure 媒體服務提供技術支援。 |
| iotc-explorer-患者             | 持續性患者監視 | 建立應用程式來擴充患者護理、減少再住院率，以及管理疾病。 |
| iotc-explorer-電源               | 日光電監視 | 建立應用程式來監視日光板狀態和能源產生趨勢。 |
| iotc-explorer-品質             | 水品質監視 | 建立應用程式以數位方式監視水品質。 |
| iotc-explorer-存放區               | 店內分析-結帳 | 建立應用程式來監視和管理您存放區內的結帳流程。 |
| iotc-explorer-浪費               | 連線浪費管理 | 建立應用程式來監視廢物箱和分派欄位操作員。 |
