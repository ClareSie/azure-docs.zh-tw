# <a name="co-sell-partner-engagement"></a>共同銷售合作夥伴參與

如果您是合作夥伴，您可以使用 Microsoft 共同銷售方案，直接與 Microsoft 合作銷售商機。 此方案可協助我們的合作夥伴觸及廣大的客戶群體，並建立可協助加速業務成長的新銷售商機。

## <a name="partner-selling-motions"></a>合作夥伴銷售運動

Microsoft 提供三個合作夥伴銷售動作：

* *透過合作夥伴銷售*：您可以透過自己的賣方轉售 Microsoft 技術。
* *服務共同銷售*：銷售人員和合作夥伴共同作業，透過內嵌 Microsoft 技術來銷售您的專案或受控服務。
* *IP 共同銷售*：賣方和合作夥伴共同作業，以銷售可重複、已封裝的 IP 解決方案。 *

\*當這些 IP 共同銷售解決方案成功銷售和與 Microsoft 現場銷售團隊一起部署時，可能會收到額外的獎勵。
