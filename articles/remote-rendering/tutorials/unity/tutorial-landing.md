# <a name="azure-remote-rendering-tutorials"></a>Azure 遠端轉譯教學課程

歡迎使用 Azure 遠端轉譯教學課程。 除了學習 Azure 遠端轉譯的重要概念，這些互動式課程也會引導您檢視、操作和自訂遠端轉譯的模型，並凸顯建置安全、可供商業使用的應用程式的各項考量。

### <a name="azure-remote-rendering-tutorials"></a>Azure 遠端轉譯教學課程

由於每個教學課程都是以上一個教學課程為基礎，因此我們建議您依序完成下列教學課程。

#### <a name="viewing-a-remotely-rendered-model"></a>[檢視遠端轉譯模型](view-remote-models/view-remote-models.md)

在本教學課程中，您已建立基本應用程式，能夠使用 Azure 遠端轉譯來檢視遠端轉譯的模型。

#### <a name="interfaces-and-custom-models"></a>[介面和自訂模型](custom-models/custom-models.md)

接下來，您將了解如何將自己的模型載入 Azure 遠端轉譯，並在您的應用程式中加以檢視。 [檢視從遠端轉譯的模型](view-remote-models/view-remote-models.md)是本教學課程的必要內容。

#### <a name="manipulating-models"></a>[操作模型](manipulate-models/manipulate-models.md)

在本教學課程中，您將操作並與遠端轉譯的模型互動。 建議您先完成[檢視遠端轉譯的模型](view-remote-models/view-remote-models.md)以及[介面和自訂模型](custom-models/custom-models.md)，再進行本教學課程。

#### <a name="refining-materials-lighting-and-effects"></a>[調整材質、光源和效果](materials-lighting-effects/materials-lighting-effects.md)

接下來，我們將討論一些重要的概念，像是修改材質、改變光源，以及將效果套用至您遠端轉譯的模型。 [檢視遠端轉譯的模型](view-remote-models/view-remote-models.md)、[介面和自訂模型](custom-models/custom-models.md)，以及[操作模型](manipulate-models/manipulate-models.md)都是本教學課程建議的必要內容。

#### <a name="securing-azure-remote-rendering-and-model-storage"></a>[保護 Azure 遠端轉譯和模型儲存體](security/security.md)

您將在此了解如何保護您的 Azure 遠端轉譯帳戶和 Blob 儲存體。 此內容涵蓋透過這些首要步驟來發行使用 Azure 遠端轉譯的商業應用程式。 本教學課程假設您已完成上述課程。


#### <a name="creating-a-commercial-ready-azure-remote-rendering-application"></a>[建立商業就緒的 Azure 遠端轉譯應用程式](commercial-ready/commercial-ready.md)

最後，我們將討論使用 Azure 遠端轉譯來準備商業應用程式的一些重要考量與秘訣。 如果您在簽出此資源之前已完成所有 Azure 遠端轉譯教學課程，會對您很有幫助。

**開始** [檢視遠端轉譯的模型](view-remote-models/view-remote-models.md)。
