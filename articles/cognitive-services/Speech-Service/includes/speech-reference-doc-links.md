## <a name="reference-docs"></a>參考文件

語音服務提供兩個 SDK。 第一個 SDK 是主要的[語音 SDK,](../speech-sdk.md)它提供了與語音服務交互所需的大部分功能。 第二個 SDK 特定於裝置,適當命名為[語音裝置 SDK](../speech-devices-sdk.md)。 兩種 SDK 都提供多種語言版本。

### <a name="speech-devices-sdk-reference-docs"></a>語音裝置 SDK 參考文件

[語音裝置 SDK](../speech-devices-sdk.md)是語音 SDK 的超集,具有特定設備的擴充功能。 要下載語音裝置 SDK,必須選擇[開發工具套件](../get-speech-devices-sdk.md#choose-a-development-kit)。
