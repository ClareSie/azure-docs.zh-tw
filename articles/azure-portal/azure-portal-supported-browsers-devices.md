# <a name="supported-devices"></a>支援的裝置

[Azure 門戶](https://portal.azure.com)是基於 Web 的主控台，在所有現代桌面和平板電腦設備的瀏覽器中運行。 如果需要從行動裝置管理 Azure 資源，請嘗試[Azure 移動應用](https://azure.microsoft.com/features/azure-portal/mobile-app/)。 它可用於 iOS 和安卓。

## <a name="recommended-browsers"></a>推薦的瀏覽器

建議使用與您的作業系統相容的最新瀏覽器。 支援下列瀏覽器：

* Microsoft Edge (最新版本)
* Internet Explorer 11
* Safari (最新版本，僅限 Mac)
* Chrome (最新版本)
* Firefox (最新版本)
