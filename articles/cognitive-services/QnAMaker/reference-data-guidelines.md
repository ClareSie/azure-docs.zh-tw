# <a name="import-and-export-data-reference"></a>匯入及匯出資料參考

查看此導入和匯出參考,以獲得知識庫備份、存儲和替換的最佳結果。

## <a name="import-and-export-knowledge-base"></a>進出口知識庫

**TSV 和 XLS 檔**(來自匯出的知識庫)只能透過從 QnA Maker 入口中的 **「設定」** 頁匯入檔案來使用。 在建立知識庫期間,也不能將其用作資料源,也不能從 **「設定」** 頁上的 **「添加檔案**」或 **「添加 URL」** 功能使用。
