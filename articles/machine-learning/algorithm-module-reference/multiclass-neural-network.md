# <a name="multiclass-neural-network-module"></a>多類神經網路模組

本文介紹 Azure 機器學習設計器（預覽）中的模組。

使用此模組建立類神經網路模型，以用來預測有多個值的目標。 

例如，此類神經網路可用於複雜的電腦視覺任務，如數位或字母識別、文檔分類和模式識別。

使用神經網路進行分類是一種受監督的學習方法，因此需要一個包含標籤列*的標記資料集*。

您可以通過提供模型和標記的資料集作為訓練模型的輸入來[訓練模型](./train-model.md)。 然後，訓練模型可用於預測新輸入示例的值。  

## <a name="about-neural-networks"></a>關於神經網路

神經網路是一組互連層。 輸入是第一層，並通過由加權邊和節點組成的環形圖連接到輸出圖層。

在輸入層和輸出圖層之間，可以插入多個隱藏圖層。 大部分的預測工作可以與一個或幾個隱藏層輕鬆完成。 然而，最近的研究表明，具有多層的深層神經網路（DNN）可以在圖像或語音辨識等複雜任務中有效。 連續層用於對語義深度增加的建模。

輸入和輸出之間的關係是從訓練輸入資料的神經網路中瞭解到的。 圖形的方向從輸入到隱藏圖層和輸出圖層。 圖層中的所有節點都通過加權邊緣連接到下一層中的節點。

要計算特定輸入的網路輸出，在隱藏圖層中的每個節點和輸出圖層中計算值。 該值是通過計算上一層節點值的加權總和來設置的。 然後，啟動函數應用於該加權總和。

## <a name="configure-multiclass-neural-network"></a>配置多類神經網路

1. 將**多類神經網路**模組添加到設計器中的管道中。 您可以在 **"機器學習**初始**化**"類別中找到此模組。 **Classification**

2. **創建培訓師模式**：使用此選項可指定如何訓練模型：

    - **單一參數**：如果您已經知道要如何配置模型，請選擇此選項。

    - **參數範圍**：如果不確定最佳參數，並且想要運行參數掃描，請選擇此選項。 選擇要反覆運算的值範圍，[調諧模型 Hyper 參數](tune-model-hyperparameters.md)反覆運算您提供的所有可能組合設置，以確定生成最佳結果的超參數。  

3. **隱藏層規範**：選擇要創建的網路架構的類型。

    - **完全連接的情況下**：選擇此選項可使用預設神經網路架構創建模型。 對於多類神經網路模型，預設值如下：

        - 一個隱藏層
        - 輸出層完全連接到隱藏層。
        - 隱藏圖層完全連接到輸入層。
        - 輸入層的節點數目取決於定型資料中的特徵數目。
        - 使用者可以設置隱藏層中的節點數。 預設值為 100。
        - 輸出層的節點數目取決於級別的數目。
  
   

5. **隱藏節點的數量**：此選項允許您自訂預設體系結構中的隱藏節點數。 鍵入隱藏節點的數量。 預設值為具有 100 個節點的隱藏圖層。

6. **學習速率**：在更正之前定義每次反覆運算所採取的步驟的大小。學習速率值越大，模型收斂速度越快，但會超過本地最小值。

7. **學習反覆運算次數**：指定演算法處理訓練案例的最大次數。

8. **初始學習權重直徑**：指定學習過程開始時的節點權重。

9. **動量**：指定在學習到以前反覆運算中的節點時要應用的權重。
  
11. **隨機播放示例**：選擇此選項可在反覆運算之間隨機排列事例。

    如果取消選擇此選項，則每次運行管道時都會以完全相同的連續處理案例。

12. **亂數種子**：鍵入用作種子的值，如果要確保在同一管道的運行中重複。

14. 連接訓練資料集和其中一個[培訓模組](module-reference.md)： 

    - 如果將 **"創建訓練器模式"** 設置為 **"單一參數**"，請使用[訓練器模型](train-model.md)。  
  

## <a name="results"></a>結果

培訓完成後：

- 要保存已訓練的模型的快照，請在 **"訓練"模型**模組的右側面板中選擇 **"輸出**"選項卡。 選擇 **"註冊"資料集**圖示以將模型保存為可重用模組。
