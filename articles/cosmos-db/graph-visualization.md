# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>使用資料視覺化解決方案視覺化存儲在 Azure 宇宙 DB Gremlin API 中的圖形資料

您可以使用各種資料視覺化解決方案來視覺化存儲在 Azure 宇宙 DB Gremlin API 中的資料。 [Apache Tinkerpop 社區](https://tinkerpop.apache.org/#poweredby)建議以下解決方案用於圖形資料視覺化。

## <a name="linkurious-enterprise"></a>連結企業

![連結徽標](./media/graph-visualization/linkurious-logo.jpg)

[Linkuris 企業](https://linkurio.us/product/)使用圖形技術和資料視覺化將複雜的資料集轉換為互動式視覺化網路。 該平臺連接到您的資料來源，使調查人員能夠無縫地跨數十億個實體和關係進行導航。 結果是，無需與查詢或表雜耍即可檢測可疑關係的新能力。

Linkurious 企業的互動介面提供了一種調查複雜資料的簡便方法。 您可以搜索特定實體、展開連接以發現隱藏關係，以及應用您選擇的佈局來解開複雜的網路。 連結企業現在與 Azure 宇宙 DB Gremlin API 相容。 它適用于端到端圖形視覺化方案，並支援使用者介面中的讀取和寫入功能。 您可以使用 Azure[宇宙 DB 請求連結的演示](https://linkurio.us/contact/)

![連結演示](./media/graph-visualization/linkurious-demo.gif)

## <a name="cambridge-intelligence"></a>劍橋情報

![劍橋智慧標誌](./media/graph-visualization/ci-logo.png)

[劍橋智慧的](https://cambridge-intelligence.com/products/)圖形視覺化檢視包現在支援 Azure 宇宙 DB。 Azure Cosmos DB 支援以下兩個視覺化檢視包：

- [JavaScript 開發人員的鍵線](https://cambridge-intelligence.com/keylines/)

- [為 React 開發人員重新繪製圖形](https://cambridge-intelligence.com/regraph/)

![劍橋智慧演示](./media/graph-visualization/ci-demo-2.gif)

這些工具組允許您為用例設計高性能圖形視覺化和分析應用程式。 它們利用強大的 Web 圖形庫 （WebGL） 渲染和精心設計的代碼，為使用者提供快速而有見地的視覺化體驗。 這些工具與任何瀏覽器、設備、伺服器或資料庫相容，並附帶分步教程、完整文檔記錄的 API 和互動式示範。

![劍橋智慧演示](./media/graph-visualization/ci-demo-1.gif)
