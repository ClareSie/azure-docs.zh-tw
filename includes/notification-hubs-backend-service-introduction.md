[ASP.NET Core Web API](https://dotnet.microsoft.com/apps/aspnet/apis) 後端的用途是處理用戶端的[裝置註冊](https://docs.microsoft.com/azure/notification-hubs/notification-hubs-push-notification-registration-management#what-is-device-registration)，並且會使用最新和最佳的[安裝](https://docs.microsoft.com/azure/notification-hubs/notification-hubs-push-notification-registration-management#installations)方法。 該服務也會以跨平台的方式傳送推播通知。 

這些作業會使用[適用於後端作業的通知中樞 SDK](https://www.nuget.org/packages/Microsoft.Azure.NotificationHubs/) 來處理。 如需有關整體方法的詳細資訊，請參閱[從應用程式後端進行註冊](https://docs.microsoft.com/azure/notification-hubs/notification-hubs-push-notification-registration-management#registration-management-from-a-backend)的文件。
