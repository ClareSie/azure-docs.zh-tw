# <a name="connect-azure-time-series-insights-gen-2-to-power-bi"></a>將 Azure 時間序列深入解析 Gen 2 連線至 Power BI  

您是否正在尋找解決方案，以協助將您的時間序列資料相互關聯、建立生動的視覺效果，並在整個組織中分享見解？

Azure 時間序列深入解析現在與 [Power BI](https://powerbi.microsoft.com/)緊密整合，提供您更強大的視覺效果和儀表板功能來處理串流資料，並可讓您在整個組織中共用見解和結果。

## <a name="summary"></a>摘要

   [![Azure 時間序列深入解析中的匯出選項](./media/concepts-connect-power-bi/tsi-power-bi-export-example.png)](./media/concepts-connect-power-bi/tsi-power-bi-export-example.png#lightbox)

藉由將您的資料連線到 Power BI，您現在可以：

* 使用 Power BI 所支援的其他資料來源來執行相互關聯，並存取不同資料視覺效果選項的主控制項。
* 使用您的時間序列深入解析資料來建立 Power BI 的儀表板和報表，並與您的組織共用。
* 以簡單、容易使用的方式來解除鎖定資料 interop 案例，並比以往更快取得見解。
* 使用強大的進階編輯器來修改 Power BI 內 Azure 時間序列深入解析資料。

## <a name="native-connector"></a>原生連接器

連接您的資料所需的時間不到一分鐘。 藉由將您的視圖從 Azure 時間序列深入解析 Explorer 快速匯出到 Power BI，您組織中的任何人都可以直覺地存取並以視覺化方式轉換資料。

設定選項包括在 **原始** 和 **匯總層級** 資料之間做選擇，並選取所需的時間範圍。

連接資料之後，就可以使用所有 Power BI 豐富的視覺效果和圖表工具，進一步增強您的資訊共用功能。

   [![Power BI 中的圖表範例](./media/concepts-connect-power-bi/power-bi-tsi-example.png)](./media/concepts-connect-power-bi/power-bi-tsi-example.png#lightbox)

如需逐步說明，請參閱 [如何將 Azure 時間序列深入解析連接到 Power BI](./how-to-connect-power-bi.md) 。

## <a name="flexible-workflow"></a>彈性的工作流程

在工作流程中的任何位置使用連接器：

* 如果您在 Azure 時間序列深入解析 Explorer 中，可以將您的視圖快速匯出到 Power BI。
* 如果您已經在 Power BI 中，請使用 Azure 時間序列深入解析來查詢資料，以增強現有的視覺效果案例。

如需 Power BI 中的先進查詢和編輯功能，請使用 Power BI 的進階編輯器。
