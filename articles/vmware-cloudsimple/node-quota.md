# <a name="cloudsimple-node-quota-limits"></a>CloudSimple 節點配額限制

當您的訂用帳戶已啟用 CloudSimple 服務時，四個節點是可供購買的預設數量。  您可以從 Azure 入口網站購買任何 [節點類型](cloudsimple-node.md) 。  至少需要三個相同 SKU 的節點，才能建立私人雲端。  如果您已購買節點，當您嘗試購買其他節點時，可能會看到錯誤。

## <a name="quota-increase"></a>配額增加

您可以藉由提交支援要求來增加節點配額。 服務作業小組將會評估要求，並與您合作來增加節點配額。  當您開啟新的票證時，請選取下列選項：

* 問題類型： **技術**
* 訂用帳戶：**您的訂**用帳戶識別碼
* 服務類型： **VMware Solution By CloudSimple**
* 問題類型： **專用節點配額**
* 問題子類型： **增加專用節點的配額**
* 主體： **配額增加**

在支援票證的詳細資料中，提供必要的節點數目和節點 SKU。

* 節點 SKU
* 您正在要求增加配額的額外節點數目
