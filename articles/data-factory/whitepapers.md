# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory 白皮書 

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

白皮書可讓您探索 Azure Data Factory 更深入的層級。 本文提供 Azure Data Factory 的可用白皮書清單。

| **白皮書** | **描述** |
| --- | --- |
|[Azure Data Factory-雲端中的資料整合](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | 本檔將說明 Azure Data Factory 如何讓您建立新式資料倉儲、啟用 advanced analytics 來推動智慧型 SaaS 應用程式，並將您的 SQL Server 整合服務套件隨即轉移至 Azure。|
|[使用 Azure Data Factory 將資料從內部部署的關聯式資料倉儲遷移至 Azure](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | 這份檔說明從現有的內部部署關聯式資料倉儲遷移數十 TB 資料的複雜度 (例如，Netezza、Oracle、Teradata、SQL server) 至 Azure (例如，Blob 儲存體或使用) Azure Data Lake Storage Azure Data Factory。 這些挑戰和最佳作法是以 Azure Data Factory 的大規模資料內嵌旅程的彈性、效能、擴充性、管理及安全性為說明。 |
|[使用 Azure Data Factory 的 SAP 資料整合](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | 本白皮書摘要說明 Azure Data Factory 目前對 SAP 資料整合的支援，包括目標案例、SAP 連接器選項以及不同需求的比較，以及 Data Factory 中每個 SAP 連接器的簡介。 | 
|[Azure Data Factory：雲端中的 SSIS](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| 這份檔將說明您想要將現有的 SSIS 工作負載遷移到 Azure Data Factory，並解決常見的考慮和顧慮的原因。 接著，我們將逐步引導您瞭解建立 Azure-SSIS IR 的技術詳細資料，然後示範如何使用您熟悉的工具（例如 SQL Server Management Studio (SSMS) ），透過 Azure Data Factory 來上傳、執行及監視套件。 |
|[Azure data Factory –傳遞參數](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| 本文將逐步引導您瞭解如何在管線和活動之間以及活動之間傳遞參數。 |
|[Azure data Factory – DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| 本白皮書涵蓋 Azure Data Factory 持續整合和部署方面的一些最佳作法。 |
