# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>設置支援資訊（Azure 實驗室服務中的實驗室帳戶擁有者）
本文介紹您（作為實驗室帳戶擁有者）如何指定支援資訊，如果實驗室建立者（講師）和實驗室使用者（學生）在使用實驗室帳戶或實驗室時遇到任何技術問題，他們可以使用這些資訊獲取説明。 

支援資訊包括：

- URL
- 電子郵件
- 電話
- 其他指示 

## <a name="specify-support-information"></a>指定支援資訊
1. 登錄到[Azure 門戶](https://portal.azure.com)。
2. 在搜索欄中，輸入**實驗室服務**，並在搜尋結果中選擇 **"實驗室服務**"。 
3. 從實驗室帳戶清單中選擇您的實驗室帳戶。 
4. 切換到 **"內部支援**"頁，執行以下步驟：
    1. 輸入**支援 URL**。 
     2. 輸入**支援電子郵件**。 
     3. 輸入**支援電話**。
     4. 輸入詳細的**支援說明**（可選）。 實驗室擁有者和使用者將看到此文本以及支援聯繫資訊。 URL 將自動轉換為連結。 
     5. 在工具列上選取 [儲存]****。
