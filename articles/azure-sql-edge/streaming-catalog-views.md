# <a name="streaming-catalog-views-transact-sql"></a>串流目錄檢視 (Transact-SQL)

本節包含與 T-SQL 串流相關的可用目錄檢視和函式。
  
## <a name="in-this-section"></a>本節內容  
  
|檢視|描述|  
|:---|:---|
|[sys.external_streams](sys-external-streams.md) |針對在資料庫範圍內建立的每個外部串流物件傳回一個資料列。|
|[sys.external_streaming_jobs](sys-external-streaming-jobs.md) |針對在資料庫範圍內建立的每個外部串流作業傳回一個資料列。|
|[sys.external_job_streams](sys-external-job-streams.md)|針對對應至外部串流作業的輸入或輸出外部串流物件各傳回一個資料列。|
