# <a name="deprecated-configure-baidu-cloud-push-settings-for-a-notification-hub-in-the-azure-portal"></a>已棄用：為 Azure 門戶中的通知中心配置百度雲推送設置

本文介紹如何使用 Azure 門戶為 Azure 通知中心配置百度雲推送設置。

> [!IMPORTANT]
> 本教程已棄用。 

## <a name="prerequisites"></a>Prerequisites
如果您尚未建立通知中樞，立即建立一個。 如需詳細資訊，請參閱[在 Azure 入口網站中建立 Azure 通知中樞](create-notification-hub-portal.md)。 

## <a name="configure-baidu-cloud-push"></a>設定百度雲端推播
以下過程為您提供了為通知中心配置百度雲推送設置的步驟：

1. 在 Azure 門戶中，在**通知中心**頁面上，選擇左側功能表上的**百度（安卓中國）。** 
2. 在百度雲推送專案中，輸入從百度主控台取得的 **API 金鑰**。 
3. 在百度雲推送專案中，輸入從百度主控台取得的**祕密金鑰**。 
4. 選取 [儲存]****。
