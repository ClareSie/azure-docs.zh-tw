<!-- Create a resource group -->

使用下列命令來建立資源群組。 選取將用來儲存媒體服務帳戶之媒體和元資料記錄的地理區域。 此區域將用於處理和串流媒體。

```azurecli
az group create --name amsResourceGroup --location westus2
```
