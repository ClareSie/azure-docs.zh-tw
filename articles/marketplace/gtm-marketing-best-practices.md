# <a name="marketing-best-practices"></a>行銷最佳做法

優化的市場上市和上市活動可説明您加快客戶獲取增長,從而在雲業務戰略和引擎中發揮重要作用。 準備好列出優惠,提供客戶試用,並與 Microsoft 客戶和合作夥伴社區建立聯繫。 下載[AppSource & Azure 應用商店行銷最佳實踐指南](https://aka.ms/marketplacebestpractices),瞭解如何充分利用在線行銷工作。

**後續步驟**

若要深入了解 Marketplace GTM 服務，請移至 [Go-to-market Services](https://partner.microsoft.com/reach-customers/gtm)。

登錄到[合作夥伴中心](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/partnership)以創建和配置您的產品/服務。
