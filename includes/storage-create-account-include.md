## <a name="create-an-azure-storage-account"></a>建立 Azure 儲存體帳戶

建立您第一個 Azure 儲存體帳戶最簡單的方法，就是使用 [Azure 入口網站](https://portal.azure.com)。 若要深入了解，請參閱 [建立儲存體帳戶](../articles/storage/common/storage-account-create.md)。

您也可以使用 [Azure PowerShell](/powershell/module/az.storage/)、[Azure CLI](../articles/storage/blobs/storage-quickstart-blobs-cli.md)，或 [Azure Storage Resource Provider for .NET](https://azure.microsoft.com/resources/samples/storage-dotnet-resource-provider-getting-started/) 來建立 Azure 儲存體帳戶。

如果您目前不想要在 Azure 中建立儲存體帳戶，您也可以使用 Azurite 儲存體模擬器在本機環境中執行並測試您的程式碼。 如需詳細資訊，請參閱 [使用 Azurite 模擬器進行本機 Azure 儲存體開發](../articles/storage/common/storage-use-azurite.md)。
