> [!TIP]
> 儘管 JavaScript 語音 SDK 作為 npm 套件提供,因此 Node.js 和用戶端 Web 瀏覽器都可以使用它 - 考慮每個環境的各種體系結構含義。 例如,<a href="https://en.wikipedia.org/wiki/Document_Object_Model" target="_blank">文件物件<span class="docon docon-navigate-external x-hidden-focus"></span>模型 (DOM)</a>不適用於伺服器端應用程式,就像<a href="https://nodejs.org/api/fs.html" target="_blank">檔案系統<span class="docon docon-navigate-external x-hidden-focus"></span></a>對用戶端應用程式不可用一樣。

### <a name="nodejs-package-manager-npm"></a>Node.js 套件管理員 (NPM)

要安裝 JavaScript 語音 SDK,請`npm install`執行以下命令。

```nodejs
npm install microsoft-cognitiveservices-speech-sdk
```

有關詳細資訊,請參閱<a href="https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/javascript/node" target="_blank">Node.js 語音 SDK 快速<span class="docon docon-navigate-external x-hidden-focus"></span>入門</a>。
