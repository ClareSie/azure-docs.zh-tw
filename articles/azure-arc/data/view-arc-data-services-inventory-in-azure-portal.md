# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>在 Azure 入口網站中查看實例的清查

上傳您的 [計量、記錄](upload-metrics-and-logs-to-azure-monitor.md)或 [使用量](view-billing-data-in-azure.md)之後，您可以從 Azure 入口網站中，將您的實例視為 Azure 資源來查看。 若要查看 Azure 入口網站中的資源，請移至特殊 URL [https://portal.azure.com](https://portal.azure.com) 並完成下列步驟：

1. 移至 [ **所有服務**]。
1. 搜尋您的資料庫實例類型。
1. 將類型新增至 [我的最愛]。
1. 在左側功能表中，選取實例類型。
1. 在與其他 Azure SQL 或 Azure 于 postgresql 超大規模資源相同的視圖中，查看您的實例 (使用篩選來進行細微的查看) 。
