# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS 保護

內容傳遞網路依設計提供 DDoS 保護。 除了可吸收體積型攻擊的全球容量之外，Azure CDN 還有其他的 DDoS 保護（如下所述），不需額外費用。

## <a name="azure-cdn-from-microsoft"></a>來自 Microsoft 的 Azure CDN

來自 Microsoft 的 azure CDN 受 [Azure 基本 DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview)保護。 根據預設，它會整合到 Microsoft 平臺的 Azure CDN，而且不會產生額外費用。 從 Microsoft 的全球部署網路，Azure CDN 的完整規模和容量可讓您透過永遠可用的流量監視和即時風險降低來防禦常見的網路層攻擊。 基本的 DDoS 保護也可以抵禦最常見、經常發生的第7層 DNS 查詢氾濫，以及以 CDN 端點為目標的第3層和第4體積型攻擊。 這項服務也有經過證實的追蹤記錄，可保護 Microsoft 的企業和取用者服務免于大規模的攻擊。

## <a name="azure-cdn-from-verizon"></a>Azure CDN from Verizon

來自 Verizon 的 Azure CDN 受 Verzion 專屬 DDoS 緩和平臺的保護。 依預設，它會從 Verizon 整合到 Azure CDN，且不會產生額外費用。 它可針對最常見、經常發生的第7層 DNS 查詢氾濫，以及以 CDN 端點為目標的第3層和第4體積型攻擊提供基本保護。

## <a name="azure-cdn-from-akamai"></a>Azure CDN from Akamai

來自 Akamai 的 Azure CDN 受 Akamai 專屬 DDoS 緩和平臺的保護。 依預設，它會從 Akamai 整合到 Azure CDN，且不會產生額外費用。 它可針對最常見、經常發生的第7層 DNS 查詢氾濫，以及以 CDN 端點為目標的第3層和第4體積型攻擊提供基本保護。
