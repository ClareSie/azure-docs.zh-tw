# <a name="managing-your-academic-grant"></a>管理您的學術授與

您在 Azure Education Hub 中的主要登陸頁面是 [總覽] 頁面。 此頁面包含您學術授與的所有相關資訊，例如您已建立的類別數目，以及配置及從這些類別使用的總點數。 它也會顯示 Education Hub 所提供的其他好處的快捷方式：軟體、學習和快速入門範本。

## <a name="overview-page"></a>概觀分頁
:::image type="content" source="media/hub-overview-page/overview.png" alt-text="登入錯誤訊息。" border="false":::

1. **課程**顯示講師帳戶的使用中課程總數。
1. **實驗室**會顯示已向學生傳來的作用中實驗室總數。
1. **必要動作**會列出您需要完成的任何動作，例如，接受實驗室邀請。
1. **軟體**列出免費的軟體，可供教師下載。
1. **瞭解**可透過 Microsoft Learn 取得的免費 Azure 學習路徑連結。
1. **快速入門範本**包含 Azure 範本，可協助加速並簡化一般工作的部署作業。
