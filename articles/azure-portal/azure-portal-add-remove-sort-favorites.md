# <a name="add-remove-and-rearrange-favorites"></a>添加、刪除和重新排列我的最愛

從**我的最愛**清單中添加或刪除專案，以便快速轉到最常用的服務。 我們已經將一些常見服務添加到您的**我的最愛**清單中，但您可能需要對其進行自訂。 你是唯一一個看到你對**我的最愛**所做的改變的人。

## <a name="add-a-favorite"></a>添加我的最愛

"**我的最愛"** 下列出的專案將從**所有服務**中選擇。 將滑鼠懸停在服務名稱上，以顯示與服務相關的資訊和資源。 服務名稱旁邊的填充![星形圖示](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-graystar.png)"填充星形"圖示表示該專案顯示在**我的最愛**清單中。 選擇星形圖示以將服務添加到**我的最愛**清單中。

### <a name="add-cost-management--billing-to-favorites"></a>將成本管理 + 計費添加到我的最愛

1. 從 Azure 門戶功能表中選擇**所有服務**。

    ![顯示所有選定服務的螢幕截圖](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-new-all-services.png)

1. 在搜索欄位中輸入單詞"成本"。 將顯示標題中具有"成本"或關鍵字為"成本"的服務。

   ![顯示所有服務中的搜索的螢幕截圖](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-find-service.png)

1. 將滑鼠懸停在服務名稱上，以顯示**成本管理和計費**資訊卡。 選擇星形圖示。

   ![螢幕截圖顯示成本管理旁邊的明星 + 已選擇計費](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-add.png)

1. **成本管理 + 計費**現在添加為我的最愛清單中的最後**一**項。

## <a name="remove-an-item-from-favorites"></a>從我的最愛中刪除專案

您現在可以直接從**我的最愛**清單中刪除專案。

1. 在門戶功能表的 **"我的最愛"** 部分中，將滑鼠懸停在要刪除的服務的名稱上。

   ![在我的最愛中顯示懸停行為的螢幕截圖](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-remove.png)

2. 在資訊卡上，選擇星形，使其從填充變為未填充。 該服務將從**我的最愛**清單中刪除。

## <a name="rearrange-favorites"></a>重新排列我的最愛

您可以更改列出您最喜愛的服務的順序。 只需將功能表項目拖放到**我的最愛**下的另一個位置。

### <a name="move-cost-management--billing-to-the-top-of-favorites"></a>將成本管理和計費移動到我的最愛的頂部

1. 在**我的最愛**清單中選擇並保留**成本管理和計費**條目。

   ![顯示成本管理和所選計費的螢幕截圖](./media/azure-portal-add-remove-sort-favorites/azure-portal-favorites-sort.png)

1. 繼續按住時，將專案拖動到**我的最愛**的頂部，然後釋放。
