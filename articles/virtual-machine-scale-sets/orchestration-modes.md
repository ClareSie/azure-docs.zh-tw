# <a name="orchestration-mode-preview"></a>編排模式（預覽）

虛擬機器縮放集提供平臺管理的虛擬機器的邏輯分組。 使用規模集，您可以創建虛擬機器配置模型，根據 CPU 或記憶體負載自動添加或刪除其他實例，並自動升級到最新的作業系統版本。 傳統上，縮放集允許您使用在創建規模集時提供的 VM 配置模型創建虛擬機器，並且規模集只能管理基於配置模型隱式創建的虛擬機器。

使用比例集業務流程模式（預覽），現在可以選擇規模集是否應協調在規模集配置模型之外顯式創建的虛擬機器，還是基於配置模型。 縮放集業務流程模式還通過在容錯域和可用性區域中部署這些 VM，説明您設計 VM 基礎結構，以獲得高可用性。


虛擬機器規模集將支援 2 種不同的業務流程模式：

- ScaleSetVM – 添加到規模集的虛擬機器實例基於規模集配置模型。 虛擬機器實例生命週期 - 創建、更新、刪除 - 由規模集管理。
- VM（虛擬機器） – 可以在規模集之外創建的虛擬機器顯式添加到規模集中。 
 

> [!IMPORTANT]
> 在創建比例集時定義業務流程模式，以後無法更改或更新。 
> 
> 虛擬機器縮放集的此功能當前處於公共預覽版中。
> 此預覽版本是在沒有服務等級協定的情況下提供，不建議用於生產工作負載。 可能不支援特定功能，或可能已經限制功能。 
> 如需詳細資訊，請參閱 [Microsoft Azure 預覽版增補使用條款](https://azure.microsoft.com/support/legal/preview-supplemental-terms/)。


## <a name="orchestration-modes"></a>協調流程模式

|                             | "編排模式"："虛擬機器"（虛擬機器） | "編排模式"："縮放集VM"（虛擬機器縮放設置VM） |
|-----------------------------|--------------------------------------------|--------------------------------------------------------------|
| VM 配置模型      | None                                       | 必要 |
| 將新的 VM 添加到縮放集  | 創建 VM 時，VM 會顯式添加到規模集中。 | VM 會根據 VM 配置模型、實例計數和自動縮放規則隱式創建並添加到縮放集 | |
| 刪除 VM                   | 必須單獨刪除 VM，如果擴展設置中有任何 VM，則不會刪除其規模集。 | VM 可以單獨刪除，刪除規模集將刪除所有 VM 實例。  |
| 附加/分離 VM           | 不支援                              | 不支援 |
| 實例生命週期（通過刪除創建） | VM 及其工件（如磁片和 NIC）可以獨立管理。 | 實例及其工件（如磁片和 NIC）隱式到創建實例的規模集實例中。 它們不能分離或單獨管理在比例集之外 |
| 容錯網域               | 可以定義容錯域。 2 或 3 基於區域支援，5 個用於可用區域。 | 可以定義從 1 到 5 的容錯域 |
| 更新網域              | 更新域將自動映射到容錯域 | 更新域將自動映射到容錯域 |
| 可用性區域          | 在一個可用性區域中支援區域部署或 VM | 支援區域部署或多個可用區域;可以定義區域平衡策略 |
| 自動調整                   | 不支援                              | 支援 |
| 作業系統升級                  | 不支援                              | 支援 |
| 模型更新               | 不支援                              | 支援 |
| 實例控制            | 完全 VM 控制。 VM 具有完全合格的 URI，支援 Azure VM 管理功能的一系列（如 Azure 策略、Azure 備份和 Azure 網站恢復） | VM 是比例集的從屬資源。 只能通過規模集訪問實例以進行管理。 |
| 實例模型              | 微軟.計算/虛擬機器模型定義。 | 微軟.計算/虛擬機器規模集/虛擬機器模型定義。 |
| Capacity                    | 可以創建空比例集;例如，可以創建空比例集。最多 200 個 VM 可以添加到比例集中 | 可以使用實例計數 0 - 1000 定義縮放集 |
| 移動                        | 支援                                  | 支援 |
| 單個放置組 = 假 | 不支援                          | 支援 |
