# <a name="enrolling-students-in-azure-dev-tools-for-teaching"></a>在 Azure Dev Tools for Teaching 中註冊學生
購買 Azure Dev Tools for Teaching 之後，位於您機構網域的所有學生都會自動取得此供應項目的存取權。 在購買過程中，系統會要求您輸入訂用帳戶管理員的電子郵件地址。 這個經過大學驗證的網域將會使用相同的電子郵件網域來將學生解除鎖定，並授與他們登入和兌換軟體下載權益的能力。 如果學生尚未擁有 Microsoft 帳戶 (MSA)，或屬於大學 Active Directory 的一部分，系統會提示他們建立符合其大學電子郵件的帳戶。 如果學生有關於存取 Education Hub 的問題，請讓他們先看過以下的「疑難排解指南」，或是透過 [https://aka.ms/studentsupport](https://aka.ms/studentsupport) 與我們連絡。

## <a name="managing-access-for-students"></a>管理學生的存取權
您可以根據學生用來登入 Azure Dev Tools for Teaching 的網域來管理他們的存取權。 進入 Azure Dev Tools for Teaching 的[管理入口網站](https://azureforeducation.microsoft.com/en-us/account/Subscriptions)。 您會在那裡看到所有的訂用帳戶。 選擇您要管理其存取權的訂用帳戶。
