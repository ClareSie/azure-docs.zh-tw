# <a name="toggle-live-chat"></a>切換即時聊天

本"如何"一文介紹如何在 IoT 中央應用程式中切換即時聊天。 您可以使用即時聊天訪問技術支援。

> [!NOTE]
> 聊天選項僅適用于使用免費定價計畫創建的應用程式。

## <a name="chat-with-us"></a>與我們聊天

要獲得技術支援，請打開 IoT 中心應用程式，然後選擇 **"與我們聊天**"。

![與我們聊天](media/howto-show-hide-chat/chat-with-us.png)

您可以鍵入問題，如以下螢幕截圖所示：

![含設定和屬性詳細資料的 [設定裝置詳細資料] 表單](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>隱藏聊天

要隱藏聊天，請選擇 **"説明**"面板中的 **"隱藏聊天**"：

 ![隱藏聊天](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>啟用聊天

要顯示聊天，請選擇 **"説明"** 面板中的 **"顯示聊天**"：

 ![顯示聊天](media/howto-show-hide-chat/show-chat.png)
