> [!NOTE]
> 在 2021 年 12 月，18.6 以前的 SSMS 版本無法再透過 Azure Active Directory 進行 MFA 驗證。
>
> 若要繼續使用 Azure Active Directory 進行 MFA 驗證，您需要 [SSMS 18.6 或更新版本](https://docs.microsoft.com/sql/ssms/download-sql-server-management-studio-ssms)。
