## <a name="icsa-labs-corporate-firewall-certification"></a>ICSA 實驗室企業防火牆認證

![ICSA 認證](media/overview/icsa-cert-firewall-small.png)

ICSA Labs 是安全和健康 IT 產品以及網路連接設備的協力廠商測試和認證的領先供應商。 它們衡量大多數世界頂級技術供應商的產品合規性、可靠性和性能。

Azure 防火牆是第一個獲得 ICSA 實驗室企業防火牆認證的雲防火牆服務。 有關 Azure 防火牆認證報告，請參閱[ICSA 實驗室認證測試和審核報告](https://aka.ms/ICSALabsCertification)。 有關詳細資訊，請參閱[ICSA 實驗室防火牆認證計畫](https://www.icsalabs.com/technology-program/firewalls)頁面。
