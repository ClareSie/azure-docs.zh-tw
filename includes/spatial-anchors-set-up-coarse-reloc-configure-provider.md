## <a name="configure-the-sensor-fingerprint-provider"></a>設定感應器指紋提供者

首先，我們會建立及設定感應器指紋提供者。 感應器指紋提供者會負責讀取您裝置上的平台特定感應器，並將其讀數轉換為雲端空間錨點工作階段所取用的常見表示法。
