# <a name="orders-dashboard-in-commercial-marketplace-analytics"></a>商業市集分析中的訂單儀表板

本文提供合作夥伴中心內**訂單儀表板**的相關資訊。 此儀表板會以圖形化及可下載的格式顯示訂單相關資訊。

若要存取合作夥伴中心分析工具中的 [訂單儀表板]，請開啟商業市集底下的 **[[分析儀表板]](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** 。

>[!NOTE]
> 如需分析術語的詳細定義，請參閱[商業市集分析的常見問題集與術語](./faq-terminology.md)。

## <a name="orders-dashboard"></a>訂單儀表板

[分析] 功能表的 [訂單儀表板] 會顯示所有 SaaS 供應項目的目前訂單。 您可檢視下列項目的圖形表示：

- [訂單摘要](#order-summary)
- [依地理位置排序訂單](#orders-by-geography)
- [依供應項目排序訂單](#orders-by-offers)
- [訂單趨勢 (按照站台與按照基座的比較)](#orders-trend-per-site-versus-per-seat)
- [依方案的訂單](#orders-by-plans)
- [訂單和基座趨勢](#orders-and-seats-trend)
- [訂單詳細資料表](#order-details-table)

在合作夥伴中心建立和報告訂單之間的最大延遲為48小時。

## <a name="order-dashboard-details"></a>訂單儀表板詳細資料

本節將詳述分析報告。

### <a name="order-summary"></a>訂單摘要

訂單摘要區段會顯示所有採購訂單的計數 (不含已取消訂單)、已取消訂單和基座。

總訂單旁的百分比值代表所選日期範圍的成長量。

![合作夥伴中心分析訂單摘要](./media/order-summary.png)

- 向上指的綠色三角形表示正成長趨勢。
- 向下指的紅色三角形表示相對於上個月的負成長趨勢。
- 成長趨勢會以微型橫條圖表示。 您可以將滑鼠停留在圖表內的資料行上方，以顯示每個月的值。
- 已取消訂單是先前購買並在所選日期範圍內取消的訂單計數。
- 基座是所選日期範圍內建立的基座計數。

### <a name="orders-by-geography"></a>依地理位置排序訂單

[**依地理位置的訂單**] 熱度圖會顯示您在世界地圖上的訂單計數，並顯示根據客戶國家/地區所對應的基座。 此熱度圖的運作方式與 **[依地理位置排序的客戶熱度圖](./customer-dashboard.md#customer-by-geography)** 功能相同。

![依地理位置排序的合作夥伴中心分析訂單](./media/orders-by-geography.png)

### <a name="orders-by-offers"></a>依供應項目排序訂單

**依供應項目排序訂單**環圈圖會根據其供應項目名稱來組織訂單 (包含已取消訂單)。

- 主要的供應項目會顯示在圖形中，剩餘的供應項目則會分組為「全部待用」。
- 您可以選取圖例中的特定供應項目，以在圖形中只顯示這些供應項目。
- 將滑鼠停留在圖形上的配量時，會顯示該供應專案的訂單數目和百分比，與您跨所有供應專案的總訂單數相較。
- **依供應項目趨勢排序的訂單**會顯示每月成長趨勢。 月份資料行代表依供應項目名稱排序的訂單數目。 折線圖會顯示在 Z 軸上繪製的成長百分比趨勢。
- 您可以使用圖表頂端的滑杆，沿著 X 軸向右和向左捲動，並將焦點放在特定資料點。
- 您可以選取圖例上的特定項目來顯示趨勢圖表。
- 你也可以選擇顯示**已取消訂單**的趨勢和資料。 此圖形運作方式會與**依供應項目排序訂單**圖形提供的相同。

### <a name="orders-trend-per-site-versus-per-seat"></a>訂單趨勢 (按照站台與按照基座的比較)

[**每個網站與每個**基座的環圈圖] 表示客戶購買的每個網站 saas 和每一基座 saas 訂單的細目（此圖表包含已取消的訂單）。 直條圖代表客戶所購買的每個站台 SaaS 與每個基座 SaaS 訂單趨勢 (此圖表包含已取消訂單)。

### <a name="orders-by-plans"></a>依方案的訂單

[**依方案排序**] 圖表代表您所有供應專案之計畫層級的訂單趨勢（這包括已取消的訂單）。 環圈圖代表前五個計畫訂單的明細，而直條圖代表前五個計畫的訂單趨勢。

### <a name="orders-and-seats-trend"></a>訂單和基座趨勢

**訂單和基座趨勢**圖表會以最高訂單數來呈現前 50 個供應項目。 這些會顯示在領導者面板上，並依照最高的訂單計數和訂單百分比進行排序。

- **依方案排序**–選取供應專案以查看圖表中前五個計畫的訂單計數明細。
- **依方案**的基座–前五個計畫的每月基座趨勢。 如果您選取的供應專案不是每個基座的供應專案，這裡就不會顯示任何資料。

### <a name="canceled-orders-by-offers"></a>依供應項目排序的取消訂單

**依供應項目排序的取消訂單**圓形圖會依據其供應項目名稱來組織所有您已取消的訂單。 主要的供應項目會顯示在圖形中，剩餘的供應項目則會分組為「全部待用」。 您可以選取圖例中的特定供應項目，以將其在圖形中顯示出來。

- 將滑鼠停留在圖形上的配量時，會顯示與所有供應專案的訂單總數相比，所選供應專案的訂單數目和百分比。
- 直條圖會顯示每月趨勢。 資料行代表依供應項目名稱排序的已取消訂單數目。 您可以使用圖表頂端的滑杆，沿著 X 軸向右和向左捲動，並將焦點放在特定資料點。 您可以選取圖例上的特定項目來顯示趨勢圖表。

### <a name="order-details-table"></a>訂單詳細資料表

訂單詳細資料表會顯示依取得日期排序的前 1000 項訂單的編號清單。

- 方格中的每個資料行皆可排序。
- 如果記錄計數小於 1000 項，則可將資料擷取至 TSV 檔案。
- 如果記錄數目超過 1000，則匯出的資料會在接下來 30 天內以非同步方式放置在下載頁面中。
- 將篩選套用至**Order details 資料表**，只顯示您感興趣的資料。 依國家/地區、Azure 授權類型、商業 marketplace 授權類型、供應專案類型、訂單狀態、免費線索、商業 marketplace 訂用帳戶識別碼、客戶識別碼和公司名稱進行篩選。
- 因為 SaaS 透過 Azure Marketplace 或 AppSource 購買，不需要 Azure 訂用帳戶，所以 Marketplace 訂用帳戶識別碼在**詳細訂單資料**區段中會顯示為00000000-0000-0000-0000-000000000000。

#### <a name="orders-page-filters"></a>訂單頁面篩選器

這些篩選器會在頁面層級中套用。

您可以選取多個篩選器來轉譯您選擇要檢視的準則圖表，以及要在 [詳細訂單資料] 方格/匯出中顯示的資料。 篩選會針對您在 [訂單] 頁面右上角選取的資料範圍，套用到解壓縮的資料上。

- 供應項目類型和名稱只會在所選日期範圍內，針對您有訂單的供應項目列出。 清單中的供應項目名稱則會針對您在清單中選取的供應項目類型顯示。
- 套用的篩選會在每個選取的篩選器中，顯示每個選取範圍內的計量總數。 選擇預設選取項目則不會顯示套用的篩選。
- 如果為其中一個下拉式清單選取 [全部]，則會匯總所選取頁面中的所有計量。 例如：供應項目類型篩選選項中的「全部」表示已選取所有供應項目類型。 此為下拉式清單的預設選項。 選取 [全部] 時，套用的篩選顯示不會顯示任何內容。
- **多重值選取項目**：針對下拉式清單下的所有選取項目，系統會匯總頁面中的所有計量。 如果已選擇多項選擇項目，套用的篩選會顯示所有已選取項目的計數。 請見下圖以供參考。

    ![合作夥伴中心分析訂單內含多個套用至篩選的值](./media/filters-applied.png)

- **單一值選取項目**：如果您選取了單一的值，套用的篩選會顯示所選取的篩選計數。 請見下方影像以供參考。

     ![合作夥伴中心訂單內含單一套用至篩選的值](./media/filters-applied-single.png)
