# <a name="authoring-api-changes"></a>創作 API 更改 

為 V3 更新了創作 API，以便[使用機器學習的實體進行模型分解](luis-concept-data-extraction.md)。 

您可以在 V2 中創作或預覽 V3 APIIS。 同一應用也可以選擇使用 V2 或 V3 預測終結點回應。 不必具有相同的 API 版本，用於創作和預測終結點回應。

LUIS 門戶僅使用 V3 創作 API。
