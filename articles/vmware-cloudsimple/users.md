# <a name="view-the-list-of-cloudsimple-portal-users"></a>查看 CloudSimple portal 使用者的清單

使用者第一次存取 CloudSimple 入口網站時，會新增至使用者清單。 若要查看可透過 Azure 存取 CloudSimple 入口網站的使用者清單，請[存取 CloudSimple 入口網站](access-cloudsimple-portal.md)，選取側邊功能表上的 [**帳戶**]，然後在 CloudSimple 入口網站中選取 [**使用者**]。

* 若要顯示使用者詳細資料（包括 Azure 訂用帳戶、租使用者和使用者識別碼），請按一下 [**使用者**] 頁面上的專案。

* 若要查看使用者活動的審核記錄，請選取 [**審核記錄**] 索引標籤。
* 若要鎖定或解除鎖定使用者帳戶，請在顯示使用者詳細資料時，按一下 [**鎖定**] 切換。 當帳戶解除鎖定時，使用者可以存取 CloudSimple 入口網站。 當帳戶被鎖定時，就會封鎖對入口網站的存取。
