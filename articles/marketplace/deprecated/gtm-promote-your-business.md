# <a name="promote-your-business-with-microsoft"></a>透過 Microsoft 推廣您的業務

Marketplace 夥伴必須是 Microsoft 合作夥伴網路 (MPN) 的一部分。 作為 MPN 成員，您可以存取其他商業導向的優點。 請瀏覽 [partner.microsoft.com](https://partner.microsoft.com)，了解如何使用網路存取：

* 新的商機。
* 與小組或夥伴聯繫。
* 可協助提升您技能的解決方案和訓練。

## <a name="benefits-and-resources"></a>權益及資源

若要推廣您的供應項目或應用程式，請使用下表來存取更多權益和資源：

| 優點 | 資源 |
|:--- |:--- |
| 使用您的核心權益 | 加入 Microsoft 合作夥伴網路，您就會收到一組核心權益，幫助您節省時間和金錢。 權益可協助您 <ul> <li> 加強您的能力、 </li> <li> 滿足客戶，並且 </li> <li> 建立聯繫以充分發揮業務潛力。 </li> </ul> 如需核心權益的詳細資訊，請參閱[核心權益與要求條件](https://partner.microsoft.com/membership/core-benefits#tab-content-1)。 |
| 獲得雲端平台專長認證 | 獲得專長認證有助於讓您 <ul> <li> 從 Microsoft 合作夥伴中脫穎而出。 </li> <li> 在市場中展現技術專業能力和客戶成功案例。 </li> </ul> 專長認證是許多重要合作夥伴方案 (例如「共同銷售」) 的必要條件。 如需詳細資訊，請參閱[雲端平台專長認證](https://partner.microsoft.com/reach-customers/selling-with-microsoft)。 |
| IP 共同銷售準備就緒 | 您可能有資格參加 IP 共同銷售準備就緒方案。  此方案可讓您： <ul> <li> 此方案可讓您和 Microsoft 銷售人員與其他合作夥伴直接合作，爭取目標客戶商機和進行帳戶規劃。 </li> <li> 在銷售人員解決方案目錄中向 Microsoft 銷售人員顯示您的解決方案。 </li> <li> 與您合作並贏得交易的 Microsoft 銷售人員會獲得獎勵。 </li> </ul> 如需有關包括資格需求在內的詳細資訊，請參閱[共同銷售準備就緒計劃和推廣您的業務](https://partner.microsoft.com/reach-customers/selling-with-microsoft)。 |

## <a name="next-steps"></a>後續步驟

若要深入了解 Marketplace GTM 服務，請移至 [Go-to-market Services](https://partner.microsoft.com/reach-customers/gtm)。

請登入 [Cloud Partner 入口網站](https://cloudpartner.azure.com)，以建立及設定您的供應項目。
