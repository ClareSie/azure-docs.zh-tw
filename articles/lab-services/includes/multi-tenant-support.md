如果您是屬於多個 Azure AD 租使用者的系統管理員或實驗室擁有者，您可以從 Azure 實驗室服務入口網站的租使用者之間流覽 (右上角) 。 

![挑選租使用者](../media/multi-tenant-support/picker.png)

挑選租使用者之後，請在您的 Azure Active Directory 中選取實驗室帳戶， (在左上角 Azure AD) 租使用者。
