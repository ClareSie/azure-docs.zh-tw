# <a name="integration-patterns"></a>整合模式

Azure FarmBeats 是一種企業對企業產品，可在 Azure 應用商店中提供。 FarmBeats 通過融合資料集，通過構建人工智慧 （AI） 或機器學習 （ML） 模型，實現跨提供程式聚合農業資料集，並生成可操作的見解。

![專案 FarmBeats](./media/architecture-for-farmbeats/farmbeats-architecture-1.png)

以下各節介紹 Azure FarmBeats 的整合模式。

## <a name="why-integrate-with-azure-farmbeats"></a>為什麼要與 Azure 伺服器場節拍集成？

本節重點介紹希望將其資料系統集成到 Azure FarmBeats 中的合作夥伴。

Azure FarmBeats 是一種可擴展的產品，它使農業企業能夠將不同的歷史和即時農業資料集添加到單個平臺中。 Azure FarmBeats 可説明農業企業在伺服器場的上下文中正常化、上下文化和聚合其資料。

通過成為 Azure FarmBeats 的資料合作夥伴，您可以打開系統以更廣泛地採用，並通過資料產品聯繫更多客戶。 Azure FarmBeats 提供了一個稱為 Datahub 的可擴展 API 層，它可説明您系統地將設備中的資料引入到標準化架構中。

在客戶的 Azure FarmBeats 實例中提供資料後，您的客戶可以在資料的基礎上構建更豐富的分析和工具。
