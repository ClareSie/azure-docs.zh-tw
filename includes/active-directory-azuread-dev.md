> [!IMPORTANT]
> [Microsoft 標識平臺 （v2.0）](/azure/active-directory/develop/v2-overview)是 Azure 活動目錄 （Azure AD） 開發人員平臺 （v1.0） 的演變。 它可讓開發人員建置應用程式以登入所有 Microsoft 身分識別，並取得權杖以呼叫 Microsoft Graph 等 Microsoft API，或開發人員所建置的 API。
> 此內容適用于較舊的 Azure AD v1.0 終結點。 針對新的專案，建議您使用 v2.0 端點。 有關詳細資訊，請閱讀["為什麼更新至 Microsoft 標識平臺 （v2.0）"？](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) 以及[微軟的身份平臺限制](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations)。
