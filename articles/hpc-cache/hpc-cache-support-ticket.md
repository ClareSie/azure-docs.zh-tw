# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>開啟 Azure HPC Cache 的支援票證

使用 Azure 入口網站來開啟支援票證。 流覽至您的快取實例，然後按一下出現在側邊欄底部的 [ **新增支援要求** ] 連結。

若要在沒有作用中快取時開啟票證，請使用 Azure 入口網站的主要 [說明 + 支援] 頁面。 從畫面左上方的控制項開啟入口網站功能表，然後向下移至底部，然後按一下 [說明 **+ 支援**]。

選擇 [ **新增支援要求** ]，然後選取 [ **技術** ] 以 Azure HPC Cache 特定的 [說明]。

從清單中選取您的訂用帳戶。

若要尋找 Azure HPC Cache 服務，請按一下 [ **所有服務** ] 按鈕並搜尋 HPC。

![[支援要求-基本] 索引標籤的螢幕擷取畫面，部分填妥如下所述](media/hpc-cache-support-request.png)

使用您的資訊和喜好填寫其餘欄位，然後在準備就緒時提交票證。

提交要求之後，您會收到一封包含票證號碼的確認電子郵件。 支援人員成員將會與您聯絡要求。
