# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>地理空間 API，用於使用 Azure 地圖添加地圖、空間分析和移動解決方案

[Azure 地圖](https://azure.microsoft.com/services/azure-maps/)在雲中提供了本地位置平臺，並提供地理空間服務組合，這些服務使用新的映射資料為 Web 和移動應用程式提供準確的地理環境。 開發人員、企業和 ISV 可以使用 Azure 地圖創建位置感知應用程式和 IoT、移動性、物流和資產跟蹤解決方案。

## <a name="azure-maps-features"></a>Azure 地圖功能
- **搜索**： 提供有關附近興趣點的詳細資訊，渲染地圖上的位置，並使用地理編碼（或反向地理代碼）位址獲取相應的緯度和經度座標。
- **交通**：改善您的旅行時間，並説明您避免交通堵塞。
- **地圖**： 提供清晰、易於閱讀的地圖，以集成到解決方案中。
- **路由**：顯示在特定時間可用於多個目標的最短或最快的路由。 路線還為步行者、騎自行車者和商用車輛提供專門的路線和事實。
- **時區**：很容易看出它在世界任何地方的時間。
- **空間操作**：增強您的位置智慧。
- **地理位置**：查找 IP 位址的國家/地區。
- **資料服務**：上傳和存儲地理空間資料，以便與空間操作或圖像合成一起使用。
- **移動性**：提供有關附近公共交通服務的即時位置情報。 包括停靠點、路線資訊和行駛時間估計。
