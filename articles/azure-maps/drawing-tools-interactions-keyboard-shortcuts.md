# <a name="interaction-types-and-keyboard-shortcuts-in-the-drawing-tools-module"></a>繪圖工具模組中的互動類型與鍵盤快速鍵

本文將概述使用滑鼠、觸控式螢幕或鍵盤快速鍵來繪製和編輯圖形的各種不同方式。

繪圖管理員支援三種不同的方式來與地圖互動，以繪製圖形。

* `click` -按一下滑鼠或觸控時，就會加入座標。
* `freehand ` -當滑鼠或觸控拖曳至地圖上時，就會加入座標。
* `hybrid` -在按一下或拖曳滑鼠或觸控時，會加入座標。

## <a name="how-to-draw-shapes"></a>如何繪製圖案

 在可以繪製任何圖形之前，請將 `drawingMode` [繪圖管理員] 的選項設定為支援的繪圖設定。 這項設定可以透過按下工具列上的其中一個繪圖按鈕來進行程式設計或叫用。 即使在圖形繪製之後，繪圖模式仍會保持啟用狀態，讓您輕鬆地繪製相同類型的其他圖形。 以程式設計方式將繪圖模式設定為閒置狀態。 或者，按一下工具列上的 [目前繪圖模式] 按鈕，切換至閒置狀態。

下一節將概述可在地圖上繪製圖形的各種不同方式。

### <a name="how-to-draw-a-point"></a>如何繪製點

當繪圖管理員處於 `draw-point` 繪圖模式時，可以執行下列動作，在地圖上繪製點。 這些方法適用于所有互動模式。

**開始繪圖**
 - 按一下滑鼠左鍵，或觸碰地圖以將點新增至地圖。 
 - 如果滑鼠在地圖上，按下 `F` 按鍵，就會在滑鼠指標的座標處加入點。 這個方法會提供更高的精確度，以將點新增至地圖。 因為按下滑鼠左鍵的動作，所以滑鼠的移動會比較少。
 - 按住 ctrl、觸控或按 `F` 以將更多點新增至地圖。
 
**完成繪圖**
 - 按一下 [繪圖] 工具列中的任何按鈕。 
 - 以程式設計方式設定繪圖模式。 
 - 按下 `C` 按鍵。

**取消繪圖**
 - 按下 `Escape` 按鍵。

### <a name="how-to-draw-a-line"></a>如何繪製線條

當繪圖管理員處於模式時 `draw-line` ，您可以根據互動模式，完成下列動作以在地圖上繪製點。

**開始繪圖**
 - 按一下模式
   * 按一下滑鼠左鍵，或觸碰地圖以在地圖上新增線條的每個點。 每次點擊或觸控的線都會加入座標。 
   * 如果滑鼠在地圖上，按下 `F` 按鍵，就會在滑鼠指標的座標處加入點。 這個方法會提供更高的精確度，以將點新增至地圖。 因為按下滑鼠左鍵的動作，所以滑鼠的移動會比較少。
   * 繼續按一下，直到所有所需的點都已新增至該行為止。
 - 手繪模式
   * 按住滑鼠左鍵，或在地圖上向下觸控，然後拖曳滑鼠或觸控點。 當滑鼠或觸控點在地圖周圍移動時，會將座標新增至該行。 一旦觸發滑鼠或觸控事件，就會完成繪圖。 加入座標的頻率是由 [繪圖管理員] 選項所定義 `freehandInterval` 。
 - 混合模式
   * 繪製單一線條時，視需要在點擊和手繪方法之間替代。 例如，按一下幾個點，然後按住並拖曳滑鼠以新增許多點，然後再按幾下。 

**完成繪圖**
 - 混合/按一下模式
   * 按兩下最後一個點上的地圖。 
   * 按一下 [繪圖] 工具列中的任何按鈕。 
   * 以程式設計方式設定繪圖模式。 
 - 手繪模式
   * 放開滑鼠按鍵或觸控點。
 - 按下 `C` 按鍵。

**取消繪圖**
 - 按下 `Escape` 按鍵。

### <a name="how-to-draw-a-polygon"></a>如何繪製多邊形

當繪圖管理員處於模式時 `draw-polygon` ，您可以根據互動模式，完成下列動作以在地圖上繪製點。

**開始繪圖**
 - 按一下模式
   * 按一下滑鼠左鍵，或觸碰地圖以在地圖上加入多邊形的每個點。 每次點擊或觸控的座標都會加入至多邊形。 
   * 如果滑鼠在地圖上，按下 `F` 按鍵，就會在滑鼠指標的座標處加入點。 這個方法會提供更高的精確度，以將點新增至地圖。 因為按下滑鼠左鍵的動作，所以滑鼠的移動會比較少。
   * 繼續按一下，直到所有所需的點都已新增至多邊形為止。
 - 手繪模式
   * 按住滑鼠左鍵，或在地圖上向下觸控，然後拖曳滑鼠或觸控點。 當滑鼠或觸控點在地圖周圍移動時，會將座標新增至多邊形。 一旦觸發滑鼠或觸控事件，就會完成繪圖。 加入座標的頻率是由 [繪圖管理員] 選項所定義 `freehandInterval` 。
 - 混合模式
   * 在繪製單一多邊形時，視需要在點擊和手繪方法之間替代。 例如，按一下幾個點，然後按住並拖曳滑鼠以新增許多點，然後再按幾下。 

**完成繪圖**
 - 混合/按一下模式
   * 按兩下最後一個點上的地圖。 
   * 按一下多邊形中的第一個點。
   * 按一下 [繪圖] 工具列中的任何按鈕。 
   * 以程式設計方式設定繪圖模式。 
 - 手繪模式
   * 放開滑鼠按鍵或觸控點。
 - 按下 `C` 按鍵。

**取消繪圖**
 - 按下 `Escape` 按鍵。

### <a name="how-to-draw-a-rectangle"></a>繪製矩形的方式

當繪圖管理員處於模式時 `draw-rectangle` ，您可以根據互動模式，完成下列動作以在地圖上繪製點。 產生的圖形將遵循 [矩形的擴充 GeoJSON 規格](extend-geojson.md#rectangle)。

**開始繪圖**
 - 按下滑鼠左鍵，或在地圖上向下觸控以加入矩形的第一個角落，然後拖曳以建立矩形。 

**完成繪圖**
 - 放開滑鼠按鍵或觸控點。
 - 以程式設計方式設定繪圖模式。 
 - 按下 `C` 按鍵。

**取消繪圖**
 - 按下 `Escape` 按鍵。

### <a name="how-to-draw-a-circle"></a>如何繪製圓形

當繪圖管理員處於模式時 `draw-circle` ，您可以根據互動模式，完成下列動作以在地圖上繪製點。 產生的圖形將遵循 [圓形的擴充 GeoJSON 規格](extend-geojson.md#circle)。

**開始繪圖**
 - 按下滑鼠左鍵，或在地圖上向下觸控，以新增圓形的中心，並將圓形的半徑設為圓形。 

**完成繪圖**
 - 放開滑鼠按鍵或觸控點。
 - 以程式設計方式設定繪圖模式。 
 - 按下 `C` 按鍵。

**取消繪圖**
 - 按下 `Escape` 按鍵。

## <a name="keyboard-shortcuts"></a>鍵盤快速鍵

繪圖工具支援鍵盤快速鍵。 當地圖處於焦點時，這些鍵盤快速鍵會正常運作。

| Key      | 動作                            |
|----------|-----------------------------------|
| `C` | 完成任何進行中的繪圖，並將繪圖模式設定為閒置。 焦點將移至最上層的地圖元素。  |
| `Escape` | 取消任何進行中的繪圖，並將繪圖模式設定為閒置。 焦點將移至最上層的地圖元素。  |
| `F` | 如果滑鼠在地圖上，則將座標加入點、線條或多邊形。 按一下或混合模式時，按一下對應的對等動作。 此快捷方式可讓您更精確且更快速的繪圖。 您可以使用一手勢來定位滑鼠，而另一個手指按下按鈕，而不是從按下手勢移動滑鼠。 |
