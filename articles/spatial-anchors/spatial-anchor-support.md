# <a name="azure-spatial-anchors-support-options"></a>Azure 空間錨點支援選項

## <a name="open-a-tech-support-ticket"></a>打開技術支援票證

要在 Azure 門戶中為 Azure 空間錨點打開技術支援票證，可以執行以下規定：

1. 打開[Azure 門戶](https://azure.microsoft.com/account/)後，從頂部功能表列中選擇説明圖示，然後選擇 **"説明 + 支援**"按鈕。

![Azure 門戶説明和支援](./media/spatial-anchor-support.png)

1. 打開"説明 +支援"頁後，選擇 **"新支援請求**"。

![Azure 門戶新支援請求](./media/spatial-anchor-support2.png)

1. 完成支援票證欄位時：

- 問題類型：技術
- 服務：空間錨

![Azure 門戶支援票證欄位](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>社群支援

有關使用者社區的支援，請參閱[堆疊溢位上的 Azure 空間錨點](https://stackoverflow.com/questions/tagged/azure-spatial-anchors)。

## <a name="provide-content-feedback"></a>提供內容回饋

在每個內容文章的底部，有機會打開 GitHub 問題，並提供有關 Azure 空間錨點文檔內容的回饋。

## <a name="provide-product-feedback"></a>提供產品意見反應

要提供回饋、共用 Azure 空間錨點服務的想法或建議，或對其他人已提交的想法進行投票，請訪問[Azure 空間錨點回饋論壇](https://feedback.azure.com/forums/919252-azure-spatial-anchors)。

## <a name="next-steps"></a>後續步驟
