# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>高級塊 Blob 存儲帳戶的可伸縮性目標

針對使用較小千位元組範圍物件的應用程式優化了高性能塊 Blob 存儲帳戶。 它非常適合需要高事務速率或一致的低延遲存儲的應用程式。 高級性能塊 Blob 存儲旨在隨應用程式一起擴展。 如果方案要求部署每秒需要數十萬個請求或 PB 存儲容量的應用程式，請在[Azure 門戶](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)中提交支援請求與 Microsoft 聯繫。
