## <a name="monitor-event-routes"></a>監視事件路由

您可以在[Azure 入口網站](https://portal.azure.com/)中查看路由計量，例如計數、延遲和失敗率。 

從入口網站首頁中，搜尋您的 Azure 數位 Twins 實例以提取其詳細資料。 從左側的 Azure 數位 Twins 實例的導覽功能表選取 [**計量**] 選項，以顯示 [*計量*] 頁面。

:::image type="content" source="../articles/digital-twins/media/troubleshoot-metrics/azure-digital-twins-metrics.png" alt-text="顯示 Azure 數位 Twins 的 [計量] 頁面的螢幕擷取畫面。":::

從這裡，您可以查看實例的計量，並建立自訂的視圖。

如需觀看 Azure 數位 Twins 計量的詳細資訊，請參閱[*如何：使用 Azure 監視器來查看計量*](../articles/digital-twins/troubleshoot-metrics.md)。
