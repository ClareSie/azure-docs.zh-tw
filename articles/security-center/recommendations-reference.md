# <a name="security-recommendations---a-reference-guide"></a>安全性建議 - 參考指南

本文列出了您可能會在 Azure 安全中心中看到的建議。 環境中顯示的建議取決於您保護的資源和自定義配置。

要瞭解如何回應這些建議,請參閱[Azure 安全中心的「修正」建議](security-center-remediate-recommendations.md)。

您的安全分數基於您已完成的安全中心建議數。 要決定首先解決哪些建議,請查看每個建議的嚴重性及其對安全分數的潛在影響。

>[!TIP]
> 如果建議的描述顯示"沒有相關政策",通常是因為該建議依賴於不同的建議*及其*政策。 例如,"應修復端點保護運行狀況故障..."的建議依賴於檢查是否甚至*安裝了*端點保護解決方案的建議("應安裝端點保護解決方案...")。 基本建議*確實有*一個策略。 將策略限制為僅基礎建議簡化了策略管理。
