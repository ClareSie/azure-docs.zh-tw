# <a name="co-sell-partner-engagement"></a>共同銷售合作夥伴參與

如果您是合作夥伴，則可使用 Microsoft 共同銷售方案，直接與 Microsoft 合作共同銷售商機。 此方案可協助我們的合作夥伴觸及廣大的客戶群，並創造可協助加速業務成長的新銷售商機。

## <a name="partner-selling-motions"></a>合作夥伴銷售動作

Microsoft 提供三個合作夥伴銷售動作：

* *透過合作夥伴銷售* – 您可以透過自己的銷售人員轉售 Microsoft 技術。
* *服務共同銷售* – 銷售人員和合作夥伴共同作業，透過內嵌 Microsoft 技術來銷售您的專案或受控服務。
* *IP 共同銷售* –銷售人員和合作夥伴攜手合作，一起銷售您可重複、已封裝的 IP 解決方案。*

\* 當這些 IP 共同銷售解決方案經由 Microsoft 現場銷售團隊成功售出並且部署時，可能會收到額外的獎勵。
