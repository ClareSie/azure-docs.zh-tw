# <a name="microsoft-services-in-cybersecurity"></a>Microsoft 網路安全性服務

Microsoft 服務提供安全性、身分識別和網路安全性的完整方法。 他們會提供跨策略、規劃、實作和進行中支援的安全性和身分識別服務陣列。 這些服務可協助企業客戶實作符合其策略目標的安全性解決方案。

Microsoft 服務可以建立解決方案來整合我們的產品，並強化其最新安全性和身分識別功能，以協助保護您的業務並推動創新。

我們的技術專業人員小組由訓練有素的專家組成，他們提供豐富的安全性和和身分識別經驗。
