本文使用 Azure CLI。 要運行命令，可以使用 Azure 雲外殼。 Azure Cloud Shell 是免費的互動式 Shell，可讓您用來執行本文中的步驟。 它具有預先安裝和設定的共用 Azure 工具，可與您的帳戶搭配使用。

若要開啟 Cloud Shell，只要選取程式碼區塊右上角的 [試試看]**** 即可。 您還可以在單獨的瀏覽器選項卡中啟動雲外殼，通過前往[https://shell.azure.com](https://shell.azure.com)並切換左角的下拉清單以反映 Bash 或 PowerShell。 選擇 **"複製"** 以複製代碼塊，將其粘貼到雲外殼中，然後按 Enter 以運行它。
