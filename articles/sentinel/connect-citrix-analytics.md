# <a name="connect-citrix-analytics-security"></a>連線 Citrix Analytics (Security) 

Citrix Analytics (安全性) 與 Azure Sentinel 整合，可協助您將分析的資料從 Citrix Analytics (安全性) 轉換成 Azure Sentinel。 您可以建立自訂儀表板、分析來自其他來源的資料，以及從 Citrix Analytics (安全性) 的資料，並使用 Logic Apps 來建立自訂工作流程，以監視及減輕安全性事件。 



## <a name="prerequisites"></a>必要條件

- Azure Sentinel 工作區的讀取和寫入權限。

- Citrix Cloud 中的 Citrix Security Analytics 權利。


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>將 Azure Sentinel 連線至 Citrix Analytics (安全性) 

1. 在 Azure 入口網站中，流覽至**Azure Sentinel**  >  **資料連線器**，然後選取**Citrix 分析 (安全性) **連接器。

2. 選取 [ **開啟連接器] 頁面**。

3. 遵循 **Citrix Analytics (安全性) ** 頁面上的指示。
