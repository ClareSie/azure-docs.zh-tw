# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>設定支援資訊（Azure 實驗室服務中的實驗室帳戶擁有者）
本文說明您（身為實驗室帳戶擁有者）如何指定支援資訊，讓實驗室建立者（講師）和實驗室使用者（學生）在遇到使用實驗室帳戶或實驗室的任何技術問題時，可以用來取得協助。 

支援資訊包括：

- URL
- 電子郵件
- 電話
- 其他指示 

## <a name="specify-support-information"></a>指定支援資訊
1. 登入 [Azure 入口網站](https://portal.azure.com)。
2. 在搜尋列中，輸入**實驗室服務**，然後在搜尋結果中選取 [**實驗室服務**]。 
3. 從實驗室帳戶清單中選取您的實驗室帳戶。 
4. 切換至 [**內部支援**] 頁面，執行下列步驟：
    1. 輸入 [**支援 URL**]。 
     2. 輸入**支援電子郵件**。 
     3. 輸入**支援電話**。
     4. 輸入詳細的**支援指示**（選擇性）。 實驗室擁有者和使用者會看到此文字以及支援連絡人資訊。 Url 會自動轉換成連結。 
     5. 在工具列上選取 [儲存]  。
