# <a name="micro-focus-devtest-environments-on-azure"></a>Azure 上的微焦點開發/測試環境

使用開發和測試環境開始在 Azure 上工作是推動現代化工作的好方法。 依賴于舊系統（如 IBM CICS（客戶資訊控制系統）和 IMS（資訊管理系統））的大型機應用程式可以使用合作夥伴解決方案在 Azure 上重新託管。

Micro Focus 與 Microsoft 合作，提供引人注目的大型機遷移和重新託管工具。 您可以在 Azure 和 Microsoft Windows 平臺上跨開發、測試和生產方案運行大型機應用程式。

Micro Focus 提供開發重新託管和測試重新託管解決方案，將現有的 COBOL 和 PL/I 商務應用程式擴展到 Azure。
