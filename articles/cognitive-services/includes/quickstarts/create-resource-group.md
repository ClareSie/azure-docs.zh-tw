## <a name="create-a-resource-group"></a>建立資源群組

建立認知服務資源之前，您的帳戶必須有包含資源的 Azure 資源群組。 如果還沒有資源群組，請在開始之前，先在 [Azure 入口網站](https://ms.portal.azure.com/)中建立一個。
