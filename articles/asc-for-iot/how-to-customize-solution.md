# <a name="customize-your-azure-security-center-for-iot-solution"></a>自訂適用於 IoT 解決方案的 Azure 安全中心

在本指南中,瞭解如何在 Azure IoT 安全中心自定義不同的設置。

> [!div class="checklist"]
> * 設定解決方案建議
> * 變更設定

## <a name="change-settings"></a>變更設定

管理適用於 IoT 的 Azure 安全中心設定:

在 IoT 中心上,轉到左上角的安全概述邊欄選項卡,請參閱「設定」要設定適用於 IoT 設定的 Azure 安全中心,請執行以下操作:

1. 在 Azure 入口網站中開啟您的 **IoT 中樞**。
1. 從 **"安全"** 下的左側功能表中,選擇並打開 **「概述**」。
1. 在"設置"下,選擇要更改的解決方案設置。
1. 請記住,請始終按兩下"在任何設置螢幕頂部**保存**「以保存設置更改。

## <a name="configure-solution-recommendations"></a>設定解決方案建議

要為 IoT 解決方案建議配置 Azure 安全中心,請執行以下操作:

1. 在 Azure 入口網站中開啟您的 **IoT 中樞**。
1. 在左側功能表中的 **「安全」** 下選擇並打開 **「概述**」 。
1. 在"設置"下,從左側功能表中選擇 **"推薦配置**"。
1. 禁用/啟用與您的組織和工作流相關的解決方案建議。
1. 選擇 **「儲存在**螢幕頂部以儲存您的選擇」。
