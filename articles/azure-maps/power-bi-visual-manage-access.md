# <a name="manage-the-azure-maps-visual-within-your-organization"></a>管理組織內的 Azure 地圖服務視覺效果

Power BI 提供讓設計人員和租使用者系統管理員管理 Azure 地圖服務視覺效果使用的能力。 您會在下文中發現每個角色可以採取的步驟。

### <a name="designer-options"></a>設計工具選項

在 Power BI Desktop 中，設計人員可以停用 [安全性] 索引標籤上的 Azure 地圖服務視覺效果。選取 **[** 檔案 &gt; **選項和設定** ]，然後選取 [ **選項** &gt; **預覽功能**]。 停用時，預設不會載入 Azure 地圖服務。  
  
![顯示 Azure 地圖服務視覺效果預覽功能選項的 Power BI 選項面板](media/power-bi-visual/preview-options-panel.png)

## <a name="tenant-admin-options"></a>租用戶系統管理員選項

在 PowerBI.com 中，租使用者系統管理員可以關閉所有使用者的 Azure 地圖服務視覺效果。 選取 [**設定**系統 &gt; **管理員****入口網站** &gt; **租使用者設定**]。 停用時，Power BI 將不會再于 [視覺效果] 窗格中顯示 Azure 地圖服務的視覺效果。

> [!div class="mx-imgBorder"]
> ![Power BI 系統管理員入口網站顯示 Azure 地圖服務視覺效果的租使用者設定](media/power-bi-visual/tenant-admin-settings.png)
