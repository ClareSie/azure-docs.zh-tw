# <a name="co-sell-partner-engagement"></a>共同銷售合作夥伴參與

微軟的聯合銷售計劃使合作夥伴能夠直接與微軟合作,共同尋找銷售機會。 您可以覆蓋龐大的客戶社區,創造新的銷售機會,説明加快業務增長。

## <a name="partner-selling-motions"></a>合作夥伴銷售動議

有三個合作夥伴銷售動議:

* *通過合作夥伴銷售*:您可以通過自己的賣家轉售微軟技術。
* *服務共同銷售*:賣家和合作夥伴協作銷售您的專案或託管服務與微軟技術嵌入。
* *IP 共同銷售*: 賣家/合作夥伴協作銷售您的可重複打包的 IP 解決方案*。

\*當與 Microsoft 的現場銷售團隊成功銷售並部署時,其中一些 IP 共同銷售解決方案可能會獲得額外的獎勵。
