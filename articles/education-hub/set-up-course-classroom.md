# <a name="quickstart-set-up-a-course-and-create-a-classroom"></a>快速入門：設定課程並建立教室

本快速入門說明如何在 Microsoft Azure Education Hub 中設定課程和教室，包括訂用帳戶詳細資料。

## <a name="prerequisites"></a>必要條件

- 具有已核准信用額度的學術授與
- 將存取您 Azure 點數的教室內訂用帳戶

### <a name="subscriptions"></a>訂用帳戶

每個學生都有一個訂用帳戶繫結至由教授所配置的點數金額上限。 「金額上限」一詞說明學術贊助的美元金額。 例如，$1000 金額上限使用[發佈的 WebDirect 費率](https://azure.microsoft.com/pricing/calculator/)，為收件者提供 $1000 美元的 Azure 點數。

教育工作者可以選擇為訂用帳戶配置上限，以防止非預期的使用，然後設定到期日。 以下是一些範例：

- **每個班級的一般金額**：每個學生都會取得 $x，在整季或學期做管理。

- **每個專案群組的一般金額**：專案小組會收到 $x 作為群組專案的資金。

在訂用帳戶層級上，您可以增加或減少上限，變更會以最少的延遲生效。 當班級或專案結束時，您可以在到期日之前，將未使用的上限重新配置給其他訂用帳戶。

## <a name="create-a-course-and-classroom"></a>建立課程和教室

依照下列步驟使用兩種不同方法的其中一種來建立課程和教室：

1. 選取 Azure Education Hub 中的 [課程] 頁面，開啟您用來建立及管理課程的工具。 資料表隨即開啟，其中顯示您現有的課程。

    :::image type="content" source="media/set-up-course-classroom/education-hub-courses.png" alt-text="Azure Education Hub 課程頁面" border="false":::

1. 選取資料表左上角的 [+ 新增] 圖示，以開始建立工作流程。

    :::image type="content" source="media/set-up-course-classroom/courses-add.png" alt-text="Azure Education Hub 課程頁面" border="false":::

1. 您可以使用兩種方法來建立課程名冊：上傳名冊或使用邀請代碼。
    - **名冊**：如果您已經有所有學生的名稱和登入，您可以填入並上傳名冊檔案。 若要下載上傳名冊所需的 .csv 檔案範例檔案，請選取右上角的 [下載範例檔案] 連結。
    - **邀請代碼**：如果您選擇使用邀請代碼，請決定有多少代碼可以兌換，以及何時過期。 您會將下列連結傳送給您的學生，以兌換代碼： https://aka.ms/joinedu 。

      :::image type="content" source="media/set-up-course-classroom/enter-invitation-code.png" alt-text="Azure Education Hub 課程頁面" border="false":::

1. 選取右下角的 [建立教室]。 這可能需要一些時間才能完成。

   :::image type="content" source="media/set-up-course-classroom/create-classroom.png" alt-text="Azure Education Hub 課程頁面" border="false":::
