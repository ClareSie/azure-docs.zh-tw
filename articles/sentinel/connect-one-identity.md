# <a name="connect-one-identity-safeguard"></a>連接一個身分識別保護
一種身分識別保護資料連線器可增強標準的一般事件格式（CEF）連接器，並提供特殊許可權會話特定儀表板的防護。 使用此連接器可輕鬆地開始使用您的裝置所產生的事件，以進行視覺效果、警示、調查等等。


## <a name="prerequisites"></a>先決條件

- [Azure Sentinel] 工作區的 [讀取] 和 [寫入] 許可權。

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>將 Azure Sentinel 連接到一個身分識別保護

1. 在 [Azure 入口網站中，流覽至 [ **Azure Sentinel** > **資料連線器**]，然後選取 [**一個身分識別保護**連接器]。

2. 選取 [**開啟連接器] 頁面**。

3. 依照 [**一個身分識別保護**] 頁面上的指示進行。
