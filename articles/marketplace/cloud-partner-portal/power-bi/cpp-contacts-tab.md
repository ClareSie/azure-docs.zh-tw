# <a name="power-bi-app-contacts-tab"></a>Power BI 應用程式的 [連絡人] 索引標籤

使用 [新增供應項目]**** 頁面的 [連絡人]**** 索引標籤，為您的供應項目提供技術和使用者支援資源。 它分為兩個部分：**工程和****客戶支援**。

![連絡人索引標籤](media/contacts-tab.png)


## <a name="contacts-fields"></a>連絡人欄位 

下表描述了每個欄位的用途。  必要欄位由星號 (*) 標示。 

|   欄位               |   描述                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *工程連絡人* | AppSource 與貴組織的技術連絡人。    |
| **名字\***              | 負責技術/工程支援的人員或群組名稱。 |
| **電子郵件\***             | 此技術連絡人的電子郵件地址。                                  |
| **電話\***             | 此技術連絡人的電話號碼。                                       |
| *支援連絡人*     | 接收客戶在 AppSource 中開立的支援票證。            |
| **名字\***              | 負責客戶支援的人員或群組名稱。              |
| **電子郵件\***             | 客戶支援部門的電子郵件地址。                                        |
|   |   |
