## <a name="authentication"></a>驗證

要運行以下示例，需要一個可以訪問資源的 Azure 活動目錄 （Azure AD） 應用程式和服務主體。 要創建免費的 Azure AD 應用程式並在訂閱級別添加角色指派，請參閱[創建 Azure AD 應用程式](/azure/active-directory/develop/howto-create-service-principal-portal)。 您還需要目錄（租戶）ID、應用程式 ID 和用戶端金鑰。
