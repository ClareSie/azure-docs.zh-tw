# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

本文說明 **503003 PartitionNotFound** 錯誤的原因和解決方案。

## <a name="symptoms"></a>徵兆

對 IoT 中樞的要求失敗，錯誤為 **503003 PartitionNotFound**。

## <a name="cause"></a>原因

此錯誤是 IoT 中樞的內部錯誤，可能是暫時性的。 請參閱 [IoT 中樞內部伺服器錯誤原因](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause)。

## <a name="solution"></a>解決方法

請參閱 [IoT 中樞內部伺服器錯誤的解決方案](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution)。
