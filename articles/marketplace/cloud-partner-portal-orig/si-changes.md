# <a name="seller-insights-release-notes"></a>Seller Insights 版本資訊 

（發佈日期：2019 年 3 月 1 日）

此文章提供 [Cloud Partner 入口網站](https://cloudpartner.azure.com/#insights)中 Seller Insights 功能變更的相關資訊。

## <a name="release-highlights-for-march-1-2019"></a>2019 年 3 月 1 日發佈的亮點

* *添加到摘要的客戶趨勢*
* 摘要上*的前五個客戶*反映客戶擁有的所有 Azure 訂閱
* *標準化使用趨勢&活動訂單趨勢*匯總移動下*每月訂單概覽*
* *付款對賬報告*更新
* 付款*方面排名前五的客戶*反映客戶擁有的所有 Azure 訂閱
* 使用客戶 ID 更新*的使用報告*
* *客戶訂單期限*&使用方式反映客戶擁有的所有 Azure 訂閱


(發行日期：2018 年 7 月 28 日)

## <a name="release-highlights-for-july-28-2018"></a>2018 年 7 月 28 日發佈亮點


-   *預估價格*提供含有貨幣轉換影響的客戶收費檢視。
-   *預測支出*提供潛在支出的預先檢視。
-  *使用量參考識別碼*提供客戶使用量與支出訂單之間的資料精確度
-   *每日使用量*提供針對客戶使用量更細微且更佳的見解。


### <a name="changes-to-data-structure-and-taxonomy"></a>變更資料結構和分類法

下表列出此版本已新增或大幅變更的計量資訊。 

| **新字詞**                   |    **定義**                                                             |
|--------------------------------|  ---------------------------------------------------------------------------- |
| 價格 (CC)                     | 指定 SKU 單位使用量的價格 (以客戶的貨幣為單位)。       |
| 預估擴充費用 (CC) | 給定 SKU（以客戶貨幣表示）的使用單位數量的估計擴展費用。 由於四捨五入或截斷錯誤，這個值可能並不精確。   |
| 發行者貨幣 (PC)        | 發行者用於支出的常用貨幣。                               |
| 預估價格 (PC)           | 根據計算日期使用方式的外匯兌換（以發行者的貨幣）計算給定 SKU 的使用量單位的估計價格。 由於四捨五入或截斷錯誤，這個值可能並不精確。   |
| 預估擴充費用 (PC) | 根據計算日期使用日期的外匯兌換（以發行者的貨幣）計算給定 SKU 的使用單位數量的估計擴展費用。 由於四捨五入或截斷錯誤，這個值可能並不精確。 |
| 預估付款 (PC)          | 根據計算使用日期（以發行者的貨幣）的外匯兌換情況，估計給定 SKU 的使用單位數量。 由於四捨五入或截斷錯誤，這個值可能並不精確。   |
| 使用量參考                | 針對指定 SKU 的一或多天客戶使用量識別碼，與付款報告中的某個項目相關聯。 |
|  |  |
