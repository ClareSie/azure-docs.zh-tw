若要執行 Cmdlet，您可以使用 Azure Cloud Shell (免費的互動式殼層)。 它具有預先安裝和設定的共用 Azure 工具，可與您的帳戶搭配使用。 只要按一下 [複製]**** 即可複製程式碼，將它貼到 Cloud Shell 中，然後按 enter 鍵加以執行。 以下有幾種啟動 Cloud Shell 的方式：


|  |   |
|-----------------------------------------------|---|
| 在您的瀏覽器中開啟 Cloud Shell。 | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| 按一下門戶右上角功能表上的 **"雲殼**"按鈕。 | [![門戶中的雲外殼](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


如果不想使用 Azure 雲殼，則可以在本地安裝 PowerShell。 如果您選擇在本地安裝和使用 PowerShell，請確保安裝最新版本的資源管理器 PowerShell Cmdlet。 PowerShell Cmdlet 經常更新，您通常需要更新 PowerShell Cmdlet 以獲得最新的功能功能，否則可能會遇到問題。

要查找您正在本地運行的 PowerShell 版本，請使用"獲取模組 -清單可用 Az"Cmdlet。 若要更新，請參閱[安裝 Azure PowerShell 模組](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)。 有關詳細資訊，請參閱[如何安裝和配置 Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/overview)。

如果您在 macOS 上使用 PowerShell，請按照在[macOS 上安裝 PowerShell 核心](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)的步驟操作。
