# <a name="summary-dashboard-in-commercial-marketplace-analytics"></a>商業市集分析中的摘要儀表板

本文提供合作夥伴中心內 [摘要] 儀表板的相關資訊。 此儀表板會顯示摘要說明您供應項目市集活動的彙總資料圖表、趨勢和值。

若要存取 [摘要] 儀表板，請開啟 [商業 Marketplace] 底下的 [[分析儀表板](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)]。

>[!NOTE]
> 如需分析術語的詳細定義，請參閱[商業市集分析的常見問題集與術語](./faq-terminology.md)。

## <a name="summary-dashboard"></a>摘要儀表板

[摘要] 儀表板會根據每個供應項目類型來提供概觀。 [深入解析] 會顯示重要資訊一覽，並針對您供應項目的銷售活動提供廣泛的觀點。 您可以使用 [摘要] 儀表板將這些報告視覺化。 本文會詳細說明下列每個元素：

- [日期範圍](#date-range)
- [摘要區段](#summary-section)
- [客戶數 (依地理位置)](#customers-by-geography)
- [成長趨勢圖表](#growth-trend-charts)
- [客戶排行榜](#customer-leaderboard)
- [基座計數趨勢](#seat-count-trend)
- [免費試用 SaaS 訂單趨勢](#free-trials-saas-orders-trend)

## <a name="elements-of-the-summary-dashboard"></a>摘要儀表板的元素

下列幾節會說明如何使用摘要儀表板以及如何閱讀資料。

### <a name="date-range"></a>日期範圍

您可以在每一頁的右上角找到日期範圍選取項目。 藉由選取日期範圍 (以過去 3 個月、6 個月或 12 個月為基礎)，或選取最長持續 12 個月的自訂日期範圍，即可自訂 [摘要] 頁面圖表的輸出。 預設的日期範圍是六個月。

![合作夥伴中心的分析儀表板](./media/analyze-dashboard.png)

### <a name="summary-section"></a>摘要區段

[摘要] 區段會顯示所選日期範圍內的所有已建立訂單計數、獲得的客戶和報告的使用量。 當月若未滿一個月，則會從這些計量的計算中排除。 例如：如果您選取 [6M] 時間範圍，則會計算當月之前六個月所計算得到的使用時數。 如果選取自訂日期範圍，則計算中會排除當月的部分數量。

![[摘要] 儀表板中的成長趨勢](./media/summary-summary-section.png)

#### <a name="reading-the-summary-section"></a>閱讀摘要區段

- **訂單**：截至目前為止，購買了您所發佈供應項目的所有訂單計數 (排除已取消的訂單)。
- **客戶**：購買了您的供應項目且至少有一個未取消訂單的所有客戶計數。
- **標準化使用時數**：定義為帳戶對於 VM 核心數目的標準化使用時數 ([VM 核心數目] x [未處理使用時數])。 指定為 "SHAREDCORE" 的 VM 使用 1/6 (或0.1666) 作為 [VM 核心數目] 乘數。
- **未處理使用時數**：以小時為單位，VM 已執行了多久的時間。 **總訂單**、**總客戶**、**標準化使用時數**、**未處理使用時數**、**頁面瀏覽數**和**動作呼叫數**旁的百分比值，代表所選日期範圍的使用量成長量 ([最後一個月使用量 - 第一個月使用量])/第一個月使用量)。 如上所述，當月的部分數量將會從此計量中排除。
- **成長趨勢**：如果您將滑鼠停留在圖表的資料行上，長條圖會顯示每個月的值。
- **指向上方的綠色三角形**：表示正成長趨勢。
- **指向下方的紅色三角形**：表示相對於上個月是負成長趨勢。

### <a name="customers-by-geography"></a>客戶數 (依地理位置)

[客戶數 (依地理位置)] 熱度圖會在世界地圖上顯示客戶計數。

![摘要儀表板中的客戶數 (依地理位置)](./media/summary-customers-by-geography.png)

- 您可以移動地圖以檢視確切位置。
- 您可以放大特定位置。
- 熱度圖有一個補充格線，可供您檢視特定位置的客戶計數、訂單計數、標準化使用時數詳細資料。
- 您可以在格線中搜尋並選取某個國家/地區，以縮放至地圖中的某個位置。 按下地圖中的 [首頁] 按鈕可還原為原始檢視。
- 在選取日期範圍內的該月份，有**新**客戶首次購買了您的其中一個供應項目。

### <a name="growth-trend-charts"></a>成長趨勢圖表

您可以根據**所購買訂單** (包括已取消的訂單)、**獲得的客戶** (包括流失的客戶) 和**報告的使用量**的成長來檢視趨勢 (根據選取的日期範圍逐月顯示)。 您也可以進一步分析這些趨勢，方法是選取圖表下方的連結，以瀏覽至個別的**訂單**、**使用量**、**客戶數**或 **Marketplace 深入解析**頁面。

針對 Azure 市集和 AppSource，Marketplace 供應項目的**頁面瀏覽數和動作呼叫數**趨勢圖會顯示在兩個索引標籤上。

![[摘要] 儀表板中頁面瀏覽數和動作呼叫數的趨勢圖表](./media/summary-page-visits-and-cta.png)

**訂單數 (依供應項目)** 圖表會根據供應項目名稱來組織您的訂單。

**訂單數 (依銷售通路)** 圓形圖會依銷售通路來組織所選日期範圍內的訂單 (包括客戶已取消的訂單)。 銷售通路是客戶用來購買 Azure 的授權合約類型，分別有雲端解決方案提供者 (CSP)、企業、透過轉銷商的企業、GTM 和隨用隨付。

**使用量 (依供應項目)** 和**使用量 (依銷售通路)** 圓形圖分別會顯示依熱門供應項目和依銷售通路的使用量明細。 內部的圓形圖代表未處理使用量，外部的圓形圖則代表標準化使用量。

**訂單數 (依市集授權類型)** 和**使用量 (依市集授權類型)** 圓形圖會依各自的授權類型顯示訂單數和使用量的明細。 授權類型包括：

- **透過 Azure 計費**：當您選擇透過 Microsoft 向此授權類型銷售您的供應項目時，Microsoft 會代替您向客戶收費。 付款類型包含透過信用卡的隨用隨付或企業發票。
- **自備授權**：Microsoft 不會向使用這類市集供應項目的客戶收費。 在市集中，此使用量會列為**立即取得 (免費)** 。
- **免費**：Microsoft 不會向使用這類市集供應項目的客戶收費。 在市集中，此使用量會列為**免費試用**。
- **Microsoft 為轉銷商**：代表 Microsoft 轉銷商在**雲端解決方案提供者 (CSP) 計畫**中銷售的供應項目。

### <a name="customer-leaderboard"></a>客戶排行榜

訂單數最高的前 50 大客戶會顯示在*排行榜*上，並依最高訂單計數和訂單百分比進行排序。

- 選取客戶即可檢視其設定檔詳細資料、依供應項目組織的訂單，或依 Azure 授權類型和定價通道組織的訂單。
- **供應項目 (依訂單)** 環圈圖會顯示前四大供應項目 (依訂單計數) 和其餘供應項目 (全都分組為「其餘所有供應向目」)。
- **標準化使用量 (依供應項目)** 環圈圖會依使用量顯示前五大供應項目。

> [!NOTE]
> 只有在客戶已同意時，才會顯示客戶的個人資訊。 如果您已使用**擁有者**角色權限層級登入，就可以檢視此資訊。 具有**參與者**角色的使用者則無法看到此資訊。 [深入了解使用者角色和權限](./manage-account.md#define-user-roles-and-permissions)。

### <a name="seat-count-trend"></a>基座計數趨勢

**訂單數 (依每個基座/每個網站)** 圖表會顯示根據定價模型所購買的所有訂單明細。 **基座計數趨勢**圖表會顯示基座與您所有的每個基座軟體即服務 (SaaS) 供應項目所購買的訂單。

### <a name="free-trials-saas-orders-trend"></a>免費試用 SaaS 訂單趨勢

**免費試用 SaaS 訂單趨勢**圖表會顯示有 30 天試用期的免費試用 SaaS 供應項目訂單趨勢。
