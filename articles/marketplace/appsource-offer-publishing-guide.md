# <a name="offer-publishing-guide-for-microsoft-365-dynamics-365-power-apps-and-power-bi"></a>Microsoft 365、Dynamics 365、Power Apps 和 Power BI 的供應專案發佈指南

Microsoft AppSource 包含以 Microsoft 365、Dynamics 365、Power Apps 和 Power BI 為建立或擴充的供應專案。 每一項產品都有特定的檔，可引導您進行發佈流程。 

AppSource 有付費、清單及試用版供應專案，且清單選項 *立即取得*，請 *洽詢我*，並 *立即試用*。

## <a name="product-specific-guides"></a>產品特定指南

如需每個供應專案類型的發佈程式和指導方針的連結，請參閱下表：

| 產品    | 發佈指導方針  |
| :------------------- | :-------------------|
| Microsoft 365 | 檢閱[發佈程序和指導方針](/office/dev/store/submit-to-appsource-via-partner-center)。 |
| Dynamics 365 Business Central | 若要在合作夥伴中心中發佈供應專案，請參閱 [建立新的 Dynamics 365 Business Central 供應](./partner-center-portal/create-new-business-central-offer.md)專案。 |
| Dynamics 365 for Finance and Operations | 建置 Enterprise Edition 時，請檢閱[發行程序和指導方針](/dynamics365/fin-ops-core/dev-itpro/lcs-solutions/lcs-solutions-app-source)。<br/>若要在合作夥伴中心中發佈供應專案，請參閱 [建立新的 Dynamics 365 For Operations 供應](./partner-center-portal/create-new-operations-offer.md)專案。  |
| Dynamics 365 for Customer Engagement | 檢閱[發佈程序和指導方針](/dynamics365/customer-engagement/developer/publish-app-appsource)。<br/>若要在合作夥伴中心中發佈供應專案，請參閱 [Power Apps 供應專案建立新的 Dynamics 365 For Customer Engagement &](./partner-center-portal/create-new-customer-engagement-offer.md)。  |
| Power Apps | 瞭解如何 [讓客戶在 AppSource 上試用您的應用程式](https://powerapps.microsoft.com/blog/appsource-test-drive/)。 |
| Power BI | 檢閱[發佈程序和指導方針](/power-bi/developer/office-store)。 |
