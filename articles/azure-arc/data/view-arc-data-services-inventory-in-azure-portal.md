# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>在 Azure 入口網站中查看實例的清查

上傳您的 [計量、記錄](upload-metrics-and-logs-to-azure-monitor.md)或 [使用](view-billing-data-in-azure.md)方式之後，您可以從 Azure 入口網站將您的實例視為 Azure 資源來查看。 若要這樣做，請流覽至特殊 URL [https://portal.azure.com](https://portal.azure.com) ，然後遵循以下的螢幕擷取畫面

1. 移至所有服務
1. 搜尋您的資料庫實例類型類型
1. 將它新增至我的最愛
1. 從左側導覽中按一下 [實例類型]
1. 在與其他 Azure SQL 或 Azure 于 postgresql 超大規模資源相同的視圖中，查看您的實例 (使用篩選器來取得細微的觀點)
