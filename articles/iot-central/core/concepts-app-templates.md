# <a name="what-are-application-templates"></a>什麼是應用程式範本？

Azure IoT 中心中的應用程式範本是説明解決方案構建者啟動其 IoT 解決方案開發的工具。 您可以將應用範本用於各種內容，從瞭解可能的內容，到完全自訂應用程式以及向客戶轉售的應用程式。

應用程式範本包括：

- 範例操作員儀表板
- 示例設備範本
- 生成即時資料的類比設備
- 預先設定的規則和作業
- 豐富的文檔，包括教程和操作方式

創建應用程式時選擇應用程式範本。 創建應用程式後無法更改範本。

## <a name="custom-templates"></a>自訂範本

如果要從頭開始創建應用程式，請選擇兩個自訂應用程式範本之一：

- 自訂應用程式
- 自訂應用程式（舊版）

除非有使用舊版範本的特定原因，否則請選擇**自訂應用程式**範本。

## <a name="industry-focused-templates"></a>以行業為中心的範本

Azure IoT 中心是一個與行業無關的應用程式平臺。 應用程式範本是當今這些行業的行業重點示例，未來還會有更多的示例：

- [零售](../retail/overview-iot-central-retail-pnp.md)
  - 連線物流
  - 數位配送中心
  - 店內分析 - 條件監視
  - 店內分析 - 結帳
  - 智慧庫存管理
- [能源](../energy/overview-iot-central-energy.md)
  - 智慧型儀器表監控
  - 太陽能電池板監測
- [政府](../government/overview-iot-central-government.md)
  - 互聯廢物管理
  - 耗水量監測
  - 水質監測
- [醫療保健](../healthcare/overview-iot-central-healthcare.md)。
  - 持續患者監測
