# <a name="keyboard-shortcuts-in-the-azure-portal"></a>Azure 入口網站的鍵盤快速鍵

本文列出了 Azure 門戶中所有位置的鍵盤快速鍵。 個別的服務可能會擁有各自的鍵盤快速鍵。

## <a name="actions"></a>動作

|若要執行這項操作 |按鍵 |
| --- | --- |
|建立資源|G+N|
|打開**所有服務**|G+B|
|搜尋資源、服務及文件|G+/|
|搜尋資源功能表項目|CTRL+/ |
|向上移動所選左邊欄專案 |ALT+Shift+向上鍵|
|向下移動所選左邊欄專案 |ALT+Shift+向下鍵|

## <a name="navigation"></a>導覽

|要執行此導航 |按鍵 |
| --- | --- |
|將焦點移至命令列 |G+, |
|在標題和左側邊欄之間切換焦點 | G+. |

## <a name="go-to"></a>移至

|轉到此位置 |按鍵 |
| --- | --- |
|轉到**儀表板** |G+D |
|轉到**所有資源**|G+A |
|轉到**資源組**|G+R |
|在此位置打開左側側邊欄專案 |G+number|
