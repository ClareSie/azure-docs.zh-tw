# <a name="storsimple-8000-series-update-51-release-notes"></a>StorSimple 8000 系列更新5.1 版本資訊

## <a name="overview"></a>概觀

下列版本資訊說明新功能，並找出 StorSimple 8000 系列 Update 5.1 的重大未解決問題。 當中也包含此版本中隨附之 StorSimple 軟體更新的清單。

更新5.1 可套用至任何執行 Update 5 的 StorSimple 裝置。 如果您使用的版本低於5，請先套用 update 5，然後套用5.1。 與更新5.1 相關聯的裝置版本為6.3.9600.17885。

在 StorSimple 方案中部署更新之前，請檢閱版本資訊中所包含的資訊。

> [!IMPORTANT]
>
> * 更新5.1 是強制性更新，必須立即安裝。 如需詳細資訊，請參閱如何套用 [更新 5.1](storsimple-8000-install-update-51.md)。
> * 更新5.1 只有安全性更新。 安裝此更新需要大約30分鐘的時間。 強烈建議您套用更新5.1，以確保您的裝置運作正常。
> * 如果是新版本，您可能不會立即看到更新，因為我們會分階段推出更新。 請等候數天然後再次掃描更新，因為很快就會提供這些更新。

## <a name="whats-new-in-update-51"></a>更新5.1 的新功能

更新5.1 中已進行下列重要的改進和 bug 修正：

* **TLS 1.2** -此 StorSimple 更新將在所有用戶端上強制執行 TLS 1.2。 這是所有 StorSimple 8000 系列裝置的強制更新。

## <a name="known-issues-in-update-51-from-previous-releases"></a>舊版更新5.1 的已知問題

更新5.1 中沒有新的已知問題。 如需從舊版升級至5.1 的問題清單，請移至 [update 3 版本](storsimple-update3-release-notes.md#known-issues-in-update-3)資訊。

## <a name="storsimple-cloud-appliance-updates-in-update-51"></a>更新5.1 中的 StorSimple 雲端設備更新

這項更新無法套用至 StorSimple Cloud Appliance (也稱為虛擬裝置)。 您必須使用更新5.1 映射來建立新的雲端設備。 如需如何建立 StorSimple 雲端設備的資訊，請移至[部署和管理 StorSimple 雲端設備](storsimple-8000-cloud-appliance-u2.md)。
