> [!IMPORTANT]
> 若要避免在沒有任何作業正在執行時產生費用，請**將最小節點設定為 0**。 此設定可讓 Azure Machine Learning 在節點不在使用中時，將它們解除配置。 任何大於0的值都會讓節點數目保持執行，即使它們不在使用中也一樣。
