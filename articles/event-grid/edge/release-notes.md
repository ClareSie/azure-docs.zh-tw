# <a name="release-notes-azure-event-grid-on-iot-edge"></a>版本資訊：IoT 邊緣的 Azure 事件網格

## <a name="100-preview1"></a>1.0.0-預覽1

在 IoT 邊緣上首次發佈 Azure 事件網格。 包括的功能：

* 主題創建
* 事件訂閱創建
* 進階篩選
* 輸出批次處理
* 重試原則
* 模組到模組發佈
* 作為目標發佈到 WebHook
* 作為目標發佈到 IoT 邊緣中心
* 作為目標發佈到 Azure 事件網格雲服務
* 中繼資料的持久狀態
* Blob 存儲模組集成

標籤：`1.0.0-preview1`

## <a name="100-preview2"></a>1.0.0-預覽2

在 IoT 邊緣上添加的 Azure 事件網格預覽 2：

* 可配置到磁片的持久化事件
* 主題指標
* 事件訂閱指標
* 作為目標發佈到事件中心
* 將匯流排佇列發佈到服務匯流排佇列作為目標
* 將服務匯流排主題發佈為目標
* 作為目標發佈到存儲佇列

標籤： `1.0.0-preview2` `1.0`， ，`latest`
