> [!IMPORTANT]
> 此文章包含的內容需要 Azure 傳統 CLI。 Azure CLI 的最新版本不支援此文章中概述的功能，因此需要傳統 CLI。 
> 
> 傳統 CLI 可以與最新的 Azure CLI，並存安裝，但我們建議所有新的指令碼和部署的 Azure CLI。
> 若要安裝傳統 CLI，請參閱[安裝 Azure 傳統 CLI](/cli/azure/install-classic-cli)。 
> 若要安裝最新版的 CLI，請參閱[安裝 Azure CLI](/cli/azure/install-azure-cli)。
