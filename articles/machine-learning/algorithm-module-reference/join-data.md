# <a name="join-data"></a>聯結資料

本文介紹如何使用 Azure 機器學習設計器中的**聯接資料**模組（預覽）使用資料庫樣式的聯接操作合併兩個資料集。  

## <a name="how-to-configure-join-data"></a>如何配置聯接資料

要對兩個資料集執行聯接，它們應該由鍵列關聯。 還支援使用多列的複合鍵。 

1. 添加要合併的資料集，然後將 **"加入資料"** 模組拖到管道中。 

    您可以在 **"操作**"下的資料**轉換**類別中找到該模組。

1. 將資料集連接到 **"聯接資料"** 模組。 
 
1. 選擇 **"啟動"欄選取器**以選擇鍵列。 請記住為左輸入和右輸入選擇列。

    對於單個鍵：

    為兩個輸入選擇單個鍵列。
    
    對於複合鍵：

    按相同順序從左輸入和右輸入選擇所有鍵列。 當所有鍵列匹配時，"**聯接資料**"模組將聯合資料表。 如果列順序與原始表不同，則選中"**允許重複"並保留列順序**的選項。 

    ![資料行選取器](media/module/join-data-column-selector.png)


1. 如果要在文本列聯接上保留區分大小寫，請選擇 **"匹配大小寫**"選項。 
   
1. 使用 **"聯接類型**"下拉清單指定資料集的合併方式。  
  
    * **內部聯接**：*內部聯接*是最常見的聯接操作。 只有當索引鍵資料行的值符合時，它才會傳回合併的資料列。  
  
    * **左外部聯結**：*左外部聯結*返回從左表中的所有行聯接行。 當左表中的行在右表中沒有匹配的行時，返回的行包含來自右表的所有列的缺失值。 還可以為缺失值指定替換值。  
  
    * **完整外部聯結**：*完整外部聯結*返回左表 （**表 1**） 和右表 （**表2）** 的所有行。  
  
         對於任一表中沒有匹配行的表中的每一行，結果包括包含缺失值的行。  
  
    * **左半聯接**：當鍵列的值匹配時，*左半聯接*僅返回左表中的值。  

1. 對於選項 **"在聯合資料表中保留正確的鍵列**"

    * 選擇此選項可查看兩個輸入表中的鍵。
    * 取消選擇以僅從左側輸入返回鍵列。

1. 提交管道。

1. 要查看結果，請按右鍵 **"加入資料"** 並選擇 **"視覺化**"。
