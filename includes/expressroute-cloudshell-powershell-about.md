可以使用 Azure 雲外殼運行大多數 PowerShell Cmdlet 和 CLI 命令，而不是在本地安裝 Azure PowerShell 或 CLI。 Azure 雲外殼是一個免費的互動式外殼，具有預先安裝的常見 Azure 工具，並配置為與帳戶一起使用。
若要在 Azure Cloud Shell 上執行本文所包含的任何程式碼，請開啟 Cloud Shell 工作階段、使用某個程式碼區塊上的 [複製]**** 按鈕來複製程式碼，然後使用 __Ctrl+Shift+V__ (在 Windows 和 Linux 上) 或 __Cmd+Shift+V__ (在 macOS 上) 將程式碼貼到 Cloud Shell 工作階段中。 粘貼的文本不會自動執行，按**Enter**以運行代碼。

以下有幾種啟動 Cloud Shell 的方式：

|  |   |
|-----------------------------------------------|---|
| 按一下程式碼區塊右上角的 [試試看]****。 | ![本文中的 Cloud Shell](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-powershell-try-it.png) |
| 在您的瀏覽器中開啟 Cloud Shell。 | [![https://shell.azure.com/powershell](./media/vpn-gateway-cloud-shell-powershell/launchcloudshell.png)](https://shell.azure.com/powershell) |
| 按一下 Azure 入口網站右上方功能表上的 [Cloud Shell]**** 按鈕。 | [![門戶中的雲外殼](./media/vpn-gateway-cloud-shell-powershell/cloud-shell-menu.png)](https://portal.azure.com) |
