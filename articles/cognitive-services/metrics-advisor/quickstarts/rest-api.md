# <a name="quickstart-use-the-rest-apis-to-customize-your-solution"></a>快速入門：使用 REST API 自訂解決方案

開始使用 Metrics Advisor REST API。 請遵循下列步驟來安裝套件，並試用基本工作的程式碼範例。

使用 Metrics Advisor 可以：

* 從資料來源新增資料摘要
* 檢查內嵌狀態
* 設定威脅偵測及警示 
* 查詢異常偵測結果
* 診斷異常

## <a name="clean-up-resources"></a>清除資源

如果您想要清除和移除認知服務訂用帳戶，則可以刪除資源或資源群組。 刪除資源群組也會刪除其關聯的任何其他資源。
