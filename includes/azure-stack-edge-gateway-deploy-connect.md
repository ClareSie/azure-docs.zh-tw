1. 開啟瀏覽器視窗，然後在 `https://192.168.100.10` 存取裝置的本機 Web UI。 在您開啟裝置之後，此動作可能需要花費幾分鐘的時間。

    您會看到錯誤或警告指出網站的安全性憑證有問題。 
   
    ![網站安全性憑證錯誤訊息](../articles/databox-online/media/azure-stack-edge-pro-r-deploy-connect/connect-web-ui-2.png)

1. 選取 [繼續瀏覽此網頁]。 這些步驟可能會因您使用的瀏覽器而有所不同。

1. 登入裝置的 Web UI。 預設密碼為 *Password1*。 
   
    ![Azure Stack Edge 裝置的登入頁面](../articles/databox-online/media/azure-stack-edge-pro-r-deploy-connect/connect-web-ui-3.png)


1. 出現提示時，變更裝置系統管理員密碼。  
    新密碼必須包含 8 到 16 個字元。 此密碼必須包含下列其中 3 種字元：大寫、小寫、數字及特殊字元。

您現在位於裝置的 **概觀** 頁面。 下一步是設定裝置的網路設定。
