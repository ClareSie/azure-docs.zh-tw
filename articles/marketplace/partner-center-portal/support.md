# <a name="support-for-the-commercial-marketplace-program-in-partner-center"></a>支援合作夥伴中心的商業市場計劃

Microsoft 提供各式各樣的產品和服務支援。 找到合適的支持團隊對於確保做出適當和及時的反應非常重要。 請考慮下列案例，這些案例應有助於將您的查詢傳送給適當的團隊： 

- 如果您是發佈者,並且對客戶有疑問,請要求客戶使用 [Azure 門戶](https://portal.azure.com/)中的支援連結請求支援。 

- 如果您是發佈者,並且對應用或服務有疑問,請查看以下支援選項。

## <a name="support-options-for-publishers"></a>發行者支援選項

1. 使用工作帳戶登入[合作夥伴中心的商業市場計劃](https://partner.microsoft.com/dashboard/commercial-marketplace/overview)。 (如果您尚未這樣做,則需要[建立合作夥伴中心帳戶](./create-account.md)。

2. 在頁面右側的上部功能表中,選擇 **"支援"** 圖示。 
 
3. **"說明和支援**"窗格將顯示在頁面右側。 
 
   ![支援下拉選單](./media/commercial-marketplace-support-pane.png)

    或者轉到**主頁**窗格並選擇 **「幫助和支援**」。

   ![首頁的說明並支援](./media/homepage-help-support.png)

4. 選擇**[文檔](../index.yml)** 以查看對問題和資源的全面答案。 

5. 選擇**["市場合作夥伴"社區論壇](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222)**,利用其他 Microsoft 發佈者的知識來回答您的問題。 

6. 選擇**[「其他説明](https://partner.microsoft.com/support/v2/?stage=1)**」以打開 **「新建支援請求**票證」。  

## <a name="how-to-open-a-support-ticket"></a>如何開啟支援票證

現在,您已準備好在 **"幫助和支援**"螢幕上打開支援票證。

![說明及支援](./media/help-and-support.png)

>[!Note]
>如果您登錄合作夥伴中心,您將獲得更好的支持體驗。

**選項 1:** 輸入關鍵字,如:應用商店、Azure 應用、SaaS 產品/服務、帳戶管理、潛在顧客管理、部署問題、支出等。

**選項 2:** 瀏覽主題 ->选择**類別**= 商業市場 ->选择适当的**主題**,然後**子主題**。

找到您選擇的主題後,選擇 **「查看解決方案**」。

![後續步驟](./media/next-step.png)

以下選項將變為可用:

* 要選擇其他主題,請在 **「選定問題**」下選擇其他主題連結。
* 查看此問題的說明(如果可用)。  它是**建議步驟**上方所示的文本。
* 查看**建議的步驟**(如果可用)。
* 查看**建議的文件**(如果可用)。

![建議的解決方案](./media/recommended-solutions.png)

如果您無法在**推薦的解決方案**中找到答案,請單擊 **「提供問題詳細資訊**」。  填寫表單和所有必需的欄位,以説明加快解決過程。  然後按下**提交**。

>[!Note]
>如果您尚未登錄合作夥伴中心,並且該主題需要身份驗證,則系統將要求您登錄,然後才能繼續。  對於公共主題,不需要身份驗證。

## <a name="track-your-existing-support-requests"></a>追蹤現有支援要求 

要檢視所有已打開和關閉的票證,請轉到左側導航列上的**商業市場**,然後選擇**支援**。
