# <a name="azure-remote-rendering-asset-tool-arrt"></a>Azure 遠端轉譯資產工具 (ARRT)

Azure 遠端轉譯資產工具 (ARRT) 是以 C++/Qt 開發的開放原始碼桌面應用程式。

![ARRT](./media/azure-remote-rendering-asset-tool.png "ARRT 螢幕擷取畫面")

## <a name="features"></a>特性

應用程式可以用來：

* 上傳 3D 模型
* 控制模型轉換
* 建立和管理遠端轉譯工作階段
* 載入 3D 模型
* 預覽 3D 模型
* 修改其材質

您也可以將這個應用程式作為開放原始碼範例，以了解如何實作 Azure 遠端轉譯 C++ SDK 的前端，並使用 Azure 儲存體用戶端程式庫來管理 3D 模型轉換。

## <a name="source-repository"></a>來源存放庫

您可以在 [GitHub ARRT 存放庫](https://github.com/Azure/azure-remote-rendering-asset-tool)中找到原始程式碼和文件。
