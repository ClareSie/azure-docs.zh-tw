# <a name="monitor-and-manage-wunderlist-in-azure-logic-apps"></a>在 Azure Logic Apps 中監視和管理 Wunderlist

從 2020 5 月6日起，Wunderlist 服務已關閉，並要求客戶遷移其資料並開始使用 Microsoft。 您可以使用 [Microsoft To Do connector](/connectors/todo/) 來取代已淘汰的 [Wunderlist 連接器](https://preview.flow.microsoft.com/connectors/shared_wunderlist/wunderlist/)。
