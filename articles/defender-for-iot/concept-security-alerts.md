# <a name="defender-for-iot-security-alerts"></a>適用于 IoT 的 Defender 安全性警示

Defender for IoT 會使用先進的分析和威脅情報，持續分析您的 IoT 解決方案，以警示您惡意活動。
此外，您還可以根據您對預期裝置行為的瞭解來建立自訂警示。
警示可作為潛在入侵的指標，且應該進行調查和補救。

在本文中，您會找到內建警示的清單，這些警示可以在您的 IoT 中樞和/或 IoT 裝置上觸發。
除了內建警示之外，Defender for IoT 還可讓您根據預期的 IoT 中樞和/或裝置行為來定義自訂警示。
如需詳細資訊，請參閱 [可自訂的警示](concept-customizable-security-alerts.md)。
