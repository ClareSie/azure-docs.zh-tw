# <a name="peering-service-partners"></a>對等互連服務合作夥伴

Microsoft 已與網際網路服務提供者（Isp）、網際網路 exchange 提供者（IXPs），以及全球軟體定義的雲端互連（SDCI）提供者合作，提供可靠且高效能的公用連線，以及從客戶到 Microsoft 網路的最佳路由。

本文提供與 Microsoft 合作提供給客戶的 Azure 對等服務連線的連線提供者相關資訊。
