# <a name="view-how-your-organization-uses-your-privacy-related-data"></a>檢視您組織如何使用您的隱私權相關資料

您可以從**我的帳戶**入口網站的 [**設定] & [隱私權**] 頁面中，查看您的組織如何使用您的資料。

## <a name="view-your-privacy-related-info"></a>檢視您的隱私權相關資訊

1. 登入您的公司或學校帳戶，然後前往 https://myaccount.microsoft.com/ 頁面。

2. 從左側導覽窗格中選取 [**設定] & 隱私權**]，或從 [**設定] & [隱私權**] 區塊選取 [**視圖設定和隱私權**] 連結。

    ![我的帳戶頁面，顯示反白顯示的設定 & 隱私權連結](media/my-account-portal/my-account-portal-privacy.png)

3. 選取 [ **隱私權**]，如下列範例螢幕擷取畫面所示，然後查看資訊，包括：

    ![顯示反白顯示的隱私權索引標籤選項 & 隱私權頁面](media/my-account-portal/my-account-portal-privacy-tab.png)

    - **服務。** 使用您的工作或學校帳戶連線到的線上服務清單。

    - **使用規定**。 您組織的使用規定。

## <a name="next-steps"></a>後續步驟

檢視組織如何使用您的資料後，您可以：

- 檢視或管理您的[安全性資訊](./security-info-setup-signin.md)。

- 檢視或管理您的已連線[裝置](my-account-portal-devices-page.md)。

- 檢視或管理您的[組織](my-account-portal-organizations-page.md)。

- 檢視您的[登入活動](my-account-portal-sign-ins-page.md)。

- 變更 [我的帳戶入口網站設定](my-account-portal-settings.md)

- 您也可以從 [Office 入口網站的 [安全性與隱私權] 頁面](https://portal.office.com/account/#security)，檢視 Microsoft Office 相關的連絡人喜好設定和隱私權資訊。
