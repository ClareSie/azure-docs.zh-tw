### <a name="prepare-an-azure-digital-twins-instance"></a>準備 Azure Digital Twins 執行個體

若要完成本教學課程，您需要**Azure Digital Twins 執行個體**以進行程式設計。 

如果您已經透過先前的工作設定 Azure Digital Twins 執行個體，您可以使用該執行個體，並跳至下一節。

否則，請遵循下列步驟來設定您的執行個體：
1. **使用下列指示建立執行個體**[操作說明：建立 Azure Digital Twins 執行個體](../articles/digital-twins/how-to-set-up-instance.md)。 
2. **針對您的執行個體設定 Azure Active Directory 應用程式註冊**，方法是*建立應用程式註冊*一節所述的方式[操作說明：驗證用戶端應用程式](../articles/digital-twins/how-to-authenticate-client.md#create-an-app-registration)。
