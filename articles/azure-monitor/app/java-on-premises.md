# <a name="java-codeless-application-monitoring-on-premises---azure-monitor-application-insights---public-preview"></a>本地無程式應用程式監視 - Azure 監視器應用程式見解 ─ 一般預覽

Java 無程式應用程式監視是簡單性的 - 沒有代碼更改,JAva代理只能通過幾個配置更改來啟用。

## <a name="overview"></a>概觀

啟用後,Java 代理將自動從使用最廣泛的庫和框架收集大量請求、依賴項、日誌和指標。

請按照所有環境(包括本地)的[詳細說明進行](https://docs.microsoft.com/azure/azure-monitor/app/java-in-process-agent)操作。
