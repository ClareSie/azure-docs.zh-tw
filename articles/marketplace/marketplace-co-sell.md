# <a name="co-sell-partner-engagement"></a>共同銷售合作夥伴參與

Microsoft 的共同銷售方案可讓合作夥伴直接與 Microsoft 合作銷售商機。 您可以觸及廣大的客戶群體，創造新的銷售機會，協助加速業務成長。

## <a name="partner-selling-motions"></a>合作夥伴銷售運動

有三個合作夥伴銷售運動：

* *透過合作夥伴銷售*：您可以透過自己的賣方轉售 Microsoft 技術。
* *服務共同銷售*：銷售人員和合作夥伴共同作業，透過內嵌 Microsoft 技術來銷售您的專案或受控服務。
* *IP 共同銷售*：賣方/合作夥伴會共同作業，以銷售可重複的封裝 IP 解決方案 *。

\*當您成功銷售及部署 Microsoft 的現場銷售團隊時，某些 IP 共同銷售解決方案可能會獲得額外的獎勵。
