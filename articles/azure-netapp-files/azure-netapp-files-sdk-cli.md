# <a name="azure-netapp-files-sdks-cli-tools-and-arm-templates"></a>Azure NetApp Files Sdk、CLI 工具和 ARM 範本

本文列出 Azure NetApp Files 所支援的 Sdk、命令列 (CLI) 工具和 Azure Resource Manager (ARM) 範本。

## <a name="supported-sdks"></a>支援的 SDK 

下表列出支援的 Sdk。  您可以在 GitHub 中的已發佈位置找到支援的 Sdk 詳細資料。  

|    語言    |    GitHub 中的 SDK 發佈位置    |
|------------------|--------------------------------------------------------------|
|    .NET  |    [Azure/azure---.net](https://github.com/Azure/azure-sdk-for-net/tree/master/sdk/netapp)    |
|    Python  |  [Azure/azure-適用于 python 的 sdk](https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/netapp)    |
|    Go    |    [Azure/azure-適用于 go 的 sdk](https://github.com/Azure/azure-sdk-for-go/tree/master/services/netapp)       |
|    Java |     [Azure/azure-適用于 java 的 sdk](https://github.com/Azure/azure-sdk-for-java/tree/master/sdk/netapp) |
|    JavaScript    |    [Azure/azure-sdk-for-js](https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/netapp/arm-netapp)    |
|    Ruby   |    [Azure/azure-適用于 ruby 的 sdk](https://github.com/Azure/azure-sdk-for-ruby/tree/master/management/azure_mgmt_netapp)    |

## <a name="azure-resource-manager-templates"></a>Azure 資源管理員範本  

Azure Resource Manager (ARM) 可讓您使用宣告式範本布建應用程式。 在單一的範本中，您可以部署多個服務及其相依性。 您可以使用相同的範本，在應用程式生命週期的每個階段重複部署應用程式。   

查看 [適用于 Azure NetApp Files 的 ARM 範本](https://azure.microsoft.com/resources/templates/?term=anf)。
