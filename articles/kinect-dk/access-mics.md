# <a name="access-azure-kinect-dk-microphone-input-data"></a>存取 Azure Kinect DK 麥克風輸入資料

[語音 SDK 快速入門](../cognitive-services/speech-service/index.yml)提供如何在各種程式設計語言中使用 Azure Kinect DK 麥克風陣列的範例。
例如，請參閱 **使用語音 SDK 快速入門中的在 Windows 上以 c + + 辨識語音** 。 您可以 [從 GitHub](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/cpp)取得此程式碼。

也可以透過 Windows API 存取麥克風陣列。 如需 Windows 檔的詳細資訊，請參閱下列檔：

* [Windows 音訊架構](/windows-hardware/drivers/audio/windows-audio-architecture)
* [Windows Media. Capture 檔](/uwp/api/Windows.Media.Capture)
* [網路攝影機捕獲教學課程](/windows/uwp/audio-video-camera/basic-photo-video-and-audio-capture-with-mediacapture)
* [USB 音訊資訊](/windows-hardware/drivers/audio/usb-2-0-audio-drivers)

您也可以參閱 [麥克風陣列硬體規格](hardware-specification.md#microphone-array)。
