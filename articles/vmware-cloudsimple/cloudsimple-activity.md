# <a name="activity-management-overview"></a>活動管理概述

CloudSimple 跟蹤可能影響私有雲環境功能的所有活動。 活動包括警報、事件、任務和審核活動。 ["活動"頁面](monitor-activity.md)匯總了所有當前活動，並允許您向下切入以獲取其他詳細資訊。

## <a name="events"></a>事件

事件跟蹤雲簡單門戶上的使用者和系統活動。 事件顯示與特定資源關聯的活動以及影響的嚴重性。  您可以在雲簡單門戶中查看事件。

## <a name="alerts"></a>警示

警報是雲簡單環境中任何重要活動的通知。 影響計費或使用者訪問的事件顯示為警報。  您可以確認來自雲簡單門戶的警報。

## <a name="tasks"></a>工作

任務跟蹤任何需要 30 秒以上才能完成的使用者操作。 您可以通過 CloudSimple 門戶監視任務的進度。  對於已完成的任務，該資訊包括完成的總時間。

## <a name="audit"></a>稽核

稽核記錄跟蹤使用者操作。 稽核記錄包含使用者為操作提供的參數。  您可以使用稽核記錄監視所有使用者的使用者活動。
