# <a name="getting-started-with-azure-education-hub"></a>開始使用 Azure Education Hub

Education Hub [開始使用] 頁面會在第一次登陸至 Education Hub 時提供快速連結。 您可以在那裡找到如何設定課程的相關資訊、透過 Microsoft Learn 了解不同的服務，或透過快速入門範本輕鬆部署您的第一個服務。

:::image type="content" source="media/get-started-education-hub/get-started-page.png" alt-text="Azure Education Hub 中的 [開始使用] 頁面。" border="false":::

根據預設，後續登入之後，您會看到 [概觀] 頁面。
