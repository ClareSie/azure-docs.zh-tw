# <a name="quickstart-use-the-computer-vision-client-library"></a>快速入門：使用電腦視覺用戶端程式庫

開始使用電腦視覺用戶端程式庫。 請遵循下列步驟來安裝套件，並試用基本工作的程式碼範例。 電腦視覺可供您存取進階演算法，以處理影像及傳回資訊。

使用電腦視覺用戶端程式庫可執行下列作業：

* 分析影像中的標籤、文字描述、臉部、成人內容等等。
* 透過 Batch 讀取 API 辨識印刷和手寫文字。
