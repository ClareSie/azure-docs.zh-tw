# <a name="cloud-partner-portal-migration-to-partner-center"></a>從 Cloud Partner 入口網站遷移到合作夥伴中心

為了簡化您的帳戶並提供管理體驗，Microsoft 正在將 Cloud Partner 入口網站的供應專案遷移至合作夥伴中心。 如果您目前在一個商業夥伴的進入市場（OCP 轉型）目錄中管理共同銷售的宣傳，您很快就可以在合作夥伴中心更新您的共同銷售宣傳。

| **供應專案類型** | **從 Cloud Partner 入口網站遷移至合作夥伴中心的供應專案管理** | **能夠在合作夥伴中心更新共同銷售的宣傳** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| 軟體即服務（SaaS） |     現已推出       |   公開上市2020年4月      |
| Dynamics 365 for Customer Engagement & PowerApps      |     現已推出        |   現已推出      |
| Dynamics 365 for Operations     |     現已推出       |   現已推出     |
| Dynamics 365 Business Central      |     現已推出        |   公開上市2020年4月     |
| Azure 應用程式     |     現已推出       |   公開上市2020年4月     |
| Azure 虛擬機器      |     年4月正式運作-5 月2020       |   即將宣布    |
| Azure 容器     |     年4月正式運作-5 月2020       |   即將宣布     |
| 諮詢服務      |     年4月正式運作-5 月2020       |   即將宣布     |
| IoT Edge 模組      |     年4月正式運作-5 月2020       |   即將宣布     |
| Power BI 服務應用程式     |     年4月正式運作-5 月2020       |   即將宣布     |
