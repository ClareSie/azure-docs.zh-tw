虛擬機器 （VM） 將根據部署 VM 的子網從您指定的範圍自動分配私人 IP 位址。 VM 保留位址，直到刪除 VM。 Azure 從創建 VM 的子網動態分配下一個可用的私人 IP 位址。 如果希望從分配給 VM 的子網中分配特定的 IP 位址，請分配靜態 IP 位址。
