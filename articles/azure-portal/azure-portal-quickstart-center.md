# <a name="get-started-with-the-azure-quickstart-center"></a>開始使用 Azure Databricks Center

Azure 快速入門中心是 Azure 入口網站的引導式體驗，可供任何想要改善其 Azure 知識的人員使用。 對於 Azure 新手的組織而言，這是上架及設定雲端環境的最快方式。

## <a name="overview"></a>概觀

Azure 快速入門中心有三個入門選項：

* **設定指南**：專為 IT 系統管理員和雲端架構設計人員所設計，我們的指南會介紹 Azure 採用的重要概念。 結構化步驟可協助您在學習時採取行動，並套用 Microsoft 的建議最佳作法。 當您準備轉移至 Azure 時，「遷移指南」可協助您評估準備和規劃。

* **啟動專案**：如果您已準備好建立資源，此區段可讓您在認可至服務選項之前，先深入瞭解您的選擇。 您將探索更多關於此服務的資訊，以及您應該使用它的原因、探索成本，以及識別必要條件。 進行選擇之後，您就可以直接開始建立。

* **線上學習**： Azure 快速入門中心的這一節會強調 Microsoft Learn 的免費簡介課程模組。 選取磚以啟動課程，並深入瞭解雲端概念和在 Azure 中管理資源。

## <a name="how-to-use-azure-quickstart-center"></a>如何使用 Azure 快速入門中心

1. 登入 [Azure 入口網站](https://portal.azure.com)。

1. 從 Azure 入口網站] 功能表中選取 [ **所有服務** ]。

1. 選取 **[一般**  >  **快速入門中心**]。
