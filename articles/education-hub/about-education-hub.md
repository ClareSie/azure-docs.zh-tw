# <a name="what-is-the-azure-education-hub"></a>什麼是 Azure Education Hub？

Microsoft Azure [Education Hub](https://portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/quickstart) 是一項工具，可協助學術使用者在許多不同的 Azure 訂用帳戶中佈建和管理雲端點數。 當教職員成員需要管理許多雲端式學生專案時，這十分實用。 當您尚未得知是否需要 Azure 點數時，Education Hub 也很適合用於研究用途。

使用者可以輕鬆地調整每個訂用帳戶內已配置的點數，或大量調整。 您也可以將訂用帳戶暫止 (例如，當學期或季專案完成時)，以充分利用可用的點數。

## <a name="prerequisites"></a>必要條件

若要存取 Education Hub，您應該已經收到已核准學術用途授與的電子郵件通知。 此電子郵件包含系統核准您使用的的點數數目，以及 [Education Hub](https://aka.ms/startedu) 的連結。

若要使用學術授與權限，您必須在 Education Hub 中建立教室，並在教室中使用會存取您 Azure 點數的訂用帳戶。
