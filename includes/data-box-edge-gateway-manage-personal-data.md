- **訂單詳細資訊**。 創建訂單時，使用者的運輸位址、電子郵件地址和聯繫資訊將存儲在 Azure 門戶中。 儲存的資訊包括︰
  - 連絡人姓名
  - 電話號碼
  - 電子郵件地址
  - 街道地址
  - City
  - 郵遞區號
  - State
  - 國家/省/地區
  - 運送追蹤號碼

    訂單詳細資訊經過加密並存儲在服務中。 該服務保留資訊，直到您顯式刪除資源或訂單。 從設備發貨到設備返回 Microsoft，將阻止刪除資源和相應的訂單。

- **送貨位址**。 下訂單後，資料箱服務向協力廠商運營商（如 UPS）提供送貨位址。

- **共用使用者**。 設備上的使用者還可以訪問共用上的資料。 可以查看可以訪問共用資料的使用者清單。 刪除共用時，也會刪除此清單。
