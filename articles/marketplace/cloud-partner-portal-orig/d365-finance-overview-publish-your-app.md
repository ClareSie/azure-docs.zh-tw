# <a name="bring-your-microsoft-dynamics-365-business-central-app-into-microsoft-appsource"></a>將 Microsoft Dynamics 365 Business Central 應用程式發佈到 Microsoft AppSource 中

[雲合作夥伴門戶](https://cloudpartner.azure.com/)不再支援創建和管理 Dynamics 365 業務中心產品/ 相反，此功能已移動到 Microsoft[合作夥伴中心門戶](https://partner.microsoft.com/)。 有關詳細資訊，請參閱[創建新的 Dynamics 365 業務中心產品/](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-new-business-central-offer)

## <a name="next-steps"></a>後續步驟

在創建新的 Dynamics 365 商業中心產品/服務之前，應閱讀["創建合作夥伴中心"帳戶](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-account)，然後在選中 **"概覽"** 頁後打開["商業市場"儀表板](https://partner.microsoft.com/dashboard/directory)。 按照[創建新動態 365 商業中心產品/服務](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-new-business-central-offer)中的說明創建您的產品/服務。
