# <a name="update-your-profile-and-account-information-on-the-my-apps-portal"></a>在"我的應用"門戶上更新您的個人資料和帳戶資訊

您可以將工作或學校帳戶與基於 Web**的"我的應用"** 門戶一起使用， 

- 查看和啟動組織的許多基於雲的應用
- 更新您的一些個人資料和帳戶資訊
- 查看**您的群組**資訊
- 執行應用和組**的訪問審核**。 

如果您沒有 [我的應用程式]**** 入口網站的存取權，則必須連絡技術服務人員以取得權限。

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

> [!Important]
> 本內容適用於**我的應用程式**使用者。 如果您是系統管理員，可以在[應用程式管理文件](https://docs.microsoft.com/azure/active-directory/manage-apps)中找到更多關於如何設定和管理雲端式應用程式的資訊。

## <a name="view-your-organization-related-profile-information"></a>查看與組織相關的設定檔資訊

根據您的組織選擇在 **"我的應用"** 門戶的**個人資料**頁面上包含的內容，您可能會看到特定與工作相關的詳細資訊以及您的設備和活動以及您所屬的任何其他組織。

### <a name="to-view-your-profile-information"></a>查看您的個人資料資訊

1. 登錄到您的工作或學校帳戶，[轉到 **"我的應用"** 門戶](my-apps-portal-end-user-access.md)。

2. 在 **"應用"** 頁面中，從右上角選擇您的個人資料圖片，其中顯示您的姓名和組織，然後選擇 **"設定檔**"。

    將顯示 **"設定檔"** 頁，顯示您的個人資料資訊。

    ![設定檔頁面，顯示所有可用資訊](media/my-apps-portal/my-apps-portal-profile-page.png)

3. 在 **"設定檔"** 頁面中，您可以：

    - **查看與組織相關的詳細資訊。** 查看您的照片、姓名、標題、關聯的電子郵件地址和辦公室資訊。 此資訊由您的組織管理，您將無法更改它。 如果您看到錯誤，請聯繫您的説明台。

    - **查看您的設備&活動**。 確保每個設備都熟悉並正確連接到您的組織。 如果您無法識別設備，請選擇 **"禁用設備"** 以刪除與您的帳戶的關聯。 禁用設備後，設備將從此頁面中刪除。

    - **查看您的組織。** 確保您仍與每個指定的組織一起工作。 如果您不再與組織合作，我們強烈建議您選擇 **"登錄"以離開組織**。 離開組織後，它將從此頁面中刪除。

## <a name="manage-your-work-or-school-account-information"></a>管理您的工作或學校帳戶資訊

從 **"我的應用"** 門戶的 **"設定檔"** 頁面更新和管理您的工作或學校帳戶資訊。 此頁面可説明您：

- 更改工作或學校帳戶密碼。

- 打開密碼重設（如果您的管理員已打開此功能）。

- 提供其他安全驗證資訊。

- 查看組織的使用條款。

- 從任何地方登出。

## <a name="change-your-password"></a>變更密碼

如果要更改工作或學校帳戶的密碼，可以從 **"設定檔"** 頁的 **"管理帳戶**"區域選擇 **"更改密碼**"。

### <a name="to-change-your-password"></a>更改密碼

1. 從 **"設定檔"** 頁的 **"管理帳戶**"區域中，選擇 **"更改密碼**"。

2. 在 **"更改密碼"** 頁上，請確保您的使用者 ID 正確，然後在框中鍵入舊密碼和新密碼。

    ![更改密碼頁](media/my-apps-portal/my-apps-portal-change-password-page.png)

3. 選取 [提交]****。

    您的密碼已更改。 您需要登錄到以前登錄使用工作或學校帳戶的任何應用。

## <a name="set-up-and-use-password-reset"></a>設置和使用密碼重設

如果您忘記了密碼，從未從公司支援獲得密碼，或者帳戶被鎖定，您可以重置自己的密碼。

>[!Important]
>管理員必須打開此功能，您必須註冊才能參與。 有關如何註冊以及如何重置密碼的詳細資訊，請參閱[註冊自助服務密碼重設](active-directory-passwords-reset-register.md)和[重置工作或學校密碼](active-directory-passwords-update-your-own-password.md)。

## <a name="change-your-security-verification-information"></a>更改安全驗證資訊

如果您的組織要求您使用雙重驗證，則可以從 **"其他安全驗證**"頁添加、更新和刪除關聯的安全資訊。

雙重驗證要求您使用兩段驗證資訊，如密碼和 PIN，然後才能訪問您的帳戶或組織的資訊。 有關雙因素驗證的詳細資訊，請參閱[設置我的帳戶進行兩步驗證](multi-factor-authentication-end-user-first-time.md)。

### <a name="to-change-your-security-information"></a>更改安全資訊

1. 從 **"設定檔"** 頁的 **"管理帳戶**"區域中，選擇 **"其他安全驗證**"。

    ![其他安全驗證頁](media/my-apps-portal/my-apps-portal-additional-verification-page.png)

2. 在 **"其他安全驗證**"頁上，您可以添加、更改或刪除以下資訊：

    - **預設驗證選項。** 選擇用於雙因素驗證的預設輔助方法。 輸入使用者名和密碼後，任何時間都需要雙重驗證時，都會自動使用此方法。

    - **添加、更新或刪除驗證方法。** 您可以添加新、更新現有或刪除不再適用的舊資訊。

    - **設置 Microsoft 身份驗證器應用。** 您可以選擇將 Microsoft 身份驗證器應用設置為驗證方法。 有關 Microsoft 身份驗證器應用的詳細資訊，請參閱什麼是[Microsoft 身份驗證器應用？](user-help-auth-app-overview.md)

3. 選取 [儲存]**** 來儲存變更。

## <a name="review-your-organizations-terms-of-use-statement"></a>查看組織的使用條款聲明

您可以選擇查看組織的使用條款聲明（如果可用）。

1. 從 **"設定檔"** 頁的 **"管理帳戶**"區域中，選擇 **"查看使用條款**"。

2. 查看組織的使用條款**並選擇"接受"** 以確認您已閱讀並理解組織的使用條款。

    !["我的應用"門戶中的使用條款頁面](media/my-apps-portal/my-apps-portal-tou-page.png)

    如果您的組織沒有使用條款，則可以選擇 **"完成"** 以返回到 **"設定檔"** 頁。

## <a name="sign-out-of-everywhere"></a>登出無處不在

您可以選擇在您當前登錄到您的工作或學校帳戶的任何地方登出。 這包括所有應用和設備。

### <a name="to-sign-out-of-everywhere"></a>到處登出

1. 從 **"設定檔"** 頁的 **"管理帳戶**"區域中，選擇 **"隨處登出**"。

2. 在"**到處登出"** 確認框中，選擇 **"是**"以確認要登出所有會話和設備。 如果您改變主意，請選擇 **"否**"。
