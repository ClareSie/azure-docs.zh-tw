# <a name="storsimple-8000-series-update-51-release-notes"></a>StorSimple 8000 系列更新 5.1 發行說明

## <a name="overview"></a>概觀

以下發行說明描述了新功能,並確定了 StorSimple 8000 系列更新 5.1 的關鍵打開問題。 當中也包含此版本中隨附之 StorSimple 軟體更新的清單。

更新 5.1 可應用於運行更新 5 的任何 StorSimple 設備。 如果使用的版本低於 5,請先應用更新 5,然後應用 5.1。 與 Update 5.1 關聯的設備版本為 6.3.9600.17885。

在 StorSimple 方案中部署更新之前，請檢閱版本資訊中所包含的資訊。

> [!IMPORTANT]
>
> * 更新 5.1 是強制更新,必須立即安裝。 有關詳細資訊,請參閱如何[應用更新 5.1](storsimple-8000-install-update-51.md)。
> * 更新 5.1 僅具有安全更新。 安裝此更新大約需要 30 分鐘。 我們強烈建議您應用 Update 5.1 以確保設備運行。
> * 如果是新版本，您可能不會立即看到更新，因為我們會分階段推出更新。 請等候數天然後再次掃描更新，因為很快就會提供這些更新。

## <a name="whats-new-in-update-51"></a>更新 5.1 中的新增功能

在 Update 5.1 中進行了以下關鍵改進和錯誤修復:

* **TLS 1.2** - 此簡單更新將強制 TLS 1.2 對所有用戶端。 這是所有 StorSimple 8000 系列設備的必修更新。

## <a name="known-issues-in-update-51-from-previous-releases"></a>以前版本中更新 5.1 中的已知問題

更新 5.1 中沒有新的已知問題。 有關從以前的版本轉到 Update 5.1 的問題清單,請轉到['更新 3"發行說明](storsimple-update3-release-notes.md#known-issues-in-update-3)。

## <a name="storsimple-cloud-appliance-updates-in-update-51"></a>更新 5.1 中的 StorSimple 雲端裝置更新

這項更新無法套用至 StorSimple Cloud Appliance (也稱為虛擬裝置)。 需要使用 Update 5.1 映射創建新的雲端裝置。 如需如何建立 StorSimple 雲端設備的資訊，請移至[部署和管理 StorSimple 雲端設備](storsimple-8000-cloud-appliance-u2.md)。
