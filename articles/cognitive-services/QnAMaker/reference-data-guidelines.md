# <a name="import-and-export-data-reference"></a>導入和匯出資料引用

查看此導入和匯出參考，以獲得知識庫備份、存儲和替換的最佳結果。

## <a name="import-and-export-knowledge-base"></a>進出口知識庫

**TSV 和 XLS 檔**（來自匯出的知識庫）只能通過從 QnA Maker 門戶中的 **"設置"** 頁導入檔來使用。 在創建知識庫期間，也不能將其用作資料來源，也不能從 **"設置"** 頁上的 **"添加檔**"或 **"添加 URL"** 功能使用。
