# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>立即取得 - 從 Azure Marketplace 新增應用程式

就快完成了！ 

如果您嘗試使用 Azure AD 作為應用程式的識別提供者，那麼您找對地方了。 您只需要將其新增至 Azure AD 租用戶即可。 若要了解如何執行此操作，請遵循以下的快速入門系列：[檢視 Azure AD 租用戶中的應用程式](view-applications-portal.md)。

如果您嘗試在本機電腦或行動裝置上安裝應用程式，那麼您找錯地方了。 Azure Marketplace 是針對搭配其 Azure 租用戶使用應用程式的組織所設計的。 若為個人電腦和裝置，請參閱 [Microsoft 應用程式市集](https://www.microsoft.com/store/apps)。
