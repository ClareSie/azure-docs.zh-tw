# <a name="sign-in-activity-report-error-codes"></a>登入活動報告錯誤碼 

使用[使用者登錄報告](concept-sign-ins.md)提供的資訊，您可以找到問題的答案，例如：

- 誰登入我的應用程式？
- 已登入哪些應用程式？
- 哪些登錄失敗，為什麼？

當登錄失敗時，您將看到與失敗對應的錯誤代碼。 [查找錯誤代碼的說明](https://login.microsoftonline.com/error)。
