## <a name="prerequisites"></a>Prerequisites

- 使用 Bash 環境的 [Azure Cloud Shell](/azure/cloud-shell/quickstart)。

   [![內嵌啟動](https://shell.azure.com/images/launchcloudshell.png "啟動 Azure Cloud Shell")](https://shell.azure.com)  
- 如果您想要的話，請執行 Azure CLI 的[本機安裝](/cli/azure/install-azure-cli)以執行 CLI 參考命令。
   - 如果您使用的是本機安裝，請使用 [az login](/cli/azure/reference-index#az-login) 命令，透過 Azure CLI 來登入。  請遵循您終端機上顯示的步驟，來完成驗證程序。  如需其他登入選項，請參閱[使用 Azure CLI 登入](/cli/azure/authenticate-azure-cli)。
  - 出現提示時，請在第一次使用時安裝 Azure CLI 擴充功能。  在[使用擴充功能搭配 Azure CLI](/cli/azure/azure-cli-extensions-overview) 中深入了解擴充功能。
  - 執行 [az version](/cli/azure/reference-index?#az_version) 以尋找已安裝的版本和相依程式庫。 若要升級至最新版本，請執行 [az upgrade](/cli/azure/reference-index?#az_upgrade)。
