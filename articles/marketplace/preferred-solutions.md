# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Azure Marketplace 和 AppSource 中的慣用解決方案

Microsoft 慣用解決方案是指根據品質、效能，以及在特定產業垂直市場或解決方案領域中滿足客戶需求的能力等條件，所選出的雲端應用程式。 一組 Microsoft 專家從具有特定經過驗證的能力和功能的合作夥伴處驗證解決方案。 這些解決方案在我們的雲市場網店、Azure 應用商店和 AppSource 以及 Azure 門戶中都有特色。 AppSource 上的首選解決方案可以通過行業垂直部發現。 Azure 應用商店和 Azure 門戶上的首選解決方案可以跨水準解決方案類別發現。
