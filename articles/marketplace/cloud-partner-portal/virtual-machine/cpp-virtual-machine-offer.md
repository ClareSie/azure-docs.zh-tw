# <a name="azure-virtual-machine-offer"></a>Azure 虛擬機器供應項目

[Cloud Partner 入口網站](https://cloudpartner.azure.com/)不再支援建立和管理 Azure 虛擬機器供應項目。 此功能已移至 Microsoft [合作夥伴中心](https://partner.microsoft.com/pcv/)。 如需詳細資訊，請參閱[建立 Azure 虛擬機器供應項目](../../partner-center-portal/azure-vm-create-offer.md)。
