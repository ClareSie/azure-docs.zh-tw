# <a name="release-notes-azure-event-grid-on-iot-edge"></a>版本資訊： IoT Edge 上的 Azure 事件方格

## <a name="100-preview1"></a>1.0.0-preview1

IoT Edge 上的 Azure 事件方格的初始版本。 包含的功能：

* 主題建立
* 建立事件訂閱
* 進階篩選
* 輸出批次處理
* 重試原則
* 模組對模組發佈
* 以目的地的形式發行至 WebHook
* 以目的地的形式發佈至 IoT Edge 中樞
* 以目的地的形式發佈至 Azure 事件方格雲端服務
* 中繼資料的保存狀態
* Blob 儲存體模組整合

標籤： `1.0.0-preview1`

## <a name="100-preview2"></a>1.0.0-preview2

已新增 IoT Edge 的 Azure 事件方格 Preview 2：

* 可設定將事件保存到磁片
* 主題計量
* 事件訂用帳戶計量
* 以目的地的形式發佈到事件中樞
* 以目的地的形式發行至服務匯流排佇列
* 以目的地的形式發行至服務匯流排主題
* 以目的地的形式發行至儲存體佇列

標記： `1.0.0-preview2` 、 `1.0` 、 `latest`
