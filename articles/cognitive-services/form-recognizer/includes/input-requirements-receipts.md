* 支援的檔案格式： JPEG、PNG、BMP、PDF 和 TIFF 
* 針對 PDF 和 TIFF，最多會處理2000個頁面。 若為免費層訂閱者，則只會處理前兩個頁面。 
* 檔案大小必須小於 50 MB，以及至少 50 x 50 圖元和最多 10000 x 10000 圖元的大小。 
* PDF 尺寸必須最多為 17 x 17 英寸，對應于合法或 A3 紙張大小，且較小。
