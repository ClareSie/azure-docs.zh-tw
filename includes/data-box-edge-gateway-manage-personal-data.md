- **訂單詳細資料**。 建立訂單時，會將使用者的交貨位址、電子郵件地址和連絡人資訊儲存在 Azure 入口網站中。 儲存的資訊包括︰
  - 連絡人姓名
  - 電話號碼
  - 電子郵件地址
  - 街道地址
  - City
  - 郵遞區號
  - 狀況
  - 國家/省/地區
  - 運送追蹤號碼

    訂單詳細資料會經過加密，並儲存在服務中。 服務會保留資訊，直到您明確刪除資源或訂單為止。 刪除資源和對應的訂單，會在裝置寄送至 Microsoft 之前遭到封鎖。

- **寄送位址**。 訂購之後，資料箱服務會提供運送位址給協力廠商電信公司，例如 UPS。

- **共用使用者**。 您裝置上的使用者也可以存取位於共用上的資料。 可以查看可以存取共用資料的使用者清單。 刪除共用時，也會一併刪除此清單。
