- 若要存取 Data Box Edge 服務，您的組織必須有 Enterprise 合約 (EA) 或雲端解決方案提供者 (CSP) 訂用帳戶。 如需詳細資訊，請參閱 [註冊 Azure 訂](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/)用帳戶。
- 因為此管理服務是裝載在 Azure 中，所以會受到 Azure 安全性功能的保護。 如需 Azure 提供之安全性功能的詳細資訊，請移至 [Microsoft Azure 信任中心](https://azure.microsoft.com/support/trust-center/security/)。
- 針對 SDK 管理作業，您可以在 **裝置屬性** 中取得資源的加密金鑰。 只有當您有 Resource Graph API 的許可權時，才可以看到加密金鑰。
