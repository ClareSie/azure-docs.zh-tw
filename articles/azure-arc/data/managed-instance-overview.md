# <a name="azure-arc-enabled-sql-managed-instance-overview"></a>啟用 Azure Arc 的 SQL 受控執行個體概觀

啟用 azure Arc 的 SQL 受控執行個體是一項 Azure SQL 資料服務，可建立在您選擇的基礎結構上。 

[!INCLUDE [azure-arc-data-preview](../../../includes/azure-arc-data-preview.md)]

## <a name="description"></a>描述

啟用 Azure Arc 的 SQL 受控執行個體可提供與最新 SQL Server 資料庫引擎接近 100% 的相容性，並可讓現有的 SQL Server 客戶只需變更最少的應用程式和資料庫項目，就能將其應用程式隨即轉移至 Azure Arc 資料服務，並維護資料主權。 SQL 受控執行個體也包含內建的管理功能，可大幅降低管理負擔。 

啟用 Azure Arc 的 SQL 受控執行個體目前處於預覽狀態。
