您可以[免費試用 Azure Cosmos DB](https://azure.microsoft.com/try/cosmosdb/)，而不需要 Azure 訂用帳戶；不但可免費使用也不需承諾用量；也可以建立 [Azure Cosmos DB 免費層帳戶](../articles/cosmos-db/optimize-dev-test.md#azure-cosmos-db-free-tier)，其中前 400 RU/秒和 5 GB 的儲存體免費。 您也可以搭配使用 [Azure Cosmos DB 模擬器](https://docs.microsoft.com/azure/cosmos-db/local-emulator)與 `https://localhost:8081` 的 URI。 如需要搭配模擬器使用的金鑰，請參閱[驗證要求](../articles/cosmos-db/local-emulator.md#authenticating-requests)。
