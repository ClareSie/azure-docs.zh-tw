# <a name="application-and-version-settings"></a>應用程式和版本設定

在 LUIS 入口網站中設定您的應用程式設定，例如表達正規化和應用程式隱私權。

## <a name="view-application-name-description-and-id"></a>檢視應用程式名稱、描述和識別碼

您可以編輯您的應用程式名稱與描述。 您可以複製您的應用程式識別碼。 文化特性無法變更。

1. 登入 [LUIS 入口網站](https://www.luis.ai)。
1. 從 [我的應用程式] 清單中選取應用程式。
。
1. 從頂端導覽列選取 [管理]，然後從左側導覽列選取 [設定]。
