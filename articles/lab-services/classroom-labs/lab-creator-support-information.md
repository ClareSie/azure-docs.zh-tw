# <a name="view-support-information-lab-creator-in-azure-lab-services"></a>查看支援資訊（Azure 實驗室服務中的實驗室建立者）
本文介紹您（作為實驗室建立者）如何查看以下支援資訊：

- URL
- 電子郵件
- 電話
- 其他指示

在實驗室帳戶中創建實驗室時遇到任何技術問題時，可以使用此資訊獲取説明。

 
## <a name="view-support-information"></a>檢視支援資訊
1. 登錄到[Azure 實驗室服務門戶](https://labs.azure.com)。
2. 在頁面右上角選擇問號 （**？** 
3. 確認您看到指向**視圖支援網站**、**電子郵件支援****和支援電話號碼的連結**。

    ![檢視支援資訊](../media/lab-creator-support-information/support-information.png)
