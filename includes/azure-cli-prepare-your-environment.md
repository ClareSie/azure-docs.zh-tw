## <a name="prepare-your-environment"></a>準備您的環境

1. 若要執行 Azure CLI 命令，請執行 Azure CLI 的[本機安裝](/cli/azure/install-azure-cli)，或使用 bash 環境啟動 [Azure Cloud Shell](/azure/cloud-shell/quickstart)。

1. 如果您使用的是本機安裝，請使用 [az login](/cli/azure/reference-index#az-login) 命令，透過 Azure CLI 來登入。  請遵循您終端機上顯示的步驟，來完成驗證程序。  如需其他登入選項，請參閱[使用 Azure CLI 登入](/cli/azure/authenticate-azure-cli)。
