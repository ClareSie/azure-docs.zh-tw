# <a name="accessing-the-azure-education-hub"></a>存取 Azure Education Hub

若要存取 Azure Education Hub，您應該已經收到通知您已通過「學術授與」核准的電子郵件通知。 此電子郵件包含系統核准您使用的點數數目，以及 [Education Hub](https://aka.ms/startedu) 的連結。 最受支援的學術供應項目包含免費的 Azure 支援開發人員層級。 如果您在設定過程中遇到任何問題，請洽詢[教育工作者服務中心](mailto:azuredu@microsoft.com)或連絡當地的 Microsoft 代表。

> [!IMPORTANT]
> 若要使用學術授與權限，您必須在 Education Hub 中建立教室，並在教室中使用會存取您 Azure 點數的訂用帳戶。

## <a name="how-to-access-the-azure-education-hub"></a>如何存取 Azure Education Hub

1. 在瀏覽器中，開啟私密瀏覽索引標籤，例如 Chrome 的 Incognito 或 Microsoft Edge 的 InPrivate。

    :::image type="content" source="media/access-education-hub/browsing-in-private.png" alt-text="開啟私密瀏覽器視窗。" border="false":::

1. 使用您的 Microsoft 認證登入 [Azure 入口網站](https://portal.azure.com)。

   > [!IMPORTANT]
   > 確認您是使用組織/公司帳戶 (例如貴機構的 @domain.edu) 進行登入。 若是如此，請先在視窗左側選取此選項。 這會將您帶到不同的登入畫面。

    :::image type="content" source="media/access-education-hub/sign-in.png" alt-text="組織登入對話方塊。" border="false":::

1. 登入之後，系統會將您導向至 Azure 入口網站。 若要尋找 Education Hub，請移至 [所有服務] 功能表，然後搜尋 [教育]。
   第一次登入時，會顯示 [開始使用] 頁面。
