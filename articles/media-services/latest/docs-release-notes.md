# <a name="azure-media-services-v3-documentation-updates"></a>Azure 媒體服務 v3 文件更新

>以複製和貼上此網址`https://docs.microsoft.com/api/search/rss?search=%22Azure+Media+Services+v3+documentation+updates%22&locale=en-us`: 到 RSS 源閱讀器,獲得有關何時重新存取此頁面以進行更新的通知。

本文介紹最新的媒體服務 v3 文件更新。

## <a name="april-2020"></a>2020 年 4 月

* Azure 媒體播放器文件已移到[Azure 文件](../azure-media-player/azure-media-player-overview.md)。
* 添加了[帶開放廣播工作室 (OBS)](live-events-obs-quickstart.md)快速啟動的即時流。 它演示如何使用 Azure 門戶和 OBS 創建媒體服務即時流。

## <a name="march-2020"></a>2020 年 3 月

新增了[頻線播快速啟動的即時流](live-events-wirecast-quickstart.md)。 它演示如何使用 Azure 門戶和遠端有線廣播創建 Azure 媒體服務即時流。
