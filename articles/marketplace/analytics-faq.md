# <a name="commercial-marketplace-analytics-common-questions"></a>商業 marketplace 分析的常見問題

本文說明合作夥伴中心中分析訊息的常見問題。

## <a name="common-questions"></a>常見問題

### <a name="why-am-i-unable-to-view-my-analytics-data-in-partner-center"></a>為什麼我無法在合作夥伴中心中查看分析資料？

當我存取分析頁面時，我看到下列訊息。

[![您的供應專案尚未提供任何資料。](./media/analytics-faq-no-data.png)](./media/analytics-faq-no-data.png#lightbox)

您可能會看到此訊息的原因：

- 商業 marketplace 中已發佈的供應專案目前不會有任何收購。 這可能表示您的供應專案已在商業 marketplace 中，且客戶正在查看您的產品清單頁面，但客戶尚未採取行動來購買及部署。
- 您的供應專案可能正在發佈，但尚未上線。 客戶只能取得上架的供應項目。 若要檢查供應專案的狀態，請參閱 [ [分析] 儀表板](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)中的 [ **摘要** ] 頁面。 如需詳細資訊，請參閱 [商業 marketplace 分析中的摘要儀表板](summary-dashboard.md)。
- 您的供應專案可能會列為「 **連絡人** 」，也就是僅限清單的供應專案，而且不能由商業 marketplace 中的客戶購買。 雖然這些供應項目會產生潛在客戶並分享給您，但無法為這些供應項目建立訂單，因為無法購買。 若要檢查您的供應專案清單類型，請移至供應專案的 [設定] 頁面。

### <a name="i-know-i-have-analytics-data-so-why-does-the-no-analytics-available-message-appear"></a>我知道有分析資料，那麼為什麼不會出現任何分析可用的訊息？

[![您的供應專案尚未提供任何資料。](./media/analytics-faq-no-data.png)](./media/analytics-faq-no-data.png#lightbox)

您可能會看到此訊息的原因：

- 如果您收到這則訊息，表示您有分析資料，但您所選取的計算期間沒有資料。 請選取不同的計算期間或月份範圍，以在2010之後查看任何資料。
- 如果您已從篩選選取範圍中的各種維度選取一或多個類別，您可能沒有選取專案的分析資料。 請嘗試重設篩選，或從篩選進行不同的選擇。

## <a name="next-steps"></a>後續步驟

- 如需合作夥伴中心中可用分析報告的總覽，請參閱 [合作夥伴中心中的商用 Marketplace 存取分析報表](./partner-center-portal/analytics.md)。
- 如需以圖形和可下載格式顯示訂單的詳細資訊，請參閱  [商用 marketplace 分析中的訂單儀表板](orders-dashboard.md)。
- 針對虛擬機器 (VM) 供應項目使用量和計量帳單計量，請參閱[商業市集分析中的使用量儀表板](usage-dashboard.md)。
- 如需客戶的詳細資訊，包括成長趨勢，請參閱 [商業市場分析中的客戶儀表板](customer-dashboard.md)。
- 如需過去 30 天內的下載要求清單，請參閱[商業市集分析中的下載儀表板](./partner-center-portal/downloads-dashboard.md)。
- 若要查看 Azure Marketplace 和 Microsoft AppSource 上提供的客戶意見反應的匯總觀點，請參閱 [合作夥伴中心 & 審核分析儀表板中的評](./partner-center-portal/ratings-reviews.md)等。
