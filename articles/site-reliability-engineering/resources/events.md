# <a name="participate-in-the-global-site-reliability-engineering-community"></a>參與全球網站可靠性工程小組

與您的同儕彼此討論就和研讀 SRE 一樣重要，而且往往更為重要。 談論您關於 SRE 的挑戰、成功與挫折，對於理解這個主題有著關鍵影響。

以下有一些主打 SRE 內容的聚會與會議。 或許最直接的關聯是由 USENIX 發表的全球分散式 [SREcon 會議](https://www.usenix.org/conferences) 。

有越來越多的 SRE 內容正在進行像是 [LISA](https://www.usenix.org/conferences/byname/5)、 [Day DevOps](https://www.alldaydevops.com)、 [DevOps 企業峰峰](https://events.itrevolution.com) 和當地 DevOps 會議（例如 [DevOps 天](https://www.devopsdays.org)）的會議。 找出這項內容，以及您可以在何處找到這些內容的其他主題。
