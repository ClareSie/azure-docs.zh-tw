下表描述了 Azure 資料資源管理器群集的最大限制。

| 資源 | 限制 |
| --- | --- |
| 每個訂閱每個區域的群集 | 20 |
| 每個群集的實例 | 1000 | 
| 群集中的資料庫數 | 10,000 |
| 群集中附加的資料庫配置數 | 70 |

下表描述了對 Azure 資料資源管理器群集執行的管理操作的限制。

| 影響範圍 | 作業 | 限制 |
| --- | --- | --- |
| 叢集 | 讀取（例如，獲取群集） | 每5分鐘500人 |
| 叢集 | 寫入（例如，創建資料庫） | 每小時1000人 |
