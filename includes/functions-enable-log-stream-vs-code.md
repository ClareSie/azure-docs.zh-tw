要打開 Azure 中函數應用的流式處理日誌，請執行以下功能：

1. 選擇 F1 打開命令調色板，然後搜索並運行命令 Azure**函數：啟動流式處理日誌**。

1. 在 Azure 中選擇函數應用，然後選擇 **"是**"以啟用函數應用的應用程式日誌記錄。

1. 在 Azure 中觸發函數。 請注意，日誌資料將顯示在視覺化工作室代碼中的"輸出"視窗中。

1. 完成後，請記住運行**命令 Azure 函數：停止流式處理日誌**以禁用函數應用的日誌記錄。
