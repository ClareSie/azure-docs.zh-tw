您可以使用適用於 Swift 和 Objective-C 的語音 SDK，從文字合成語音。

## <a name="prerequisites"></a>必要條件

下列範例假設您具有 Azure 帳戶和語音服務訂用帳戶。 如果您沒有該帳戶和訂用帳戶，請[免費試用語音服務](../../../overview.md#try-the-speech-service-for-free)。

## <a name="install-speech-sdk-and-samples"></a>安裝語音 SDK 和範例

[認知服務語音 SDK](https://github.com/Azure-Samples/cognitive-services-speech-sdk) 包含以 Swift 和 Objective-C 撰寫的範例，適用於 iOS 和 Mac。 按一下連結可查看每個範例的安裝指示：

* [在 macOS 上以 Objective-C 合成語音](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/macos/text-to-speech)
* [在 macOS 上以 Swift 合成語音](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/macos/text-to-speech)
* [在 iOS 上以 Objective-C 合成語音](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/objectivec/ios/text-to-speech)
* [在 iOS 上以 Swift 合成語音](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/swift/ios/text-to-speech)
* [iOS 上 Objective-C 的其他範例](https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/samples/objective-c/ios)

我們也會提供[適用於 Objective-C 參考的線上語音 SDK](https://docs.microsoft.com/objectivec/cognitive-services/speech/)。
