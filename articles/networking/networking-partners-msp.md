# <a name="about-azure-networking-msp-partners"></a>關於 Azure 網路 MSP 合作夥伴

[Azure 網路託管服務提供者 (MSP) 合作夥伴計畫](https://azure.microsoft.com/blog/enhancing-the-customer-experience-with-the-azure-networking-msp-partner-program/)支援以網路服務為中心的 MSP、電信公司和系統集成者 (SI) 提供以 Azure 網路產品和服務組合為中心的雲端和混合網路服務。

Azure 網路 MSP 是一組專門的託管服務提供者,可滿足雲和混合網路各個方面的企業雲網路需求和挑戰。 託管網路服務包括以下一個或多個服務:網路體系結構、規劃、部署、操作、維護和優化。

![Msp][0]

客戶可以在 Azure 應用商店中購買 MSP 託管的服務產品,也可以直接從 MSP 購買服務。 MSP 可以使用[Azure 燈塔](https://azure.microsoft.com/services/azure-lighthouse/)在客戶的訂閱和資源組中部署和管理 Azure 網路資源。
