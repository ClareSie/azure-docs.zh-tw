虛擬機器 (VM) 會根據 VM 部署所在的子網，自動指派您指定範圍內的私人 IP 位址。 VM 會保留位址，直到刪除 VM 為止。 Azure 會以動態方式將下一個可用的私人 IP 位址指派給您在其中建立 VM 的子網。 如果您想要將子網中的特定 IP 位址指派給 VM，請指派靜態 IP 位址。
