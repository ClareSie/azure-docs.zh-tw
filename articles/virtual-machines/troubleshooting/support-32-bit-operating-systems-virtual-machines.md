# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>在 Azure 虛擬機器中的 32 位元作業系統支援

Microsoft Azure 現在允許使用者將其 32 位 Windows 作業系統引入 Azure。 由於其中一些作業系統已經達成其壽命終止支援協定，Microsoft 可能不會為他們提供額外的支援。 對於在 Microsoft Azure 虛擬機器 （VM） 上運行的基於 Linux 或基於伯克利軟體分發 （BSD） 的作業系統，也不提供支援。

* [支援 Azure 中的 Linux 和開源技術](https://support.microsoft.com/help/2941892/support-for-linux-and-open-source-technology-in-azure)
