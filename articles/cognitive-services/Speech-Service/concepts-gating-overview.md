# <a name="custom-neural-voice-gating-overview"></a>自訂神經語音門控概述

詳細瞭解開始使用自訂神經語音的過程。

## <a name="commitment-to-responsible-innovation"></a>致力於負責任的創新

作為 Microsoft 設計負責任 AI 承諾的一部分，我們收集了一組材料，以指導客戶使用自訂神經語音。 此處找到的指南和見解基於 Microsoft 在[AI 中負責任創新的原則。](https://www.microsoft.com/AI/our-approach-to-ai)

### <a name="guidance-for-deploying-custom-neural-voice"></a>部署自訂神經語音的指南

- [負責任部署指南](concepts-guidelines-responsible-deployment-synthetic.md)：我們基於我們研究的頂級建議
- [語音人才的披露](https://aka.ms/disclosure-voice-talent)：您需要瞭解什麼，並告知語音人才如何負責任地使用它
- [披露設計](concepts-disclosure-guidelines.md)：如何設計體驗，以便使用者知道何時使用合成語音並信任您的服務

### <a name="why-custom-neural-voice-is-a-gated-technology"></a>為什麼自訂神經語音是門控技術

我們的目的是保護個人和社會的權利，促進透明的人機交互，並抵制有害深層虛假和誤導性內容的擴散。 因此，我們已經對自訂神經語音的使用進行了封閉。 客戶只有在對其應用程式進行審查並承諾根據我們的道德原則使用該技術後，才能訪問該技術。

### <a name="our-gating-process"></a>我們的澆注過程

要訪問自訂神經語音，您需要首先填寫我們的線上入學表單。 [在此處](https://aka.ms/custom-neural-intake-form)開始您的應用程式。

根據我們的資格標準、審核流程以及在此門控預覽期間支援有限數量的客戶的可用性，訪問自訂神經語音服務需由 Microsoft 自行決定。

作為申請過程的一部分，您需要承諾在創建語音字體之前從語音人才處獲得明確的書面許可，其中包括共用[語音人才的披露](https://aka.ms/disclosure-voice-talent)。 您還必須同意，在部署語音字體時，您的實現將向使用者[披露服務的綜合性質](concepts-disclosure-guidelines.md)，在服務條款中提供 Microsoft 綜合語音服務的歸因，並支援一個回饋管道，允許服務使用者報告問題並與 Microsoft 共用詳細資訊。 在此處瞭解有關我們的使用條款[的更多詳細資訊](https://aka.ms/custom-neural-code-of-conduct)。
