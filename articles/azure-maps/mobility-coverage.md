# <a name="azure-maps-mobility-coverage"></a>Azure 地圖服務行動性涵蓋範圍

Azure 地圖服務[行動服務](https://aka.ms/AzureMapsMobilityService)可改善具有公開傳輸功能之應用程式的開發時間，例如傳輸路由和搜尋附近的公用傳輸。 使用者可以取得有關傳輸停止、行和排程的詳細資訊。 行動服務也可讓使用者抓取停止和線條的幾何、停止、行和服務區域的警示，以及即時的公開傳輸抵達和服務警示。 此外，行動服務會提供具有限時行程規劃選項的路由功能。 多模行程規劃包括流覽、腳踏車和公用傳輸選項，全都放在一趟旅程。 使用者也可以存取詳細的多重步驟路線。 

Azure 地圖服務不會針對所有城市和國家/地區提供相同層級的資訊和精確度。 呼叫公開傳輸資料的能力取決於 metro 區域。 此外，地圖資料可能不會包含所有公開運輸選項和提供 metro 區域的機構。

下表提供 Azure 地圖服務行動服務的涵蓋範圍資訊。

| 符號 | 意義 |
|--------|---------|
| *      |該國家/地區幾乎完全涵蓋。|
