> 如果您的系統管理員已為您的組織開啟新的 **我的應用程式** 預覽體驗，您現在會在 [ **我的應用程式** ] 頁面頂端選取 [帳戶] 圖示以查看您的帳戶資訊時，看到一個連結。 如果您選取 [ **試用新的我的應用程式體驗！**]，您會立即前往新的 URL (https://myapplications.microsoft.com) 和更新的體驗。 之後，當您回到 **我的應用程式** 頁面時，將會自動移至新的體驗。<br><br>若要停止使用新的體驗，您可以 **選取 [** 帳戶] 圖示，以在 **我的應用程式** 頁面頂端查看您的帳戶資訊。 您會立即回到舊的 URL (myapps.microsoft.com) 和體驗。 之後，每當您返回 [我的應用程式] 頁面時，就會自動前往舊的體驗。
