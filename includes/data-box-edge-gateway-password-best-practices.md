請記住以下最佳實踐：

- 我們建議您將所有密碼存儲在安全的地方，這樣，如果忘記密碼，就不必重置密碼。 管理服務無法檢索現有密碼。 它只能通過 Azure 門戶重置它們。 如果重置密碼，請確保在重置密碼之前通知所有使用者。
- 您可以通過 HTTP 遠端存取設備的 Windows PowerShell 介面。 作為安全最佳實踐，應僅在受信任的網路上使用 HTTP。
- 確保裝置密碼是強的，並且保護良好。 遵循[密碼最佳實踐](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management)。
