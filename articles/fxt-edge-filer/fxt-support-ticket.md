# <a name="open-a-support-ticket-for-the-azure-fxt-edge-filer"></a>開啟 Azure FXT Edge Filer 的支援票證

本文說明如何取得 Azure FXT Edge Filer 系統的協助。

使用 Azure 入口網站，透過 Microsoft 客戶服務和支援服務來開啟支援票證。

1. 開啟 [Azure 入口網站](https://portal.azure.com/) 並登入。
1. 在左面板中，向下移至底部，然後按一下 [說明 **+ 支援**]。
1. 按一下 [ **新增支援要求**]。

   ![顯示 [說明 + 支援] 頁面的螢幕擷取畫面，其中已選取 [新增支援要求]。](media/fxt-support-blank.png)

1. 填寫申請表單。  
    1. 在 [ **基本** ] 索引標籤中，選擇 [ **問題類型**] ``Technical`` 。
    1. 選取您的訂用帳戶。
    1. 在 [ **服務**] 下，選取 [ **Azure FXT Edge Filer**]。 您可能需要按一下 [ **所有服務** ] 篩選控制項來尋找它。
    1. 輸入 **摘要** ，然後選擇 **問題類型**。
    ![空白支援要求表單的螢幕擷取畫面](media/fxt-support-populated.png)
    1. 選取 [下一步] 以繼續操作。
1. 在 [ **詳細資料** ] 索引標籤中，新增有關問題的詳細資訊，以及如何聯繫您。 您可以上傳與問題相關的檔案。
1. 在 [審核] 和 [ **建立**] 中，檢查資訊，然後按一下 [ **建立** ] 按鈕來提交您的要求。

提交要求之後，您會收到一封包含票證號碼的確認電子郵件。 支援人員成員將會與您聯絡要求。
