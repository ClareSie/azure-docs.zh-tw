# <a name="change-your-work-or-school-account-password-from-the-change-password-page"></a>從 [變更密碼] 頁面變更您的公司或學校帳戶密碼

在**我的帳戶**入口網站中，[變更密碼] 頁面可協助您更新現有的公司或學校帳戶密碼 (假設您記得該密碼，而且您的帳戶未遭到鎖定)。 如果您忘記密碼或帳戶已遭鎖定，或者您沒有從組織取得密碼，您可以使用安全性資訊和行動裝置來重設密碼。

>[!Important]
>本文適用於嘗試更新現有公司或學校帳戶已知密碼的使用者。 如果您正嘗試進入個人帳戶 (例如 Xbox、Hotmail 或 Outlook.com)，請試試[當您無法登入 Microsoft 帳戶時](https://support.microsoft.com/help/12429/microsoft-account-sign-in-cant)一文中所提供的建議。 如果您是系統管理員，而且想知道更多如何為員工或其他使用者測試自助式密碼重設的資訊，請參閱[自助式密碼重設](../authentication/tutorial-enable-sspr.md)。

## <a name="update-a-password-from-the-change-password-page"></a>從 [變更密碼] 頁面更新密碼

1. 使用現有的密碼登入您的公司或學校帳戶，然後移至 [我的帳戶] (https://myaccount.microsoft.com/) 頁面。

2. 從左側瀏覽窗格中選取 [密碼]，或從 [密碼] 區塊中選取 [變更密碼] 連結。

    ![[我的帳戶] 頁面，顯示已醒目提示的變更密碼連結](media/my-account-portal/my-account-portal-change-password.png)

3. 輸入舊密碼，然後建立並確認新密碼。

    ![顯示密碼欄位的 [變更密碼] 頁面](media/my-account-portal/my-account-portal-change-password-page.png)

4. 選取 [提交]。

    您的密碼將會變更，並要求您再次登入公司或學校帳戶。
