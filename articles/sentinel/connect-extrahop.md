# <a name="connect-extrahop-revealx"></a>連線 ExtraHop Reveal(x)

ExtraHop 顯示（x） 資料連線器可讓您輕鬆地將"顯示"系統與 Azure Sentinel 連接，以查看儀表板、創建自訂警報並改進調查。 通過此集成，您可以深入瞭解組織的網路並改進安全操作功能。


## <a name="prerequisites"></a>Prerequisites

- 讀取和寫入 Azure Sentinel 工作區的許可權。

- 具有固件版本 7.8 或更高版本的 ExtraHop 發現或命令設備，具有具有無限（管理員）許可權的使用者帳戶。

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>將 Azure 哨兵連接到外點顯示（x）

1. 在 Azure 門戶中，導航到**Azure 哨兵** > **資料連線器**，然後選擇 **"額外顯示（x）"** 連接器。

2. 選擇 **"打開連接器"頁**。

3. 按照 **"額外霍普顯示（x）"** 頁面上的說明進行操作。
