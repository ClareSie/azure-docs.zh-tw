# <a name="view-the-list-of-cloudsimple-portal-users"></a>查看雲簡單門戶使用者清單

使用者首次訪問 CloudSimple 門戶時，將添加到使用者清單中。 要查看通過 Azure 訪問雲簡單門戶的使用者清單，[請訪問雲簡單門戶](access-cloudsimple-portal.md)，選擇側功能表上的**帳戶**，然後在雲簡單門戶中選擇 **"使用者**"。

* 要顯示使用者詳細資訊（包括 Azure 訂閱、租戶和使用者 ID），請按一下 **"使用者"** 頁上的條目。

* 要查看使用者的審核活動日誌，請選擇 **"稽核記錄"** 選項卡。
* 要鎖定或解鎖使用者帳戶，請在顯示使用者詳細資訊時按一下 **"鎖定**切換"。 當帳戶解鎖時，使用者可以訪問 CloudSimple 門戶。 當帳戶被鎖定時，對門戶的訪問將被阻止。
