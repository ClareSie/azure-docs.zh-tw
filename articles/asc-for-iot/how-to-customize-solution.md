# <a name="customize-your-azure-security-center-for-iot-solution"></a>自訂 IoT 解決方案的 Azure 資訊安全中心

在本指南中，您將瞭解如何在適用于 IoT 的 Azure 資訊安全中心中自訂不同的設定。

> [!div class="checklist"]
> * 設定解決方案建議
> * 變更設定

## <a name="change-settings"></a>變更設定

管理 IoT 的 Azure 資訊安全中心設定：

在您的 IoT 中樞上，前往左上角的 [安全性總覽] 分頁，請參閱「設定」以設定 IoT 設定的 Azure 資訊安全中心，請執行下列動作：

1. 在 Azure 入口網站中開啟您的 **IoT 中樞**。
1. 從 [**安全性**] 底下的左側功能表中，選取並開啟 **[總覽**]。
1. 在 [設定] 下，選取您想要變更的解決方案設定。
1. 請記得一律按一下任何設定畫面頂端的 [**儲存**]，以儲存您的設定變更。

## <a name="configure-solution-recommendations"></a>設定解決方案建議

若要設定您的 Azure 資訊安全中心 IoT 解決方案建議，請執行下列動作：

1. 在 Azure 入口網站中開啟您的 **IoT 中樞**。
1. 在左側功能表中，選取並開啟 [**安全性**] 底下的 **[總覽**]。
1. 在 [設定] 下，從左側功能表中選取 [**建議**設定]。
1. 停用/啟用與您的組織和工作流程相關的解決方案建議。
1. 選取畫面頂端的 [**儲存**]，以儲存您的選取專案。
