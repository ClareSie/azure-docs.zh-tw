# <a name="whats-new-in-content-moderator"></a>Content Moderator 的新功能

了解該服務的新功能。 這些項目可能是版本資訊、影片、部落格文章與其他類型的資訊。 將此頁面加入書簽，以掌握服務的最新狀態。

## <a name="august-2020"></a>2020 年 8 月

### <a name="new-video-moderation-ui"></a>新的影片仲裁 UI

* 檢閱工具的影片仲裁頁面已更新而具有許多新的工具和功能，有助於標記和處理影片和部分影片。 若要深入了解，請參閱[影片仲裁指南](./video-moderation-human-review.md)。

### <a name="new-dashboard"></a>新增儀表板

* 檢閱工具儀表板已更新，會顯示更多與開啟、暫止和已完成的檢閱有關的資訊。 如需新功能的導覽，請參閱[檢閱工具指南](./Review-Tool-User-Guide/human-in-the-loop.md)。
