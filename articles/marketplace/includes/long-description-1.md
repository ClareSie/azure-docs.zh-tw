提供較詳細的供應項目描述，最多 3,000 個字元。 客戶會在 Marketplace 清單總覽中看到這項功能。

在您的描述中包含下列一或多個專案：

- 供應項目的價值與關鍵權益
- 類別及/或產業關聯
- 應用程式內的購買機會
- 任何必要揭露
