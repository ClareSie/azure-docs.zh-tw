# <a name="connect-one-identity-safeguard"></a>連接一個標識保護
"一個標識保護"資料連線器使用特權會話特定儀表板的防護功能增強標準公共事件格式 （CEF） 連接器。 使用此連接器可以輕鬆地開始使用設備生成的事件進行視覺化、警報、調查等。


## <a name="prerequisites"></a>Prerequisites

- 讀取和寫入 Azure Sentinel 工作區的許可權。

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>將 Azure 哨兵連接到一個標識保護

1. 在 Azure 門戶中，導航到**Azure 哨兵** > **資料連線器**，然後選擇 **"一個標識保護"** 連接器。

2. 選擇 **"打開連接器"頁**。

3. 按照"**一個身份保護"** 頁面上的說明進行操作。
