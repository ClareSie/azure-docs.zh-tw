# <a name="threat-modeling-tool-update-release-71607021---722019"></a>威脅建模工具更新版本 7.1.60702.1 - 2019/7/2

Microsoft 威脅建模工具 （TMT） 的版本 7.1.60702.1 于 2019 年 7 月 2 日發佈，其中包含以下更改：

- 協助工具增強功能
- 錯誤修正
- 包含提供模具集的開源社區

## <a name="feature-changes"></a>功能變更

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>提供由開源社區提供的新醫療設備模具集

開源社區貢獻了用於建模醫療設備的模具集。 更新後，新的模具集將顯示在範本選擇下拉式功能表中。 有關向範本貢獻模具或內容的資訊，請查看專案的[GitHub 頁面上](https://github.com/Microsoft/threat-modeling-templates)的資訊。

![模型驗證選項](./media/threat-modeling-tool-releases-71607021/tmt-template-selection.png)

## <a name="system-requirements"></a>系統需求

- 支援的作業系統
  - [微軟視窗 10 周年更新](https://blogs.windows.com/windowsexperience/2016/08/02/how-to-get-the-windows-10-anniversary-update/#HTkoK5Zdv0g2F2Zq.97)或更高版本
- 必要的 .NET 版本
  - [.淨 4.7.1](https://go.microsoft.com/fwlink/?LinkId=863262)或更高版本
- 其他需求
  - 必須有網際網路連線可接收工具和範本的更新。
