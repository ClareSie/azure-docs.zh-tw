|地緣政治區域 | Azure 區域|
|---|---|
| 澳洲政府 | 澳大利亞中部、澳大利亞中部 2 |
| 歐洲 | 法國中部、法國南部、德國北部、德國中西部、北歐、挪威東部、瑞士北部、瑞士西部、西歐、英國西部、英國南部 |
| 北美洲 | 美國東部、美國西部、美國東部 2、美國西部 2、美國中部、美國中南部、美國中北部、美國中西部、加拿大中部、加拿大東部 |
| Asia | 東亞、東南亞 |
| 印度 | 印度西部、印度中部、印度南部 |
| 日本  | 日本西部、日本東部 |
| 大洋洲 | 澳大利亞東南部、澳大利亞東部 |
| 南非 | 南非北部，南非西部 |
| 南美洲 |巴西南部 |
| 南韓 | 南韓中部、南韓南部 |
| 阿拉伯聯合大公國 | 阿聯酋北部，阿聯酋中部 |
