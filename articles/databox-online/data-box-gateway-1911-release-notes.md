# <a name="azure-data-box-edge-and-azure-data-box-gateway-1911-release-notes"></a>Azure 資料框邊緣和 Azure 資料框閘道 1911 發行說明

以下發行說明標識 Azure 數據框邊緣和 Azure 數據框閘道的 1911 版本的關鍵未決問題和已解決的問題。

版本資訊會持續更新，並在發現需要提出因應措施的重大問題時有所增補。 在部署數據框邊緣/數據盒閘道之前,請仔細查看發行說明中包含的資訊。

此版本對應於軟體版本:

- **數據盒閘道 1911 (1.6.1049.786)**
- **資料框邊緣 1911 (1.6.1049.786)**

> [!NOTE]
> 更新 1911 只能應用於運行軟體的通用 (GA) 版本或更高版本的所有設備。

## <a name="whats-new"></a>新增功能

1911 年版本中沒有新功能,只有錯誤修復。

## <a name="known-issues-in-ga-release"></a>GA 版本中的已知問題

此版本未注意到任何新問題。 所有發佈的注意到的問題都從以前的版本中延續過來。 要檢視已知問題的清單,請造[訪 GA 版本中的已知問題](data-box-gateway-release-notes.md#known-issues-in-ga-release)。
