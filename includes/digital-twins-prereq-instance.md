若要使用本文中的 Azure Digital Twins，您必須先 **設定 Azure Digital Twins 執行個體** 以及使用的必要權限。 如果您已經透過先前的工作設定 Azure Digital Twins 執行個體，您可以使用該執行個體。

否則，請依照[設定執行個體和驗證](../articles/digital-twins/how-to-set-up-instance-portal.md)中的指示進行。 這些指示也包含可驗證您已成功完成每個步驟，並已準備好繼續使用新執行個體的步驟。

設定 Azure Digital Twins 執行個體之後，請記下下列值以供稍後連線到執行個體時使用：
* 執行個體的 **_主機名稱_**。 您可以在 Azure 入口網站 ([指示](../articles/digital-twins/how-to-set-up-instance-portal.md#verify-success-and-collect-important-values)) 中找到此值。
* 您用來建立執行個體的 **Azure 訂用帳戶** (其名稱或識別碼都行)。 您可以在 [Azure 入口網站](https://portal.azure.com)中檢視執行個體的相同 [概觀] 頁面，以查看 Azure Digital Twins 執行個體所在的訂用帳戶。
