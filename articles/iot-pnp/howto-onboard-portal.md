# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>在 Azure IoT 認證入口網站上線

您可以使用 Azure[認證的 IoT 門戶](https://aka.ms/ACFI)：

- 完成 IoT 設備的認證流程。
- 查找 IoT 隨插即用預覽裝置功能模型。 您可以使用這些模型快速構建物聯網就緒設備並將其與解決方案集成。
- 將裝置功能模型發佈到公共存儲庫。

## <a name="sign-in"></a>登入

使用 Microsoft_工作帳戶_登錄到[適用于 IoT 門戶的 Azure 認證](https://aka.ms/ACFI)門戶。 登錄時，門戶會通過 Microsoft 合作夥伴中心檢查您的成員資格：

- 如果您的組織已經是[Microsoft 合作夥伴中心](https://docs.microsoft.com/partner-center/)的成員，則授予您訪問門戶中所有功能的許可權。
- 如果您的組織不是[Microsoft 合作夥伴中心](https://docs.microsoft.com/partner-center/)的成員，則可以運行認證測試並流覽裝置功能模型存儲庫。 您無法在 IoT 認證設備目錄中發佈設備。
- 如果您看到一條通知訊息，指出您的 Microsoft 合作夥伴網路識別碼 丟失，則應[創建合作夥伴中心帳戶](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account)。 您的帳戶獲得批准後，您可以返回到門戶併發佈設備。 有關詳細資訊，請參閱[合作夥伴中心常見問題解答](https://support.microsoft.com/help/4340639/partner-center-account-faqs)。

在將設備發佈到 IoT 認證設備目錄之前，請閱讀並接受程式協定。 您可以在 Azure 認證 IoT 門戶中的設置頁上找到協定。
