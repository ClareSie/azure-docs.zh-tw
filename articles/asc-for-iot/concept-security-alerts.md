# <a name="azure-security-center-for-iot-security-alerts"></a>適用於 IoT 的 Azure 資訊安全中心的安全性警示

IoT 的 Azure 資訊安全中心會使用先進的分析與威脅情報，持續分析您的 IoT 解決方案，以向您發出惡意活動的警示。
此外，您也可以根據您對預期裝置行為的知識，建立自訂警示。
警示會作為潛在危害的指標，且應加以調查並補救。

在本文中，您會找到可在您的 IoT 中樞和/或 IoT 裝置上觸發的內建警示清單。
除了內建警示以外，IoT 的 Azure 資訊安全中心可讓您根據預期的 IoT 中樞和/或裝置行為來定義自訂警示。
如需詳細資訊，請參閱[可自訂的警示](concept-customizable-security-alerts.md)。
