# <a name="power-bi-app-offer"></a>Power BI 應用程式供應專案

Cloud Partner 入口網站不再支援建立和管理 Power BI 的應用程式供應專案。 此功能已移至 Microsoft[合作夥伴中心](https://partner.microsoft.com/pcv/)。 如需詳細資訊，請參閱[建立適用于 Microsoft AppSource 的 Power BI 應用程式](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer)。
