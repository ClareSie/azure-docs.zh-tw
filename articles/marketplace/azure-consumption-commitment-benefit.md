# <a name="azure-consumption-commitment-benefit"></a>Azure 使用量承諾用量權益

如果您的組織具有下列其中一項合約，則一組精選的 Microsoft 商業 marketplace 供應專案將有助於貴組織的 Microsoft Azure 耗用量承諾：

- Microsoft Azure 耗用量承諾（MACC）
- 要使用的 Azure 認可（.Ctc）

我們會驗證參與此計畫的所有供應專案，以確保您能獲得高品質的解決方案。

若要利用這項權益，只要使用與您的 Azure 合約相關的訂用帳戶，就能在 Azure Marketplace 上購買合格供應專案。 Azure 預付款和承諾用量不符合此權益的資格。

> [!IMPORTANT]
> 排除專案可能適用于此 marketplace 權益之前簽署的 .Ctc 合約。 如果您對資格有任何疑問，請洽詢您的 Microsoft 帳戶主管。
