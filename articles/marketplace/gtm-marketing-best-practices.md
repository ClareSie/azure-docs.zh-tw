# <a name="marketing-best-practices"></a>行銷最佳做法

當您建立並列出供應項目給商業市集、提供客戶試用版，並與 Microsoft 客戶和合作夥伴社群建立關係時，請記住行銷最佳做法。 如果您將供應項目清單和進入市場行銷活動最佳化，您可以加快客戶獲得速度。 下載 [Azure Marketplace 與 AppSource 最佳做法指南](https://aka.ms/marketplacebestpractices) \(英文\)，以了解如何充分利用您的線上行銷工作。

若要深入了解 Microsoft 合作夥伴網路如何協助您拓展業務，請參閱[與 Microsoft 並肩進入市場](https://partner.microsoft.com/reach-customers/gtm)。 請登入[合作夥伴中心](https://partner.microsoft.com/dashboard/commercial-marketplace/)，以建立及設定您的供應項目。
