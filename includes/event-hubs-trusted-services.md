## <a name="trusted-microsoft-services"></a>受信任的 Microsoft 服務
當您啟用 [**允許受信任的 Microsoft 服務略過此防火牆**] 設定時，系統會將您事件中樞資源的存取權授與下列服務。

| 信任的服務 | 支援的使用案例 | 
| --------------- | ------------------------- | 
| Azure Event Grid | 可讓 Azure 事件方格將事件傳送至事件中樞命名空間中的事件中樞。 |
| Azure 監視器 (診斷設定)  | 允許 Azure 監視器將診斷資訊傳送至您事件中樞命名空間中的事件中樞。 |
