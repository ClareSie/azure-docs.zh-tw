# <a name="cognitive-services-helps-to-solve-business-problems-by-using-ai"></a>認知服務通過使用 AI 説明解決業務問題
[Azure 認知服務](https://azure.microsoft.com/services/cognitive-services/)提供強大的服務，為您的應用注入智慧演算法，以查看、收聽、說話、理解和解釋使用者使用人工智慧 （AI） 轉變業務所需的資訊。 您可以使移動應用程式智慧化、吸引人且與使用者上下文相關。 開發人員可以使用 API、SDK 和服務將 AI 添加到其產品中，並説明構建智慧應用程式，而無需直接掌握 AI 或資料科學技能。

## <a name="cognitive-services-capabilities"></a>認知服務功能
- **視覺**
    - 圖像標記、縮略圖
    - OCR（手寫辨識）
    - 臉部偵測
    - 情緒偵測
    - 影片見解
    - 圖像和視頻審核
    - 自訂圖像識別

- **語音**
    - 語音到文本（語音轉錄）
    - 自訂語音轉錄（複雜單詞、嘈雜環境）
    - 將文字轉換成語音
    - 揚聲器 ID 和身份驗證
    - 即時語音翻譯

- **Language**
    - 上下文語言理解，定制意圖分析
    - 情緒分析，關鍵短語檢測
    - 文本翻譯多達 60 種語言
    - 文字仲裁
    - 拼字檢查

- **決策** 
    - 內容仲裁
    - 個人化內容交付
    - 異常檢測器

- **搜尋**
    - 自動搜索建議 
    - 實體資訊增強
    - 全面新聞、圖片、視頻、網路和本地業務實體搜索
    - 定制和定制的搜索體驗

- 支援認知服務容器
