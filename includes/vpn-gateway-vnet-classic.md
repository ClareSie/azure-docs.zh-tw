1. 從瀏覽器流覽至 [Azure 入口網站](https://portal.azure.com) ，如有必要，請使用您的 Azure 帳戶登入。
1. 選取 [ **+ 建立資源**]。 在 [搜尋 Marketplace]**** 欄位中，輸入「虛擬網路」。 從傳回的清單中找出 **虛擬網路** ，然後加以選取以開啟 [ **虛擬網路** ] 頁面。
1. 在 [虛擬網路] 頁面的 [建立] 按鈕底下，您會看到「使用 Resource Manager 部署 (變更為傳統) 」。 Resource Manager 是建立 VNet 的預設值。 您不想要建立 Resource Manager VNet。 選取 ** (變更為傳統) ** 以建立傳統 VNet。 然後，選取 [ **總覽** ] 索引標籤，然後選取 [ **建立**]。
1. 在 [ **建立虛擬網路 (傳統) ** ] 頁面的 [ **基本** ] 索引標籤上，使用範例值設定 VNet 設定。
1. 選取 [ **審核 + 建立** ] 以驗證您的 VNet。
1. 驗證執行。 驗證 VNet 之後，請選取 [ **建立**]。
