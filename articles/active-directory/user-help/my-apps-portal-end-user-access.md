# <a name="sign-in-and-start-apps-from-the-my-apps-portal"></a>從"我的應用"門戶登錄並開始應用

您可以將工作或學校帳戶與基於 Web**的"我的應用"** 門戶一起使用，以便：

- 查看和啟動組織的許多基於雲的應用
- 更新您的個人資料和帳戶資訊
- 查看**您的群組**資訊
- 執行應用和組**的訪問審核**

如果您沒有 [我的應用程式]**** 入口網站的存取權，請連絡技術服務人員以取得權限。

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

> [!Important]
> 本內容適用於我的應用程式使用者。 如果您是系統管理員，可以在[應用程式管理文件](https://docs.microsoft.com/azure/active-directory/manage-apps)中找到更多關於如何設定和管理雲端式應用程式的資訊。

## <a name="supported-browsers"></a>支援的瀏覽器

您可以通過以下任何 Web 瀏覽器訪問 **"我的應用"** 門戶：

- Google Chrome

- Mozilla 火狐，版本 26.0 或更高版本

- Microsoft Edge

- 互聯網瀏覽器，版本 11（有限支援）

## <a name="download-and-install-the-my-apps-secure-sign-in-extension"></a>下載並安裝"我的應用安全登錄擴展"

如果系統提示，請下載並安裝"我的應用安全登錄擴展"。 此擴展可説明您啟動組織的任何需要您使用單個登錄過程的雲應用。 如果您的組織已為您設置單一登入，則會自動安裝擴展，您可以跳過此部分。

此擴展可説明您：

- 直接從登錄頁面登錄到應用。

- 使用**快速搜尋**功能啟動任何應用。

- 請參閱"**最近使用"** 部分中使用的最後一個應用。

- 使用內部公司 URL 時使用[應用程式代理](https://docs.microsoft.com/azure/active-directory/active-directory-application-proxy-get-started)遠端。

### <a name="to-download-and-install-the-extension"></a>下載並安裝擴展

下載並安裝擴展，具體取決於您使用的瀏覽器：

- **谷歌瀏覽器。** 從 Chrome 網上商店轉到["我的應用安全登錄擴展"](https://chrome.google.com/webstore/detail/my-apps-secure-sign-in-ex/ggjhpefgjjfobnfoldnjipclpcfbgbhl)功能，然後選擇"添加到**Chrome**"。

- **莫齊拉·火狐**從**Firefox 載入頁**，轉到["我的應用程式安全登錄擴展"](https://addons.mozilla.org/firefox/addon/access-panel-extension/)功能，然後選擇"**添加到 Firefox**"。

- **微軟邊緣**從 Microsoft 應用商店轉到["我的應用安全登錄擴展"](https://www.microsoft.com/p/my-apps-secure-sign-in-extension/9pc9sckkzk84?rtc=1&activetab=pivot%3Aoverviewtab)功能，然後選擇 **"獲取"** 以獲取 Microsoft 邊緣舊版瀏覽器的擴展。  
如果你正在使用新的微軟邊緣瀏覽器（鉻的邊緣），那麼你可以從[微軟邊緣載入項商店](https://microsoftedge.microsoft.com/addons/category/EdgeExtensionsEditorsPick)獲得興奮。

**位址**欄右側會添加一個圖示，允許您登錄並自訂擴展。

## <a name="to-change-your-my-apps-portal-using-the-extension"></a>使用擴展更改"我的應用"門戶

您可以在"**最近使用"** 部分中選擇要查看的應用數，並決定是否允許組織的內部 URL 重定向。

1. 選擇**位址**欄右側的新 **"我的應用安全登錄擴展"**![圖示](media/my-apps-portal/my-apps-portal-extension-icon.png)擴展圖示，然後選擇"**登錄以開始**"。

2. 按右鍵 **"設置"** 圖示!["](media/my-apps-portal/my-apps-portal-extension-settings-icon.png)設置"圖示，然後選擇 **"設置**"。

3. 在 **"設置"** 框中，選擇要在門戶上看到的最新應用數，以及是否允許組織的內部 URL 重定向，以便您可以遠端使用它們。

    ![擴展的設置頁，顯示可用的自訂項](media/my-apps-portal/my-apps-portal-extension-settings-page.png)

## <a name="access-and-use-the-my-apps-portal-by-device"></a>按設備訪問和使用"我的應用"門戶

您可以通過 Intune 管理的瀏覽器或 iOS 或 Android 行動裝置訪問和使用電腦上的"我的應用"門戶。

!["我的應用"門戶中的應用頁面](media/my-apps-portal/my-apps-portal-apps-page.png)

### <a name="access-and-use-the-my-apps-portal-on-your-computer"></a>訪問和使用電腦上的"我的應用"門戶

如果您有權訪問和使用組織的基於雲的應用，則可以通過 **"我的應用"** 門戶訪問這些應用。

1. 登錄到電腦上的工作或學校帳戶。

2. 在受支援的 Web 瀏覽器中，打開https://myapps.microsoft.com並轉到 ，或者使用組織提供的連結（如果他們將您定向到自訂頁面，如`https://myapps.microsoft.com/contoso.com`）。

    將顯示 **"應用"** 頁面，顯示組織擁有的所有可供您使用的所有基於雲的應用。

3. 在 **"應用"** 頁面中，選擇要開始使用的應用。

    將打開應用的新頁面，您可以在其中登錄（如有必要）或開始使用該應用程式。

### <a name="access-and-use-the-my-apps-portal-on-an-intune-managed-browser"></a>在 Intune 管理的瀏覽器上訪問和使用"我的應用"門戶

在 iOS 和 Android 設備上從 Intune 託管瀏覽器查看和使用組織的應用。

1. 在您的行動裝置上，從[蘋果應用商店](https://itunes.apple.com/us/app/microsoft-intune-managed-browser/id943264951?mt=8)和[谷歌Play商店](https://play.google.com/store/apps/details?id=com.microsoft.intune.mam.managedbrowser)下載並安裝 Intune 託管瀏覽器應用程式。

2. 打開 Intune 託管瀏覽器應用，轉到https://myapps.microsoft.com，或使用組織提供的連結（如果他們將您定向到自訂頁面，如https://myapps.microsoft.com/contoso.com）。

    將顯示 **"應用"** 頁面，顯示組織擁有並可供您使用的所有基於雲的應用。

3. 在 **"應用"** 頁面中，選擇要開始使用的應用。

    將打開應用的新頁面，您可以在其中登錄（如有必要）或開始使用該應用程式。

### <a name="access-and-use-the-my-apps-portal-on-an-ios-device"></a>在 iOS 設備上訪問和使用"我的應用"門戶

從 iPhone 或 iPad 設備查看和使用 **"我的應用"** 門戶，運行 iOS 版本 7 或更高版本。 您還可以安裝["我的應用"移動應用](https://itunes.apple.com/us/app/my-apps-azure-active-directory/id824048653?mt=8)，以在 iOS 設備上訪問組織的應用。

1. 在行動裝置上，啟動 Web 瀏覽器應用，如 Safari。

2. 轉到https://myapps.microsoft.com，或使用組織提供的連結（如果他們將您定向到自訂頁面，如https://myapps.microsoft.com/contoso.com。

    將顯示 **"應用"** 頁面，顯示組織擁有的所有可供您使用的所有基於雲的應用。

3. 在 **"應用"** 頁面中，選擇要開始使用的應用。

    將打開應用的新頁面，您可以在其中登錄（如有必要）或開始使用該應用程式。

### <a name="access-and-use-the-my-apps-portal-on-an-android-device"></a>在 Android 設備上訪問和使用"我的應用"門戶

在 Android 設備上查看和使用 **"我的應用"** 門戶。

1. 在行動裝置上，啟動 Web 瀏覽器應用，如 Google Chrome。

2. 轉到https://myapps.microsoft.com，或使用組織提供的連結（如果他們將您定向到自訂頁面，如https://myapps.microsoft.com/contoso.com。

    將顯示 **"應用"** 頁面，顯示組織擁有並可供您使用的所有基於雲的應用。

3. 在 **"應用"** 頁面中，選擇要開始使用的應用。

    將打開應用的新頁面，您可以在其中登錄（如有必要）或開始使用該應用程式。

## <a name="add-a-new-app-to-the-my-apps-portal"></a>向"我的應用"門戶添加新應用

如果管理員已授予您許可權，則可以向 **"應用"** 頁添加新應用。

1. 在 **"應用"** 頁面中，執行以下操作之一：
    - 如果您處於原始"我的應用"體驗中，請選擇"**添加應用**"，如下所示。

      ![在"我的應用"門戶中添加應用頁面](media/my-apps-portal/my-apps-portal-add-apps-page.png)

    - 如果您正在更新的"我的應用"體驗中，請選擇"**添加自助服務應用**"。

       ![在"我的應用"門戶中添加應用頁面，myapplications.microsoft.com](media/my-apps-portal/my-apps-portal-add-app-link.png)

2. 選擇要從提供清單中添加的應用，然後選擇 **"添加**"。

3. 應用將添加到 **"應用"** 頁上的清單。

    某些應用在添加之前可能需要管理員批准。 發生這種情況時，在管理員批准應用之前，不會將應用添加到**應用**頁面。

## <a name="start-a-cloud-based-app"></a>啟動基於雲的應用

可以從 **"我的應用"** 門戶啟動任何可用的基於雲的應用。 您只會看到您有權使用的應用。

- 在 **"應用"** 頁面中，選擇要開始使用的應用。

    將打開應用的新頁面，您可以在其中登錄（如有必要）或開始使用該應用程式。

## <a name="activities-in-the-my-apps-portal"></a>"我的應用"門戶中的活動

進入 **"我的應用"** 門戶後，您可以：

- 將應用程式組織為貴組織所建立和提供的各種類別。 如需詳細資訊，請參閱[在我的應用程式入口網站上存取和使用集合](my-applications-portal-workspaces.md)。

- 檢閱、更新或撤銷已授與應用程式的權限。 如需詳細資訊，請參閱[在我的應用程式入口網站中編輯或撤銷應用程式權限](my-applications-portal-permissions-saved-accounts.md)。

>[!Note]
>如果您未看到任何可用的集合或類別，這表示您的系統管理員尚未設定任何集合或類別或將其與您共用。 請洽詢組織的技術服務人員，以取得其他協助或權限來查看共用的集合。
