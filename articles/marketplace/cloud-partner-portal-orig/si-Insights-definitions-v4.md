<a name="seller-insights-definitions"></a>Seller Insights 定義
=======================

下表提供 Seller Insights 中使用的許多字詞的定義。

|  **術語**                 |  **定義**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Azure 授權類型**                               | 客戶用來購買 Azure 的授權合約類型。  也稱為「通道」。                                                  |
| **Azure 授權類型：雲端解決方案提供者**      | 終端客戶透過其雲端解決方案提供者 (作為您的轉銷商) 取得 Azure 和您的 Marketplace 供應項目。                 |
| **Azure 授權類型：企業**                   | 終端客戶透過 Enterprise 合約 (直接與 Microsoft 簽署) 取得 Azure 和您的 Marketplace 供應項目。                  |
| **Azure 授權類型：透過轉銷商的企業**  | 最終客戶通過轉銷商採購 Azure 和您的應用商店產品/服務，轉銷商為與 Microsoft 的企業協定提供便利。     |
| **Azure 授權類型：隨用隨付**                | 最終客戶通過直接與 Microsoft 簽署的"隨用付費"協定採購 Azure 和應用商店產品/服務。                |
| **收費金額 (CC)**                              | 向客戶收取的金額，以*客戶幣種*（CC） 計費。                                                                 |
| **收費金額 (PC)**                               | 向客戶收取的金額，以*支付貨幣*（PC） 表示。                                                                      |
| **收費日期**                                      | 計算客戶費用的日期（通常在使用期之後）。                                             |
| **雲端執行個體名稱**                              | 發生 VM 部署的 Microsoft Cloud。                                   |
| **雲實例名稱：Azure 全域**                | 公共全球微軟雲。                           |
| **雲實例名稱：Azure 政府**        | 針對政府特定的微軟雲為以下政府之一：中國，德國或美利堅合眾國。                           |
| **收集狀態**  | 對於特定費用，計費和收款生命週期的最新狀態。  例如，狀態可以是正在進行的收集、收集或退款。                  |
| **核心尺寸**  | 產品/服務運行的虛擬機器提供的虛擬內核數。               |
| **客戶縣/市**  | 客戶提供的城市名稱。 這可能與客戶 Azure 訂閱中的城市不同。  |
| **客戶通訊語言**        | 客戶慣用於通訊的語言。                                  |
| **客戶公司名稱**                  | 客戶提供的公司名稱。 這可能與客戶 Azure 訂閱中的城市不同。                                  |
| **客戶國家/地區**                       | 客戶提供的國家/地區名稱。  這可能與客戶 Azure 訂閱中的國家/地區不同。                               |
| **客戶貨幣 (CC)**                 | 客戶的慣用定價與計費貨幣。                            |
| **客戶電子郵件**                         | 最終客戶提供的電子郵件地址。  這可能與客戶 Azure 訂閱中的電子郵件地址不同。                   |
| **客戶名字**                              | 客戶提供的第一個名字。  這可能與客戶的 Azure 訂閱中提供的名稱不同。                    |
| **客戶姓氏**                               | 客戶提供的姓氏。  這可能與客戶的 Azure 訂閱中提供的名稱不同。                    |
| **客戶 ID**                               | 分配給客戶的唯一識別碼。  客戶可能具有零個或多個 Azure 應用商店訂閱。                    |
| **客戶付款類型**                            | 客戶使用的付款工具類型。  例如，這可能是卡、發票或電子支票。                             |
| **客戶郵遞區號**                             | 客戶提供的郵遞區號。  這可能與客戶的 Azure 訂閱中提供的郵遞區號不同。                                                                 |
| **客戶州**                                   | 客戶提供的狀態（位址）。  這可能與客戶 Azure 訂閱中提供的姓氏不同。                                                                 |
| **取得日期**                                    | 客戶購買您發佈的任何產品/服務的第一個日期。                 |
| **損失日期**                                        | 客戶取消以前購買的所有優惠中的最後一個日期。                                                         |
| **預估擴充費用 (CC)**                   | 給定 SKU（以客戶貨幣表示）的使用單位數量的估計擴展費用。 由於四捨五入或截斷錯誤，這個值可能並不精確。                            |
| **預估擴充費用 (PC)**                   | 根據日期使用方式的外匯兌換計算（以發行者的貨幣）計算給定 SKU 的使用單位數量的估計擴展費用。 由於四捨五入或截斷錯誤，這個值可能並不精確。                  |
| **預估付款 (PC)**                            | 根據計算使用日期（以發行者的貨幣）的外匯兌換計算，對給定 SKU 的使用單位數量的估計付款。 由於四捨五入或截斷錯誤，這個值可能並不精確。                        |
| **預估價格 (PC)**                             | 根據日期使用方式上的外匯兌換計算給定 SKU 的使用量單位的估計價格（以發行者的貨幣表示）。 由於四捨五入或截斷錯誤，這個值可能並不精確。                                             |
| **是新客戶**                                  | 該值將讓您知道，如果是新客戶首次消費或首次購買一個或多個應用商店產品/服務（或不）。  如果值為"獲取日期"的同一日曆月，則該值將為"是"。  如果客戶在報告日曆月之前購買了您的任何優惠，則該值將為"否"。                                                       |
| **是預覽 SKU**                                  | 該值將告知您是否將 SKU 標記為"預覽"。 如果 SKU 已相應地標記，並且只有您授權的 Azure 訂閱可以部署和使用此映射，則該值將為"是"。  如果 SKU 尚未標識為"預覽"，則該值將為"否"。                                          |
| **促銷連絡人是否加入宣告**                   | 該值將讓您知道，如果客戶主動選擇從發佈商進行促銷聯繫。 目前，我們沒有向客戶顯示該選項，因此我們一律指出 [否]。 一旦部署了此功能，我們就會開始據以更新。                                          |
| **Marketplace 授權類型**                         | Marketplace 供應項目的計費方法。                             |
| **Marketplace 授權類型：透過 Azure 計費**   | Microsoft 是您對於此 Marketplace 供應項目的代理人，並且會代替您向客戶收取費用。 (PAYG 信用卡或企業發票)       |
| **Marketplace 授權類型：自備授權** | VM 需要客戶提供的許可證金鑰才能部署。 Microsoft 不會針對此 Marketplace 供應項目的使用量向客戶收取費用。 |
| **Marketplace 授權類型：免費**                   | Marketplace 供應項目設定為對所有使用者都是免費的。 Microsoft 不會向客戶針對此 Marketplace 供應項目的使用量收取費用。    |
| **Marketplace 授權類型：Microsoft 為轉銷商**  | Microsoft 是您此 Marketplace 供應項目的轉銷商。                               |
| **應用商店訂閱公司名稱**  | 客戶在其 Azure 訂閱中提供的公司名稱。                           |
| **應用商店訂閱 ID**            | 與客戶用於購買應用商店產品/服務所用的 Azure 訂閱關聯的唯一識別碼。  它以前是 Azure 訂閱 GUID。  在某些情況下，如 SAAS 訂單，不需要訂閱。           |
| **Microsoft 費用 (CC)**                               | 以客戶的貨幣為單位之交易的 Microsoft 費用。                              |
| **優惠名稱**                                | 應用商店產品的名稱。                                                             |
| **優惠類型**                                | 微軟應用商店產品的類型。  例如，它可以是託管應用程式、虛擬機器許可證或容器。                           |
| **訂單取消日期**                         | Marketplace 訂單取消日期。                                                       |
| **訂單識別碼**                                  | 市場服務的客戶訂單的唯一識別碼。  基於虛擬機器使用方式的優惠不與訂單相關聯。                 |
| **訂單購買日期**                       | 「Marketplace 訂單」建立日期。                                                        |
| **Order Quantity**                       | 與 Azure 訂閱關聯的同一 SKU 的訂單數量。  對於 SAAS 訂單，它表示與此訂單實例關聯的席位數。                                            |
| **訂單狀態**                              | 在資料上次重新整理時，Marketplace 的狀態。  例如，訂單可以是活動訂單或已取消。             |
| **訂單狀態：有效**                           | 客戶已購買訂單，尚未取消訂單。   |
| **訂單狀態：已取消**                        | 客戶以前購買了訂單，隨後取消了訂單。        |
| **付款金額 (PC)**                             | 以您的慣用付款貨幣 (PC) 支付給您的金額。                                       |
| **付款貨幣 (PC)**                           | 用於您付款的貨幣。                                                                |
| **付款日期**                                    | 付款請求從 Microsoft 發送到您選擇的金融機構的日期。                                                                         |
| **付款狀態**                                  | 上次刷新資料時的支出狀態。                                   |
| **付款狀態：正在進行中**               | 交易尚未準備付款。 （有關詳細資訊，請參閱收集狀態）                             |
| **付款狀態：已付費**                          | 交易包含在過去的付款計算。 正的值是已付費，而負的值是對總金額的淨值。 |
| **付款狀態：即將付款**                   | 交易已經準備好進行付款，並且會包含在下次的可用付款計算。                      |
| **價格 (CC)**                                   | 指定 SKU 單位使用量的價格 (以客戶的貨幣為單位)。                                                                  |
| **供應商電子郵件**                                 | 參與 Microsoft 與最終客戶之間關係的供應商的電子郵件地址。 如果客戶是通過轉銷商的企業，則轉銷商將是轉銷商。  如果涉及雲解決方案供應商 （CSP），則為 CSP。                                                                 |
| **提供程式名稱**                                 | Microsoft 和最終客戶之間的關係中涉及的供應商的名稱。 如果客戶是通過轉銷商的企業，則轉銷商將是轉銷商。  如果涉及雲解決方案供應商 （CSP），則為 CSP。                                                                 |
| **資源 URI**                                  | 部署工件的路徑。                                            |
| **Sku**                                           | 發佈期間定義的 SKU 名稱。 供應項目可以有多個 SKU，但 SKU 只能與單一供應項目相關聯。              |
| **SKU 計費類型**                                 | SKU 的計費方法。                                                                                                             |
| **SKU 計費類型：自帶許可證**         | VM 或服務需要客戶提供的許可證金鑰才能部署。 Microsoft 不會向客戶收取使用這些應用商店產品/服務的費用。   |
| **SKU 計費類型：微軟作為轉銷商**          | Microsoft 是您此 SKU. 的轉銷商。   |
| **SKU 計費類型：免費**                           | SKU 設定為對所有使用者都是免費的。 Microsoft 不會針對此 SKU 的使用量向客戶收取費用。                           |
| **SKU 計費類型：付費**                           | Microsoft 是您對於此 SKU 供應項目的代理人，並且會代替您向客戶收取費用。 (PAYG 信用卡或企業發票)                   |
| **SKU 計費類型：試用**                          | 客戶在其試用期間內，如果他們沒有取消或刪除，就會轉換成付費。                                         |
| **稅額 (CC)**                                  | 以客戶貨幣 (CC) 為單位套用至客戶帳單的稅額。                                                               |
| **交易日期**                                 | 您的付款報告中的交易記錄日期。                                                                           |
| **事務類型**                               | 報告的交易類型。  例如，這可能是費用、退款或支出調整。                                                                                        |
| **試用結束日期**                                 | 此訂單將結束或已結束的日期。                                                                           |
| **使用日期**                                       | 客戶使用量發生的日期。                                                                                                             |
| **使用數量**                                       | 所報告客戶的 SKU 使用量。 對於 VM 映像，使用量記錄代表該 VM 大小和 SKU 在報告期間的使用量。                                               |
| **使用量參考**                                       | 針對指定 SKU 的一或多天客戶使用量識別碼，與付款報告中的某個項目相關聯。                                               |
| **使用類型**                                       | 測量之使用量類型的描述。 (「標準化使用量」或「未處理使用量」)                                                              |
| **使用類型：標準化使用**                | 帳戶對於使用量相關之 VM 核心數目的標準化使用時數：[VM 核心數目] x [未處理使用量時數]。 指定為 "SHAREDCORE" 的 VM 使用 1/6 (或0.1666) 作為 [VM 核心數目] 乘數。                                        |
| **使用類型：原始用法**                        | 您的 Marketplace 供應項目使用時數。  它以"日曆"小時形式呈現。  它是虛擬機器運行的時間數。                           |
| **使用量單位**                                   | 所述使用量的測量單位。 VM 一律都是用以小時為單位的度量來測量。                               |
| **VM 大小**                                       | 代表與 Azure 供應項目配合的虛擬機器硬體大小。 範例包括 `Basic_A0`、`Standard_A11`、`Standard_D12` 和 `Standard_G4`。   |
|  |  |
