若要執行 Cmdlet，您可以使用 Azure Cloud Shell (免費的互動式殼層)。 它具有預先安裝和設定的共用 Azure 工具，可與您的帳戶搭配使用。 選擇**複製複製程式**碼,並將其貼到雲殼體中。 然後選擇 **「輸入」** 以運行它。 有幾種方法可以啟動雲外殼:


|  |   |
|-----------------------------------------------|---|
| 在您的瀏覽器中開啟 Cloud Shell。 | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| 選擇 Azure 門戶右上角工具列上的 **「雲殼」** 按鈕。 | [![門戶中的雲外殼](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


如果您不想使用 Azure Cloud Shell，您可改為在本機安裝 PowerShell。 如果您選擇在本機安裝及使用 PowerShell，請務必安裝最新版的 Azure Resource Manager PowerShell Cmdlet。 PowerShell cmdlet 經常更新。 通常需要更新 PowerShell cmdlet 才能獲取最新的功能功能。 如果沒有,則可能會遇到問題。

要尋找您在本地執行的 PowerShell 版本,請使用 **「獲取模組 -清單可用 Az"cmdlet。** 若要更新，請參閱[安裝 Azure PowerShell 模組](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)。 有關詳細資訊,請參閱[如何安裝和設定 Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/overview)。

如果您在 macOS 上使用 PowerShell,請按照在[macOS 上安裝 PowerShell](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)的步驟操作。
