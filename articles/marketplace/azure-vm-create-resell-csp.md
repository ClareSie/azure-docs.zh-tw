# <a name="how-to-resell-your-offer-through-csp"></a>如何透過 CSP 轉售您的供應專案

## <a name="resell-through-csp"></a>透過 CSP 轉售

在 [ **轉售透過 csp** ] 頁面上 (從合作夥伴中心的左側導覽功能表中選取) 您可以將供應專案提供給 [雲端解決方案提供者 (CSP)](https://azure.microsoft.com/offers/ms-azr-0145p/) 計畫中的合作夥伴使用，以擴充供應專案的範圍。 如此可讓轉銷商向其客戶銷售您的供應項目，並建立配套解決方案。 所有自備授權 (BYOL) 方案都會自動加入方案中。 您也可以選擇加入非 BYOL 方案。

選取應能轉售您供應專案的合作夥伴：

- CSP 計畫中的任何合作夥伴
- CSP 計畫中我選定的特定合作夥伴
- 不提供給 CSP 計畫中的任何合作夥伴

選取 [ **儲存草稿** ]，然後繼續前往左側導覽功能表中的下一個索引標籤： [ **試用磁片磁碟機** ]。
