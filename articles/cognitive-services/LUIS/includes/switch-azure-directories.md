如果您沒有看到您所建立的應用程式或與您共用的應用程式，您可能需要切換至不同的 Azure 目錄。

1. 按一下螢幕右上角的 [頭像]。 然後，按一下 [ **切換到不同的 Azure 目錄**]。 
2. 在出現的視窗中，請務必選取包含與您共用的 LUIS 資源所在的 Azure 目錄。
