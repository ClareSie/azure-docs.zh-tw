# <a name="view-or-change-your-settings-in-the-my-account-portal"></a>在我的帳戶入口網站中查看或變更您的設定

您可以從**我的帳戶**入口網站的 [**設定] & [隱私權**] 頁面，查看或變更我的帳戶入口網站中的帳戶設定，例如 [語言] 或 [時區]。

## <a name="view-and-manage-your-language-and-regional-settings"></a>查看及管理您的語言和地區設定

1. 登入您的公司或學校帳戶，然後前往 https://myaccount.microsoft.com/ 頁面。

1. 從左側導覽窗格中選取 [**設定] & 隱私權**]，或從 [**設定] & [隱私權**] 區塊選取 [**視圖設定和隱私權**] 連結。

    ![[我的帳戶] 頁面，顯示反白顯示的設定和隱私權連結](media/my-account-portal/my-account-portal-privacy.png)

1. 在 [ *Language & 區域] ([預覽]) *中，檢查或變更下列設定：
    * *顯示語言*：用於按鈕、功能表和 Microsoft 網站中某些內容的語言。
    * *慣用語言*：您瞭解並用來建立或取用內容的語言。
    * *地區格式*： Microsoft 網站會使用此設定來格式化日期和時間。
    * *時區*：用來有效地排程 Microsoft 行事曆中的會議。

變更顯示語言之後，建議您先登出並登入您的工作或學校帳戶，以確保您的顯示語言會重新整理。

> [!NOTE]
> 在某些組織中，您無法管理 *顯示語言* 設定。 如果您的顯示語言已停用，如下列範例螢幕擷取畫面所示，請與您的 IT 系統管理員聯繫以取得協助：
>
> ![我的帳戶頁面，顯示組織已停用顯示語言設定](media/my-account-portal/my-account-portal-managed-language-settings.png)
