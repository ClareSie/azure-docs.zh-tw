# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS 保護

內容傳遞網路依設計提供 DDoS 保護。 除了可吸收體積型攻擊的全球容量以外，Azure CDN 還有額外的 DDoS 保護，而不需要額外成本。

## <a name="azure-cdn-from-microsoft"></a>來自 Microsoft 的 Azure CDN

來自 Microsoft 的 azure CDN 會受到[Azure 基本 DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview)的保護。 根據預設，它會從 Microsoft 平臺整合到 Azure CDN，而且不會有額外費用。 Microsoft 全球部署網路的 Azure CDN 完整規模和容量，可讓您透過 always on 流量監視和即時緩和功能，防禦常見的網路層攻擊。 基本 DDoS 保護也會對抗以 CDN 端點為目標的最常見且經常發生的第7層 DNS 查詢洪水和第3層和第4體積型攻擊。 這項服務也具有經證實的追蹤記錄，可保護 Microsoft 的企業和取用者服務免于大規模的攻擊。

## <a name="azure-cdn-from-verizon"></a>Azure CDN from Verizon

來自 Verizon 的 Azure CDN 受到 Verzion 的專屬 DDoS 緩和平臺保護。 根據預設，它會從 Verizon 整合到 Azure CDN，而且不會有額外費用。 它會針對最常見、經常發生的第7層 DNS 查詢洪水和以 CDN 端點為目標的第3層和第4層體積型攻擊提供基本保護。

## <a name="azure-cdn-from-akamai"></a>Azure CDN from Akamai

來自 Akamai 的 Azure CDN 受到 Akamai 的專屬 DDoS 緩和平臺保護。 根據預設，它會從 Akamai 整合到 Azure CDN，而且不會有額外費用。 它會針對最常見、經常發生的第7層 DNS 查詢洪水和以 CDN 端點為目標的第3層和第4層體積型攻擊提供基本保護。
