# <a name="find-help-and-open-a-support-ticket-for-azure-active-directory"></a>尋求 Azure Active Directory 方面的協助及開立支援票證
Microsoft 對 Azure Active Directory (Azure AD) 提供全球性的技術、售前、帳單和訂用帳戶支援。 Microsoft Azure 付費訂用帳戶與試用訂用帳戶則提供線上和電話支援 。 電話支援和線上帳單支援已推出其他語言版本。 

## <a name="find-help-without-opening-a-support-ticket"></a>在不開啟支援票證的情況下尋找說明

在建立支援票證之前，請先查看下列資源以尋求解答和資訊。 

* 如需 IT 專業人員和開發人員所適用的操作說明資訊或程式碼範例等內容，請參閱 [docs.microsoft.com 上的技術文件](https://docs.microsoft.com/azure/active-directory/)。

* [Microsoft 技術社區](https://techcommunity.microsoft.com/)是我們 IT 專業合作夥伴和客戶協作、共用和學習的場所。 [微軟技術社區資訊中心](https://techcommunity.microsoft.com/t5/Community-Info-Center/ct-p/Community-Info-Center)用於公告、博客文章、詢問我任何內容 （AMA） 與專家的互動等。 您也可以[加入社群以提交意見](https://techcommunity.microsoft.com/t5/Communities/ct-p/communities)。


## <a name="open-a-support-ticket"></a>開啟支援票證

如果使用自助資源找不到解答，您可以開啟線上支援票證。 開啟的每個支援票證應該只包含一個問題，如此我們才能針對您的問題為您聯繫身為主題專家的支援工程師。 此外，Azure Active Directory 工程小組會根據發生的事件設定其工作優先順序，因此您通常要參與服務改進。

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-azure-portal"></a>如何在 Azure 入口網站中開啟 Azure AD 的支援票證

> [!NOTE]
> 對於計費或訂閱問題，您必須使用[Microsoft 365 管理中心](https://admin.microsoft.com)。
> 

1. 登入 [Azure 入口網站](https://portal.azure.com)，並開啟 **Azure Active Directory**。
   
2. 向下捲動至 [疑難排解 + 支援]****，然後選取 [新增支援要求]****。
   
3. 在 [基本概念]**** 刀鋒視窗中，針對 [問題類型]**** 選取 [技術]****。

4. 選擇您的**訂閱**。

5. 對於**服務**，請選擇**Azure 活動目錄**。

6. 為請求創建**摘要**。 摘要必須低於 140 個字元。
  
7. 選擇**問題類型**，然後為該類型選擇一個類別。 此時，您還會收到問題類別的自助資訊。
  
7. 新增其餘的問題資訊，然後按一下 [下一步]****。 

8. 此時，在 **"解決方案"** 邊欄選項卡中為您提供自助解決方案和文檔。 如果解決方案沒有解決您的問題，請按一下 **"下一步**"。 

9. 在 **"詳細資訊"** 邊欄選項卡上，填寫所需的詳細資訊並選擇[嚴重性](https://azure.microsoft.com/support/plans/response/)。 
 
![image](https://user-images.githubusercontent.com/13383753/76565580-1c284900-6468-11ea-8c0f-85af98097b6f.png)
 
10. 提供您的聯繫資訊，然後選擇 **"下一步**"。 


8. 提供您的連絡資訊，然後選取 [建立]****。
  ![問題類別自助螢幕截圖](./media/active-directory-troubleshooting-support-howto/open-support-ticket.png)

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-microsoft-365-admin-center"></a>如何在 Microsoft 365 管理中心打開 Azure AD 的支援票證

> [!NOTE]
> [Microsoft 365 管理中心](https://admin.microsoft.com)中支援 Azure AD 僅為管理員提供。 

1. 使用具有企業移動性與安全 （EMS） 許可證的帳戶登錄到[Microsoft 365 管理中心](https://admin.microsoft.com)。

2. 在 [支援]**** 圖格中，選取 [新增服務要求]****：

3. 在 [支援概觀]**** 頁面上，選取 [身分識別管理]**** 或 [使用者和網域管理]****：

4. 對於 [功能]****，選取您要支援的 Azure AD 功能。

5. 對於 [徵兆]****，選取適當的徵兆、摘要說明您的問題並提供相關的詳細資料，然後選取 [下一步]****。

6. 選取其中一個提供的自助資源，或者選取 [是，繼續]**** 或 [否，取消要求]****。

7. 如果繼續，系統會要求您提供更多詳細資料。 您可以附加任何表示此問題的檔案，然後選取 [下一步]****。

8. 提供您的連絡資訊，然後選取 [提交要求]****。
