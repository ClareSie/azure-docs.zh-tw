# <a name="team-data-science-process-for-developer-operations"></a>適用於開發人員作業的 Team Data Science Process

本文將探索對「進階分析」和「認知服務」解決方案實作而言特定的「開發人員作業」(DevOps) 功能。 這些訓練教材會實行小組資料科學程式（TDSP）和 Microsoft 與開放原始碼軟體和工具組，以協助構思、執行和傳遞資料科學解決方案。 它會參考涵蓋 DevOps 工具鏈的主題，專屬於資料科學和 AI 專案與解決方案。
