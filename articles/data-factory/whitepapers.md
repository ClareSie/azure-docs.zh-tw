# <a name="azure-data-factory-whitepapers"></a>Azure 資料工廠白皮書 

白皮書允許您在更深的級別上探索 Azure 資料工廠。 本文為您提供了 Azure 資料工廠可用的白皮書清單。

| **白皮書** | **描述** |
| --- | --- |
|[Azure 資料工廠 — 雲中的資料整合](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | 本文介紹了 Azure 資料工廠如何使您能夠構建現代資料倉儲、啟用高級分析以驅動智慧 SaaS 應用程式以及將 SQL Server 整合服務包提升到 Azure。|
|[使用 Azure 資料工廠從本地關係資料倉儲到 Azure 的資料移轉](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | 本文討論了使用 Azure 資料工廠將數十TB資料從現有本地關係資料倉儲（例如，Netezza、Oracle、Teradata、SQL 伺服器）遷移到 Azure（例如 Blob 存儲或 Azure 資料湖存儲）的複雜性。 圍繞 Azure 資料工廠向 Azure 進行大資料引入之旅的恢復能力、性能、可擴充性、管理和安全性進行了說明。 |
|[使用 Azure 資料工廠進行 SAP 資料整合](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | 本文總結了 Azure 資料工廠當前對 SAP 資料整合的支援，包括目標方案、SAP 連接器選項和不同要求的比較，以及資料工廠中每個 SAP 連接器的介紹。 | 
|[Azure 資料工廠：雲中的 SSIS](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| 本文介紹了為什麼要將現有的 SSIS 工作負載遷移到 Azure 資料工廠並解決常見的注意事項和問題。 然後，我們將介紹創建 Azure-SSIS IR 的技術詳細資訊，然後演示如何使用您可能熟悉的工具（如 SQL Server 管理工作室 （SSMS））通過 Azure 資料工廠上載、執行和監視包。 |
|[Azure 資料工廠 = 傳遞參數](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| 本文將介紹如何在管道和活動之間以及活動之間傳遞參數。 |
|[Azure 資料工廠 = DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| 本白皮書介紹了有關 Azure 資料工廠持續集成和部署的一些最佳實踐。 |
