# <a name="change-to-full-screen-view-for-a-vm-session-azure-bastion"></a>更改為 vm 會話的全屏視圖：Azure 堡壘

本文可説明您將虛擬機器視圖更改為全屏並在瀏覽器中返回。 在使用 VM 之前，請確保已遵循[創建堡壘主機](bastion-create-host-portal.md)的步驟。 然後，使用[RDP](bastion-connect-vm-rdp.md)或[SSH](bastion-connect-vm-ssh.md)連接到要使用的 VM。

## <a name="launch-the-clipboard-tool"></a>啟動剪貼簿工具

在遠端會話期間，通過選擇位於會話左中心的兩個箭頭啟動 Bastion 剪貼簿訪問工具調色板。

![工具](./media/bastion-vm-manage/left.png)

## <a name="select-full-screen"></a>選擇全屏

選擇 **"全屏**"按鈕可將會話切換到全屏體驗。 切換後，會話將重新初始化為全屏。

![全螢幕](./media/bastion-vm-manage/full-screen.png)
