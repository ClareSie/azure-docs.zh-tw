# <a name="cloudsimple-private-cloud-permission-model-of-vmware-vcenter"></a>VMware vCenter 的雲簡單私有雲許可權模型

CloudSimple 保留對私有雲環境的完全管理存取權限。 每個 CloudSimple 客戶都被授予足夠的管理許可權，以便能夠在其環境中部署和管理虛擬機器。  如果需要，您可以臨時升級執行管理功能的許可權。

## <a name="cloud-owner"></a>雲擁有者

創建私有雲時，雲**擁有者**使用者將創建在 vCenter 單一登入域中，並具有**雲擁有者角色**存取權限來管理私有雲中的物件。 此使用者還可以設置其他[vCenter 標識源](set-vcenter-identity.md)，以及私有雲 vCenter 的其他使用者。

> [!NOTE]
> 雲簡單私有雲 vCenter 的預設使用者cloudowner@cloudsimple.local是在創建私有雲時。

## <a name="user-groups"></a>使用者群組

在部署私有雲期間創建名為**雲擁有者組**。 此組中的使用者可以管理私有雲上的 vSphere 環境的各個部分。 此組將自動授予**雲擁有者角色**許可權，**並且 CloudOwner**使用者將作為此組的成員添加。  CloudSimple 創建具有有限許可權的其他組，便於管理。  您可以將任何使用者添加到這些預先創建的組，下面定義的許可權將自動分配給組中的使用者。

### <a name="pre-created-groups"></a>預創建組

| 群組名稱 | 目的 | 角色 |
| -------- | ------- | ------ |
| 雲擁有者組 | 此組的成員具有私有雲 vCenter 的管理許可權 | [雲擁有者角色](#cloud-owner-role) |
| 雲-全球-群集-管理-組 | 此組的成員在私有雲 vCenter 群集上具有管理許可權 | [雲群集-群集-管理員角色](#cloud-cluster-admin-role) |
| 雲-全球存儲-管理-管理組 | 此組的成員可以管理私有雲 vCenter 上的存儲 | [雲存儲-管理-角色](#cloud-storage-admin-role) |
| 雲-全球-網路-管理-組 | 此組的成員可以在私有雲 vCenter 上管理網路和分散式埠組 | [雲網路-管理員-角色](#cloud-network-admin-role) |
| 雲-全球-VM-管理員-組 | 此組的成員可以在私有雲 vCenter 上管理虛擬機器 | [雲-VM-管理員-角色](#cloud-vm-admin-role) |

要授予單個使用者管理私有雲的許可權，請創建添加到相應組的使用者帳戶。

> [!CAUTION]
> 新使用者只能添加到*雲擁有者組*、*雲-全球群集-管理員組*、*雲-全球-存儲-管理員組*、*雲-全球-網路-管理員組*或*雲-全球-VM-管理員組*。  添加到*管理員*組的使用者將自動刪除。  只能將服務帳戶添加到*管理員*組，並且不得使用服務帳戶登錄到 vSphere Web UI。
