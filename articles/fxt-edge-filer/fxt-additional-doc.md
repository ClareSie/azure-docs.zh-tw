# <a name="additional-documentation-for-azure-fxt-edge-filer"></a>Azure FXT Edge Filer 的其他檔

此 docs.microsoft.com 網站以外的部分資源可能會協助您瞭解及使用您的 Microsoft Azure FXT Edge 檔案管理工具混合式快取。

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT for Azure 快取檔

[Avere vFXT for Azure](../avere-vfxt/index.yml) 是使用 Microsoft Azure 中虛擬節點的類似叢集快取產品。 它是使用 Azure 入口網站所建立，但使用與 Azure FXT Edge Filer 硬體快取相同的控制台設定軟體。

## <a name="legacy-hardware-documentation"></a>舊版硬體檔

這項技術先前在 Avere 系統的產品中使用。 您可以在 [Avere 舊版檔網站](https://azure.github.io/Avere/)上取得這些舊版產品的使用者檔。

具體來說，這些檔可能會有有用的詳細資料：

* 叢集設定[指南](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html)-包含一般叢集設定和管理工作的總覽資訊，以及叢集設定頁面的詳細說明

* [儀表板指南](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) -說明 [控制台] **儀表板** 索引標籤的功能

* [FXT 叢集建立指南](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) -先前產品的叢集建立指南
