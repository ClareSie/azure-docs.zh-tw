# <a name="appsource-dynamics-365-finance-and-operations-functional-validation"></a>AppSource Dynamics 365 財務和作業功能驗證

若要完成 [合作夥伴中心](https://partner.microsoft.com/dashboard/home)中的第一次發行，Dynamics 365 財務和營運的優惠需要兩項功能驗證：

- 上傳顯示基本功能的 Dynamics 365 環境示範影片。
- 提供示範解決方案 [生命週期服務](https://lcs.dynamics.com/) (LCS) 環境的螢幕擷取畫面。

> [!NOTE]
> 後續的重新認證發佈不需要示範。 若要深入瞭解，請參閱 [AppSource 原則檔](https://docs.microsoft.com/legal/marketplace/certification-policies#1440-dynamics-365-finance-ops)。

## <a name="how-to-validate"></a>如何驗證

有兩個選項可供功能驗證：

- 在太平洋標準時間 (PST) 上班時間舉行30分鐘的會議通話，以示範及記錄 [LCS](https://lcs.dynamics.com/) 環境與解決方案，或
- 在合作夥伴中心中，移至 [[商業 Marketplace](https://partner.microsoft.com/dashboard/commercial-marketplace/overview)  >  **總覽**] 並上傳供應專案補充內容索引標籤上的示範影片 URL 和 LCS 螢幕擷取畫面。

Microsoft 認證小組會審核影片和檔案，然後核准解決方案或您接下來的步驟的電子郵件。

### <a name="option-1-30-minute-conference-call"></a>選項 1:30 分鐘會議通話

若要排程最後的評論通話，請與 [appsourceCRM@microsoft.com](mailto:appsourceCRM@microsoft.com) 您的供應專案名稱和太平洋時間上午8點到下午5點之間的一些可能時間點聯絡。

### <a name="option-2-upload-a-demo-video-and-lcs-screenshots"></a>選項2：上傳示範影片和 LCS 螢幕擷取畫面

1. 錄製影片，並將位址上傳至您選擇的主控網站。 請遵循這些方針：

    - 由 Microsoft 認證團隊觀看。
    - 長度不超過20分鐘。
    - 在 Dynamics 365 環境中，最多包含三個核心功能，可供您解決您的解決方案。

    > [!NOTE]
    > 如果現有的行銷影片符合指導方針，則可接受使用。

2. 採用與您想要發佈的供應專案或解決方案相符的 [LCS](https://lcs.dynamics.com/) 環境的下列螢幕擷取畫面。 它們必須清楚足以讓認證小組讀取文字。 將螢幕擷取畫面儲存為 JPG 檔案。 您可以 [appSourceCRM@microsoft.com](mailto:appSourceCRM@microsoft.com) 為您的 LCS 環境提供許可權，以便驗證設定，而不是提供螢幕擷取畫面。

    2. 將文字和任何 JPG 檔案新增至 [商用](https://partner.microsoft.com/dashboard/commercial-marketplace/overview) 市集中的 .zip 檔案，並在您的財務和營運供應專案的 **補充內容** 索引標籤上合作夥伴中心。
