# <a name="unity-showcase-application"></a>Unity 展示應用程式

Unity 展示應用程式是以 Unity 為基礎的範例應用程式，可示範 Azure 遠端轉譯功能。 其可在 Unity 編輯器中和 Hololens 2 上執行。

![展示應用程式](./media/showcase-app.png "展示應用程式")

## <a name="source-repository"></a>來源存放庫

在 [ARR 範例存放庫](https://github.com/Azure/azure-remote-rendering/tree/master/Unity/Showcase)中可找到此應用程式的原始程式碼以及建置/部署指示。
