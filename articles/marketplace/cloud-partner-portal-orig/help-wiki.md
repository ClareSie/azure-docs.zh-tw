# <a name="dynamics-365-for-operations-help-wiki"></a>Dynamics 365 Operations 說明 Wiki

Microsoft Dynamics 365 for Operations 是適用于企業的 Microsoft 商務應用程式。 人們能夠從中透過提供即時見解和資料的智慧使用者介面，快速地做出更精準的決策。 其提供經過實證的商務邏輯，讓企業能夠更快地重新設計商務程序，以進行轉型和創新。 企業能夠從中增加彈性，以依照自己的步調成長，而且業務幾乎可以隨時隨地進行，讓他們可將營運範圍擴展到全球，以符合商務需求。

本使用者指南可協助您了解如何讓 Dynamics 365 for Operations 適用於您的業務。

如需詳細資訊，請參閱 [Dynamics 365 Operations 說明 wiki](https://docs.microsoft.com/dynamics365/operations/)。
