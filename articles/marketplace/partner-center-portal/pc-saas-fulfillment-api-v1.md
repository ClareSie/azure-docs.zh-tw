# <a name="saas-fulfillment-apis-version-1-deprecated"></a>SaaS 履行 Api 第1版 (已淘汰) 

> [!WARNING]
> 此初始版本的 SaaS 履行 API 已淘汰;相反地，請使用 [SaaS 履行 API V2](./pc-saas-fulfillment-api-v2.md)。  此 API 的初始版本目前只會維護以提供現有的發行者。 

此 API 版本的檔已無法再使用。 如果您覺得需要存取它，請聯絡 saas-api-v1@microsoft.com 。

針對所有新的開發，即使您已經使用 V1，也請移至 [V2 版本的 API](./pc-saas-fulfillment-api-v2.md) 。 此版本將在未來完全淘汰。
