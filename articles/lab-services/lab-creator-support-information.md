# <a name="view-support-information-lab-creator-in-azure-lab-services"></a>在 Azure 實驗室服務中 (實驗室建立者查看支援資訊) 
本文說明如何以實驗室建立者 () 可以查看下列支援資訊：

- URL
- 電子郵件
- 電話
- 其他指示

當您在實驗室帳戶中建立實驗室時，若遇到任何技術問題，您可以使用這項資訊來取得協助。

 
## <a name="view-support-information"></a>檢視支援資訊
1. 登入 [Azure 實驗室服務入口網站](https://labs.azure.com)。
2. 選取頁面右上角的問號 (**？**) 。 
3. 確認其中顯示 [檢視支援網站]、[電子郵件支援] 及 [支援電話號碼] 的連結。

    ![檢視支援資訊](./media/lab-creator-support-information/support-information.png)
