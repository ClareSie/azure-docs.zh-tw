# <a name="azure-spatial-anchors-support-options"></a>Azure 空間錨點支援選項

## <a name="open-a-tech-support-ticket"></a>開啟技術支援票證

若要在 Azure 入口網站中為 Azure 空間錨點開啟技術支援票證：

1. 當 [Azure 入口網站](https://azure.microsoft.com/account/) 開啟時，請從頂端功能表列中選取 [說明] 圖示，然後選取 [說明 **+ 支援** ] 按鈕。

![Azure 入口網站說明和支援](./media/spatial-anchor-support.png)

1. 開啟 [說明 + 支援] 頁面，選取 [ **+ 新增支援要求**]。

![Azure 入口網站新的支援要求](./media/spatial-anchor-support2.png)

1. 完成支援票證欄位時：

- 問題類型：技術
- 服務：空間錨點

![Azure 入口網站支援票證欄位](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>社群支援

如需使用者群體的支援，請參閱 [Stack Overflow 上的 Azure 空間錨點](https://stackoverflow.com/questions/tagged/azure-spatial-anchors)。

## <a name="provide-content-feedback"></a>提供內容意見反應

在每個內容文章的底部，有機會開啟 GitHub 問題，並提供關於 Azure 空間錨點檔內容的意見反應。

## <a name="provide-product-feedback"></a>提供產品意見反應

若要提供意見反應、分享 Azure 空間錨點服務的想法或建議，或投票給其他人所提交的構想，請造訪 [Azure 空間錨點意見反應論壇](https://feedback.azure.com/forums/919252-azure-spatial-anchors)。

## <a name="next-steps"></a>後續步驟

如需 Azure 空間錨點的相關常見問題，請參閱 [常見問題](spatial-anchor-faq.md) 頁面。
