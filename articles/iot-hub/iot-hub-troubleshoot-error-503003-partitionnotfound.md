# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

本文介紹了**503003 分區未發現**錯誤的原因和解決方案。

## <a name="symptoms"></a>徵狀

向 IoT 中心的請求失敗，錯誤**503003 分區未找到**。

## <a name="cause"></a>原因

此錯誤是 IoT 中心的內部錯誤，可能是暫時性的。 請參閱[IoT 中心內部伺服器錯誤原因](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause)。

## <a name="solution"></a>解決方法

請參閱[IoT 中心內部伺服器錯誤的解決方案](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution)。
