# <a name="resource-types-that-extend-capabilities-of-other-resources"></a>擴展其他資源功能的資源類型

擴展資源是添加到其他資源功能的資源。 例如，資源鎖是擴展資源。 將資源鎖定應用於其他資源，以防止其被刪除或修改。 創建資源鎖本身沒有意義。 擴展資源始終應用於其他資源。
