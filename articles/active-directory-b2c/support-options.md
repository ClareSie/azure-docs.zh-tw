# <a name="azure-active-directory-b2c-file-support-requests"></a>Azure Active Directory B2C：提出支援要求
可以使用以下步驟在 Azure 門戶上為 Azure 門戶提交 Azure 活動目錄 B2C（Azure AD B2C） 的支援請求：

1. 從 B2C 租用戶切換至另一個擁有相關聯 Azure 訂用帳戶的租用戶。 後者通常為員工租用戶，或是您註冊 Azure 訂用帳戶時所建立的預設租用戶。 若要深入了解，請參閱 [Azure 訂用帳戶如何與 Azure AD 有關](../active-directory/fundamentals/active-directory-how-subscriptions-associated-directory.md)。

    ![突出顯示租戶選擇的 Azure 門戶](./media/support-options/support-switch-dir.png)

1. 切換租用戶之後，請按一下 [說明 + 支援] ****。

    ![説明 + Azure 門戶中突出顯示的支援磁貼](./media/support-options/support-support.png)

1. 按一下 **"新建支援請求**"。

    ![Azure 門戶中突出顯示的新支援請求磁貼](./media/support-options/support-new.png)

1. 在 [基本]**** 刀鋒視窗中，使用這些詳細資料，然後按 [下一步]****。

    * [問題類型]**** 是 [技術]****。
    * 選擇適當的 [訂用帳戶] ****。
    * [服務]**** 是 [Active Directory]****。
    * 選擇適當的 [支援方案] ****。 如果您沒有任何方案，可以在[這裡](https://azure.microsoft.com/support/plans/)註冊一個。

     ![Azure 門戶中突出顯示的"下一步"按鈕的基礎知識頁面](./media/support-options/support-basics.png)

1. 在 [問題]**** 刀鋒視窗中，使用這些詳細資料，然後按 [下一步]****。

    * 選擇適當的 [嚴重性] **** 層級。
    * [問題類型]**** 是 [B2C]****。
    * 選擇適當的 [類別] ****。
    * 在 [詳細資料] **** 欄位中描述您的問題。 提供詳細資料，例如 B2C 租用戶名稱、問題的描述、錯誤訊息、相互關聯識別碼 (如果有的話) 等等。
    * 在 [時間範圍] **** 欄位中，提供發生問題的日期和時間 (包括時區)。
    * 在 [檔案上傳] **** 底下，上傳所有螢幕擷取畫面以及您認為對解決問題有幫助的檔案。

     ![Azure 門戶中突出顯示的"下一步"按鈕的問題頁面](./media/support-options/support-problem.png)

1. 在 [連絡資訊] **** 刀鋒視窗中，加入您的連絡資訊。 按一下 **[建立]**。

    ![在門戶中突出顯示"創建"按鈕的聯繫資訊頁面](./media/support-options/support-contact.png)

1. 提交支援要求之後，您可在「開始面板」上按一下 [說明 + 支援]****，然後按一下 [管理支援要求]****。

## <a name="known-issue-filing-a-support-request-in-the-context-of-a-b2c-tenant"></a>已知問題：在 B2C 租用戶的內容中提出支援要求

若您遺漏上述的步驟 2，且嘗試在 B2C 租用戶的內容中建立支援要求，您將會看到以下錯誤。

> [!IMPORTANT]
> 請不要嘗試在 B2C 租用戶中註冊新的 Azure 訂用帳戶。

![Azure 門戶中未顯示訂閱錯誤](./media/support-options/support-no-sub.png)
