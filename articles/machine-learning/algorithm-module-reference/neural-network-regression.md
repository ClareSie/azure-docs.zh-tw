# <a name="neural-network-regression-module"></a>神經網路回歸模組

*使用類神經網路演算法建立迴歸模型*  
  
 類別：機器學習/初始化模型/回歸
  
## <a name="module-overview"></a>模組概觀  

本文介紹 Azure 機器學習設計器（預覽）中的模組。

使用此模組使用可自訂的神經網路演算法創建回歸模型。
  
 雖然廣泛用於深入學習和模型化複雜的問題，例如影像辨識已知類神經網路，也就是輕鬆地調整迴歸問題。 如果使用自動調整的加權，而且可以近似其輸入的非線性函數，任何類別的統計模型可以稱為類神經網路。 因此類神經網路的迴歸適合用於較傳統的迴歸模型無法在其中納入解決方案的問題。
  
 神經網路回歸是一種受監督的學習方法，因此需要*一個標記的資料集*，其中包括一個標籤列。 迴歸模型來預測數值，因為標籤資料行必須是數值資料類型。  
  
 您可以通過提供模型和標記的資料集作為訓練模型的輸入來[訓練模型](./train-model.md)。 然後，訓練模型可用於預測新輸入示例的值。  
  
## <a name="configure-neural-network-regression"></a>配置神經網路回歸 

神經網路可以廣泛自訂。 本節介紹如何使用兩種方法創建模型：
  
+ [使用預設體系結構創建神經網路模型](#bkmk_DefaultArchitecture)  
  
    如果接受預設神經網路架構，請使用 **"屬性"** 窗格設置控制神經網路行為的參數，例如隱藏層中的節點數、學習速率和正常化。

    如果您對神經網路是新加入的，請按一下此處。 該模組支援許多自訂項以及模型調優，而不了解神經網路。 

+ 為神經網路定義自訂體系結構 

    如果要添加額外的隱藏圖層，或完全自訂網路架構、其連接和啟動功能，請使用此選項。
    
    如果您已經有點熟悉神經網路，則此選項是最佳選擇。 您可以使用 Net+ 語言定義網路架構。  

##  <a name="create-a-neural-network-model-using-the-default-architecture"></a><a name="bkmk_DefaultArchitecture"></a>使用預設體系結構創建神經網路模型

1.  將**神經網路回歸**模組添加到設計器中的管道中。 您可以在 **"機器學習**初始**化****"類別中找到**此模組。 
  
2. 通過設置 **"創建培訓師模式"** 選項，指示您希望如何訓練模型。  
  
    -   **單一參數**：如果您已經知道要如何配置模型，請選擇此選項。

    -   **參數範圍**：如果不確定最佳參數，並且想要運行參數掃描，請選擇此選項。 選擇要反覆運算的值範圍，[調諧模型 Hyper 參數](tune-model-hyperparameters.md)反覆運算您提供的所有可能組合設置，以確定生成最佳結果的超參數。   

3.  在 **"隱藏層"規範**中，選擇 **"完全連接"的情況下**。 此選項使用預設神經網路架構創建模型，對於神經網路回歸模型，該體系結構具有以下屬性：  
  
    + 網路正好有一個隱藏層。
    + 輸出層完全連接到隱藏層，而隱藏層完全連接到輸入層。
    + 隱藏層的節點數目可以由使用者設定 (預設值為 100)。  
  
    由於輸入層中的節點數由定型資料中的要素數決定，因此在回歸模型中，輸出層中只能有一個節點。  
  
4. 對於**隱藏節點數**，鍵入隱藏節點的數量。 預設值為具有 100 個節點的隱藏圖層。 （如果使用 Net# 定義自訂體系結構，則此選項不可用。
  
5.  對於**學習速率**，鍵入一個值，用於定義在每次反覆運算中執行的步驟，然後再進行更正。 較大的學習速率值可能會導致模型更快收斂，但可能衝過局部最小值。

6.  對於**學習反覆運算次數**，指定演算法處理訓練用例的最大次數。


8.  對於**動量**，鍵入一個值，作為以前反覆運算的節點的權重在學習期間應用。

10. 選擇選項"**隨機播放示例**"以更改反覆運算之間的事例順序。 如果取消選擇此選項，則每次運行管道時都會以完全相同的連續處理案例。
  
11. 對於**亂數種子，** 可以選擇鍵入用作種子的值。 如果要確保在同一管道的運行中重複性，指定種子值非常有用。
  
13. 連接訓練資料集和其中一個[培訓模組](module-reference.md)： 
  
    -   如果將 **"創建訓練器模式"** 設置為 **"單一參數**"，請使用[訓練器模型](./train-model.md)。  
  
   
14. 提交管道。  

## <a name="results"></a>結果

培訓完成後：

- 要保存已訓練的模型的快照，請在 **"訓練"模型**模組的右側面板中選擇 **"輸出**"選項卡。 選擇 **"註冊"資料集**圖示以將模型保存為可重用模組。
