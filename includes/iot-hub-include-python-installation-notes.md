* 使用中的 Azure 帳戶。 (如果您沒有帳戶，只需要幾分鐘的時間就可以建立[免費帳戶](https://azure.microsoft.com/pricing/free-trial/)。)

* [Python 2.x 或 3.x](https://www.python.org/downloads/)。 請務必使用安裝程式所需的 32 位元或 64 位元安裝。 在安裝期間出現系統提示時，務必將 Python 新增至平台特有的環境變數。 如果您是使用 Python 2.x，可能需要[安裝或升級 pip** (Python 套件管理系統](https://pip.pypa.io/en/stable/installing/))。

* 如果需要，請使用 命令安裝[azure-iot 設備](https://pypi.org/project/azure-iot-device/)包`pip install azure-iot-device`

* 如果需要，請使用 命令安裝[azure-iothub 服務用戶端](https://pypi.org/project/azure-iothub-service-client/)包`pip install azure-iothub-service-client`
