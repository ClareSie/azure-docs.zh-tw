# <a name="azure-maps-routing-coverage"></a>Azure 地圖服務路線規劃涵蓋範圍

本文提供 Azure 地圖服務路線規劃的涵蓋範圍資訊。 搜索查詢時，Azure 地圖返回從位置 A 到位置 B 的最佳路由。您將獲得準確的旅行時間、旅行資訊的即時更新和路線說明。 您還可以添加其他搜索參數，如當前交通、車輛類型和避免的條件。 路線的優化取決於區域。 這是因為，Azure 地圖具有不同區域的不同級別資訊和準確性。 下表列出了區域以及您可以請求哪些類型的資訊。
 
檢查[**地理編碼**](geocoding-coverage.md)的涵蓋範圍。       
檢查[**流量**](traffic-coverage.md)的涵蓋範圍。  
檢查[**轉譯器**](render-coverage.md)的涵蓋範圍。
<br>  
**傳說**

| 符號             | 意義                                |
|--------------------|----------------------------------------|
| ✓                  | 國家/地區提供了詳細的資料。   |
| Ø                  | 國家/地區提供簡化資料。 |
| 遺漏國家/地區 | 未提供國家/地區資料。          |
