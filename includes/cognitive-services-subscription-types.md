您可以通過兩種不同的資源訪問 Azure 認知服務：多服務資源或單服務資源。

* 多服務資源：
    * 使用單個鍵和終結點訪問多個 Azure 認知服務。
    * 從您使用的服務合併計費。
* 單一服務資源：
    * 使用為每個創建的服務使用唯一的金鑰和終結點訪問單個 Azure 認知服務。 
    * 使用免費套餐試用該服務。
