# <a name="view-support-information-lab-user-in-azure-lab-services"></a>檢視支援資訊 (Azure 實驗室服務中的實驗室使用者)
本文將說明您 (身為實驗室使用者) 可以如何檢視下列支援資訊：

- URL
- 電子郵件
- 電話
- 其他指示

如果您在實驗室帳戶中使用實驗室時遇到任何技術問題，您可以使用這項資訊來取得協助。

 
## <a name="view-support-information"></a>檢視支援資訊
1. 登入 [Azure 實驗室服務入口網站](https://labs.azure.com)。
2. 選取您需要協助的**實驗室或虛擬機器**，然後選取 [？] ，其位於頁面的右上角。 
3. 確認其中顯示 [檢視支援網站]、[電子郵件支援] 及 [支援電話號碼] 的連結。

    ![檢視支援資訊](../media/lab-user-support-information/support-information.png)
4. 您可以在下拉式清單中切換至其他實驗室，以檢視另一個實驗室的支援連絡資訊。 

    ![切換到其他實驗室](../media/lab-user-support-information/switch-another-lab.png)
5. 現在，您會看到另一個實驗室的支援連絡資訊。 

    ![其他實驗室的支援資訊](../media/lab-user-support-information/second-lab-support-information.png)
