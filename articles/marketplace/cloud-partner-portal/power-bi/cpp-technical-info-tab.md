# <a name="power-bi-apps-technical-info-tab"></a>Power BI 應用程式 [技術資訊] 索引標籤

在 **"新產品/服務**"頁上，使用 **"技術資訊**"選項卡提供 Power BI 安裝套裝程式 URL 以及驗證新產品/服務所需的其他資訊。  對於初始版本，所有 Power BI 應用都是免費的，可從 AppSource 下載。 因此，您無法為此產品/服務類型定義庫存單位 （SKU）。

![技術資訊選項卡](./media/technical-info-tab.png)


## <a name="technical-info-fields"></a>[技術資訊] 欄位 

在"**技術資訊**"選項卡上，完成下表中描述的欄位。 欄位標籤末尾的星號 （*） 表示需要該欄位。

|        欄位          |  描述                                                                 |
|    ---------------    |  ----------------------------------------------------------------------------|
| **安裝程式 URL\***     | 發佈應用並將其提升為生產時，Power BI 將生成此 URL。  有關詳細資訊，請參閱在[Power BI 中發佈具有儀表板和報表的應用](https://docs.microsoft.com/power-bi/service-create-distribute-apps)。  |
|  **驗證指示**  |  如果需要，請添加說明（最多 3，000 個字元），以説明 Microsoft 驗證團隊配置、連接和測試你的應用。 包括可用於測試"連接資料"選項的典型配置設置、帳戶、參數或其他資訊。 此資訊僅對驗證團隊可見，並且僅用於驗證目的。  |
| **此應用程式是否會建立為 Power BI 內容套件？** | 目前，此欄位僅在內部使用。 保留預設設置 **"否**"。 如果將設置更改為 **"是**"，則可以停止發佈過程。  |  
|  |  |
