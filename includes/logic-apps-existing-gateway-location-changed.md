**問**︰為什麼我現有閘道的位置會變更？ <br/>
**答**：2017 年 5 月 3 日之前所建立的閘道資源已移至 Azure AD 租用戶的原始位置，以供建立這些閘道的工作或學校帳戶使用。 不過，這些變更應該不會影響目前執行的邏輯應用程式，而應該是繼續如往常般運作。 隨著閘道在 5 月公開上市，閘道資源位置可能會與邏輯應用程式的位置有所不同。
