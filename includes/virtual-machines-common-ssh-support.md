## <a name="supported-ssh-key-formats"></a>支援的 SSH 金鑰格式

Azure 目前支援 SSH 通訊協定 2 (SSH-2) RSA 公開/私密金鑰組，最小長度為 2048 位元。 不支援其他金鑰格式，例如 ED25519 和 ECDSA。
