## <a name="reference-docs"></a>參考文件

語音服務提供兩個 SDK。 第一個 SDK 是主要的[語音 SDK，](../speech-sdk.md)它提供了與語音服務交互所需的大部分功能。 第二個 SDK 特定于設備，適當命名為[語音設備 SDK](../speech-devices-sdk.md)。 兩種 SDK 都提供多種語言版本。

### <a name="speech-devices-sdk-reference-docs"></a>語音設備 SDK 參考文檔

[語音設備 SDK](../speech-devices-sdk.md)是語音 SDK 的超集合，具有特定設備的擴展功能。 要下載語音設備 SDK，必須首先[選擇開發工具組](../get-speech-devices-sdk.md#choose-a-development-kit)。
