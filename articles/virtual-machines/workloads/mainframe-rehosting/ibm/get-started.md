# <a name="ibm-workloads-on-azure"></a>Azure 上的 IBM 工作負載

您可以在 Azure 中複寫以 z/OS 為基礎的許多 IBM 大型主機工作負載，且不會遺失任何功能，也不會有使用者注意到其基礎系統中的變更。 Azure 上的重新裝載應用程式提供您所需的大型主機功能，以及雲端的彈性、可用性和潛在的成本節約。

Azure 支援與現有的 IBM 大型主機環境整合，可讓您遷移合理的 applicates，並在需要時執行混合式解決方案，以及隨著時間進行遷移。 雖然您可以完全重寫現有的大型大型主機程式，但更常見的情況是重新裝載它們。 重寫會增加成本、複雜度和時間來遷移專案。 透過重新裝載，您可以：

- 將應用程式移至雲端式模擬器。

- 將資料庫移轉至雲端式資料庫。

- 使用程式碼轉換引擎來取代模組和程式碼。

此外，IBM 軟體（包括 WebSphere 和 MQ）現在已在 Azure Marketplace 中。 使用 IBM 軟體的授權，您可以利用 Azure 所提供的隨選基礎結構調整來快速啟動虛擬機器。

有廣泛的合作夥伴生態系統可協助您將 IBM 大型主機系統移轉至 Azure。 在重寫或取代應用程式的階段式部署登機之前，大多數都是在可行的情況下，使用實用的方法來執行。 在 [Azure 大型主機遷移中心](https://azure.microsoft.com/migration/mainframe/)取得更多有關合作夥伴的指引和協助。
