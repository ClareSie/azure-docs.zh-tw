# <a name="quickstart-create-an-azure-cache-for-redis-instance"></a>快速入門：建立 Azure Cache for Redis 執行個體

Azure Cache for Redis 會在 Azure 內提供完全受控的[開放原始碼 Redis](https://redis.io/)。 您可以從任何階層 (基本、標準或進階) 和大小的 Azure Cache for Redis 執行個體開始著手，並加以擴縮以符合應用程式的效能需求。 本快速入門會示範如何使用 Azure 入口網站來建立新的 Azure Cache for Redis。

## <a name="prerequisites"></a>Prerequisites

在開始之前，您需要有 Azure 訂用帳戶。 如果您沒有訂用帳戶，請先建立[免費帳戶](https://azure.microsoft.com/free/)。
