# <a name="view-support-information-lab-user-in-azure-lab-services"></a>查看支援資訊（Azure 實驗室服務中的實驗室使用者）
本文說明您（身為實驗室使用者）如何能夠查看下列支援資訊：

- URL
- 電子郵件
- 電話
- 其他指示

當您在實驗室帳戶中使用實驗室時，您可以使用這項資訊來取得任何技術問題的協助。

 
## <a name="view-support-information"></a>檢視支援資訊
1. 登入[Azure 實驗室服務入口網站](https://labs.azure.com)。
2. 選取您需要協助的**實驗室或虛擬機器**，然後選取 [ **？** ] ，位於頁面的右上角。 
3. 確認您看到**視圖支援網站**、**電子郵件支援**和**支援電話號碼**的連結。

    ![檢視支援資訊](../media/lab-user-support-information/support-information.png)
4. 您可以在下拉式清單中切換至該實驗室，以查看另一個實驗室的支援連絡人資訊。 

    ![切換至另一個實驗室](../media/lab-user-support-information/switch-another-lab.png)
5. 現在，您會看到另一個實驗室的支援連絡人資訊。 

    ![其他實驗室的支援資訊](../media/lab-user-support-information/second-lab-support-information.png)
