# <a name="download-notebookspreview-data"></a>下載筆記本（預覽）資料

筆記本（預覽）功能將于*2020年4月13日*刪除。 該日期之後，"筆記本（預覽）"選項卡將消失，無法還原筆記本資料。 在 2020 年 4 月 13 日之前下載筆記本資料。

本文提供有關如何下載筆記本（預覽）資料的分步說明。

1. 轉到 Azure 機器學習工作室中的 **"筆記本"** 選項卡（經典）。
    ![導航到筆記本選項卡！](./media/download-notebooks/notebooks-list.png)

1. 選擇要下載的筆記本。 它將在朱皮特開張。
    ![不支援選擇一](./media/download-notebooks/select-notebook.png)次下載多個筆記本的筆記本。

1. 轉到 **"檔** -> **下載"作為**，然後選擇格式選項。
    ![筆記本下載選項](./media/download-notebooks/download-options.PNG)

1. 選擇選項後，筆記本檔將開始下載。 忽略出現的任何 500 或 502 錯誤，這些錯誤不會影響您的下載。
