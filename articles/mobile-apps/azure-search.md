# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>AI 支援的雲服務，具有 Azure 認知搜索功能
[Azure 認知搜索](https://azure.microsoft.com/services/search/)是一種搜索即服務雲解決方案，它為開發人員提供 API 和工具，用於在 Web、移動和企業應用程式中添加豐富的搜索體驗，而不是私有的異構內容。 您的自訂程式碼會叫用資料擷取 (索引編制) 來建立和載入索引。 另一方面，應用程式的程式碼會發出查詢要求和處理回應。 搜索體驗通過使用 Azure 認知搜索的功能在用戶端中定義，在服務中創建、擁有和存儲的持久化索引上執行查詢。

## <a name="azure-cognitive-search-features"></a>Azure 認知搜索功能
- **自由格式文本搜索**：提供全文檢索搜尋、簡單的查詢語法和 Lucene 查詢語法。
- **簡單的評分設定檔**：模型相關性作為文檔中值的函數。
- **地理搜索**：根據搜尋結果與物理位置的接近程度來流覽資料。
- **使用者體驗功能**：提供自動完成、搜索建議、排序和分頁。
- **認知搜索**：可應用於索引管道，從原始內容中提取文本資訊，用於圖像和文本分析。
- **知識存儲**：保存索引期間創建的擴充。
- **資料來源**：連接到 Azure SQL 資料庫、Azure Cosmos DB 和 Azure Blob 存儲時，在主資料存儲中提取可搜索的內容。
- **導入資料嚮導**：配置索引子。 
- **索引設計器**：站起來索引。 
- **搜索資源管理器**：測試查詢並優化評分設定檔。

在 Azure 認知搜索[功能](/azure/search/search-what-is-azure-search#feature-descriptions)上查找詳細說明。
