# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Azure 上具有 Azure 內容交付網路的內容傳遞網路
[Azure 內容交付網路](https://azure.microsoft.com/services/cdn/)為開發人員提供了一個全球解決方案，通過將視頻、圖像或靜態檔等高頻寬內容緩存到全球數百個存在點，從而快速向使用者交付其內容。 內容交付網路還可以加速無法緩存的動態內容，以縮短載入時間、節省頻寬並提高回應能力。 無論您是開發或管理網站或移動應用程式，還是對流媒體、遊戲軟體、固件更新或 IoT 終結點進行編碼和分發，您都將獲得這些優勢。

## <a name="azure-content-delivery-network-features"></a>Azure 內容交付網路功能
  - 與 Azure 服務（如 Web 應用程式、媒體服務、存儲和雲服務）無縫協作。
  - 輕鬆處理交通高峰和重負載。
  - 與 Akamai 和 Verizon 集成，構建和定制內容交付戰略。
  - 使用動態網站加速。
  - 使用緩存規則控制內容傳遞網路緩存行為。
  - 獲取 HTTPS 自訂域支援。
  - 查看 Azure 診斷日誌以查看核心分析並將其匯出到 Azure Blob 存儲、Azure 事件中心或 Azure 監視器。
  - 通過壓縮檔來提高性能。
  - 使用地理篩選按國家或地區限制內容交付網路內容。
