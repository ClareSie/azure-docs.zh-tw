# <a name="monitor-your-azure-data-box-gateway"></a>監視您的 Azure Data Box Gateway

本文描述如何監視您的 Azure Data Box Gateway。 若要監視您的裝置，您可以使用 Azure 入口網站或本機 Web UI。 使用 Azure 入口網站來檢視裝置事件、設定及管理警示和檢視計量。

在本文中，您將學會如何：

> [!div class="checklist"]
> * 檢視裝置事件和對應的警示
> * 檢視您裝置的容量和交易計量
> * 設定和管理警示
