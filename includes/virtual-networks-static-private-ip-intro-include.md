虛擬機器（VM）會根據 VM 部署所在的子網，自動從您指定的範圍指派私人 IP 位址。 VM 會保留位址，直到刪除 VM 為止。 Azure 會從您建立 VM 的子網動態指派下一個可用的私人 IP 位址。 如果您想要從指派給 VM 的子網中指定特定的 IP 位址，請指派靜態 IP 位址。
