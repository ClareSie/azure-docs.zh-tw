# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>支援 Apache Spark 和從屬元件的語言和執行時版本  

Azure 突觸分析服務中的 Apache Spark 支援幾個不同的運行時間,本文檔列出了這些版本。

## <a name="spark-version"></a>火花版本

Spark 2.4 是當前支援的版本。

## <a name="operating-system-version"></a>作業系統版本

Azure Synapse 中的 Apache Spark 運行在 Ubuntu 版本 16.04 上。

## <a name="java-version"></a>Java 版本

Azure 突觸中的 Apache Spark 包括 JAva 版本 1.8.0_222

## <a name="scala-version"></a>Scala 版本

Azure 突觸中的阿帕奇火花包括 Scala 2.11.12

## <a name="net-core-version"></a>.NET Core 版本

Azure 突觸中的 Apache Spark 包括 .NET 核心 3.0

## <a name="net-support-for-apache-spark"></a>.NET 支援阿帕奇火花

Azure 突觸包括 .NET 對 Apache Spark 0.5.0 的支援

## <a name="delta-lake"></a>Delta Lake

突觸分析包括對三角洲湖 0.5.0 的支援

## <a name="python-version"></a>Python 版本

Azure 突觸中的 Apache Spark 包括安裝 Python 版本 3.6.1
