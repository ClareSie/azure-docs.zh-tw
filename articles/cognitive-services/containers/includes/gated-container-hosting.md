> [!NOTE]
> 自2020年9月22日起，大部分的閘道預覽容器都裝載于 Microsoft Container Registry 上，因此不需要使用 docker login 命令就能下載。 您仍然需要完成線上要求以執行容器。 如需詳細資訊，請參閱本文稍後的 **要求核准以執行容器** 一節。
