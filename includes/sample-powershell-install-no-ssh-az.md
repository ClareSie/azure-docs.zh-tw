如果您需要安裝或升級，請參閱[安裝 Azure PowerShell 模組](/powershell/azure/install-az-ps)。 

執行 [Connect-AzAccount](/powershell/module/az.accounts/connect-azaccount) Cmdlet 以連線到 Azure。
