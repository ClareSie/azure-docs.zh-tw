# <a name="azure-maps-community---open-source-projects"></a>Azure 地圖社區 - 開源專案

這些開源、社區驅動的計畫由 Azure 地圖團隊創建和維護。 它們不是標準產品或服務的一部分。

下面列出了一些最流行的 Azure 映射開源專案和示例。

| 專案名稱 | 描述 |
|-|-|
| [Azure 映射代碼示例](https://github.com/Azure-Samples/AzureMapsCodeSamples) | 用於在基於 Web 的應用中使用 Azure 映射的代碼示例的集合。 |
| [Azure 映射 Gov 雲代碼示例](https://github.com/Azure-Samples/AzureMapsCodeSamples) | 用於通過 Azure 政府雲使用 Azure 映射的代碼示例的集合。 |
| [Azure 地圖& Azure 活動目錄示例](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | 演示如何將 Azure 活動目錄與 Azure 映射一起使用的示例集合。 | 
| [使用 Azure 地圖服務執行 IoT 空間分析](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | 追蹤和擷取在空間和時間中發生的相關事件，是常見的 IoT 案例。 |
| [Azure 地圖猶太筆記本示例](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | 使用 Azure 映射 REST 服務的 python 示例集合。 |
| [機器人框架 - 興趣點技能](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | 興趣點技能為使用 Azure 地圖與 Azure 機器人服務和機器人框架使用 Azure 映射的虛擬助理提供與 POI 相關的功能。 |

此處[提供了包含](https://github.com/microsoft/Maps/blob/master/AzureMaps.md)社區創建開源專案的較長清單

## <a name="supportability-of-open-source-projects"></a>開放源碼專案的支援性

以下語句適用于創建和維護的所有 Azure 地圖開源專案和示例：

- Azure 地圖開源專案和示例由 Microsoft 和社區創建。
- Azure 地圖開源專案和示例由 Microsoft 和社區維護。
- Azure 映射開源專案和示例使用支援和推薦的技術。
- Azure 地圖開源專案和示例是一個社區計畫 - 為他人謀福利的主動工作，並且也有正常工作的人員。
- Azure 地圖開源專案和示例不是產品，因此它不受高級支援或其他官方支援管道的支援。
- Azure 地圖開源專案和示例的支援方式與 Microsoft 在社區支援下完成的其他開源專案類似。
