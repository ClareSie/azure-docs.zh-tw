## <a name="vmware"></a>VMware

![VMware 徽標](./media/partners/vmware.png)

Microsoft Azure 上的 VMware Horizon 雲使組織能夠將自己的 Microsoft Azure 實例連接到簡單、直觀的 Horizon 雲控制平面，從而創建安全、全面的雲託管解決方案，以交付虛擬化Windows 應用程式和桌面。 隨著 Windows 虛擬桌面的發佈，VMware 與 Microsoft 合作，在 Microsoft Azure 上使用 Horizon Cloud 將 Windows 虛擬桌面的功能擴展到客戶。 此功能將于 2019 年底在技術預覽版中提供。
 
作為經批准的供應商，Windows 虛擬桌面的優勢將擴展到 Microsoft Azure 客戶的 VMware Horizon 雲，包括多會話的 Windows 10 企業版、具有最多三年免費擴展安全更新的 Windows 7 以及FSLogix。 此外，客戶還將獲得 Horizon Cloud 的企業級功能，例如廣泛的端點支援;全球、雲優化的仲介;靈活的桌面選項和配置;使用者-環境管理;和支援混合環境。

- [註冊在微軟 Azure 上註冊地平線雲的技術預覽](https://www.vmware.com/learn/323526_TECH_PREVIEW_REG.html)。
- [轉到合作夥伴網站](https://www.vmware.com/products/horizon-cloud-virtual-desktops.html)。
