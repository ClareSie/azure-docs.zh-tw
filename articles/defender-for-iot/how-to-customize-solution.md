# <a name="customize-your-defender-for-iot-solution"></a>自訂您的 Defender for IoT 解決方案

在本指南中，您將瞭解如何在 Defender 中自訂 IoT 的不同設定。

> [!div class="checklist"]
> * 設定解決方案建議
> * 變更設定

## <a name="configure-solution-recommendations"></a>設定解決方案建議

若要設定 Defender 的 IoT 解決方案建議，請執行下列動作：

1. 在 Azure 入口網站中開啟您的 **IoT 中樞**。
1. 在左側功能表中，選取並開啟 [**安全性**] 底下的 [**設定**]。
1. 在 [設定] 底下，選取 [**建議**的**設定**]。
1. 停用/啟用與您的組織和工作流程相關的解決方案建議。

## <a name="change-settings"></a>變更設定

管理您的 Defender for IoT 設定：

若要設定 Defender 的 IoT 設定，請執行下列動作：

1. 在 Azure 入口網站中開啟您的 **IoT 中樞**。
1. 從左側功能表的 [ **安全性**] 底下，選取並開啟 [ **設定**]。
1. 在 [資料收集] < * * * * 下，選取您要變更的方案設定。
1. 請記得一律按一下任何設定畫面頂端的 [ **儲存** ]，以儲存您的設定變更。
