您可以透過兩個不同的資源存取 Azure 認知服務：多服務資源或單一服務資源。

* 多服務資源：
    * 使用單一金鑰和端點存取多個 Azure 認知服務。
    * 合併您所使用服務的帳單。
* 單一服務資源：
    * 針對每個建立的服務，使用唯一的金鑰和端點來存取單一 Azure 認知服務。 
    * 使用免費層來試用服務。
