# <a name="micro-focus-devtest-environments-on-azure"></a>Azure 上的微焦點開發/測試環境

以開發和測試環境開始使用 Azure 是提升現代化成果的絕佳方式。 依賴傳統系統（如 IBM CICS （客戶資訊控制系統）和 IMS （資訊管理系統））的大型主機應用程式，可以使用合作夥伴解決方案在 Azure 上重新裝載。

微焦點已與 Microsoft 合作，提供吸引人的大型主機遷移和重新裝載工具。 您可以跨開發、測試和生產環境，在 Azure 和 Microsoft Windows 平臺上執行大型主機應用程式。

微焦點提供開發重新裝載和測試重新裝載解決方案，將現有的 COBOL 和 PL/I 商務應用程式擴充到 Azure。

> [!NOTE]
> 即將推出：在 Azure Vm 上設定[微焦點企業伺服器 5.0](https://techcommunity.microsoft.com/t5/azurecat/micro-focus-enterprise-server-5-0-quick-start-template-on-azure/ba-p/1160110)的指示。
