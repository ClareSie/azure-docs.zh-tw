Microsoft 365 原本就設計為可透過網際網路來安全可靠地存取。 因此，在特定案例中，建議使用 ExpressRoute。 如需使用 ExpressRoute 來存取 Microsoft 365 的相關資訊，請造訪 [Azure ExpressRoute for Microsoft 365](/microsoft-365/enterprise/azure-expressroute)。
