# <a name="show-real-time-traffic"></a>顯示即時流量

「流量層」功能會在地圖上重迭即時流量資料。 若要啟用這項功能，請將 [**格式**] 窗格中的 [**流量層**] 滑杆移至 [**開啟**] 位置。 這會將流量資料與色彩編碼的道路重迭。

> [!div class="mx-imgBorder"]
> ![顯示即時流量資料的地圖](media/power-bi-visual/traffic-layer.png)

[**流量層**] 區段中提供下列設定。

| 設定         | 描述    |
|-----------------|----------------|
| 顯示事件  | 指定是否應該在地圖上顯示交通事件（例如道路和建築）。 |
| 流量控制 | 將按鈕加入至地圖，讓報表讀取器可以開啟或關閉流量層。  |
