此實體類別包含財務資訊和官方形式的識別。 從模型版本開始提供 `2019-10-01` 。 子類型如下所示。 

### <a name="financial-account-identification"></a>財務帳戶識別

| 子類型名稱               | 描述                                                                |
|----------------------------|----------------------------------------------------------------------------|
| ABA 銀行代號        | 美國銀行家協會 (ABA) 傳輸路線號碼。                  |
| SWIFT 代碼                 | 付款指示資訊的 SWIFT 代碼。                           |
| 信用卡                | 信用卡號碼。                                                       |
| 國際銀行帳戶號碼 (IBAN)                  | 付款指示資訊的 IBAN 代碼。                            |
