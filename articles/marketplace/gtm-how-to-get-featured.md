# <a name="how-to-get-featured-in-appsource-and-azure-marketplace"></a>如何成為 AppSource 和 Azure Marketplace 中的精選項目

Azure Marketplace 和 AppSource 都有**精選應用程式**區段，您可以讓您的應用程式成為其中的精選應用程式：

* 首先，如果你有一個TRIAL或交易優惠，你可以使用你的"類別促銷"的好處，通過[商業市場的好處](https://docs.microsoft.com/azure/marketplace/gtm-your-marketplace-benefits)。
* 第二，檢閱下方最佳做法和準則清單，以爭取曝光度。

特色應用選擇演算法會為 Microsoft 的應用生成分數，就像一個人的信用評分在美國一樣。  每週精選應用程式會根據應用程式和服務效能的分數計算來選取。

## <a name="steps-to-take"></a>需採取的步驟

您可以採取下列動作項目來提升您的分數：

1. *確保應用或服務已正確分類*：選擇表示應用或服務功能的三個類別。
2. Azure Marketplace 應用程式：每月增加您的 Azure 耗用量。** 如果您能夠達到每月 1,000 小時的 Azure 使用量，您將可大幅增加獲選為精選應用程式的機會。
3. AppSource 應用程式：增加取得您供應項目的次數。** 如果您能夠達到每月 10 次的取得次數，您將可大幅增加獲選為精選應用程式的機會。 
4. 達到可共同銷售 (Co-Sell Ready) 狀態**：完成可共同銷售的需求。
5. 改善您的供應項目品質**：請參閱[內容清單指導方針](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation)，取得有關如何修改您供應項目的資訊
6. 在 Marketplace 中發佈多個供應項目：您的所有核心應用程式和服務都已列出嗎？ 您有提供試用版體驗嗎？
7. 鼓勵您的客戶撰寫評論。

精選應用程式的推廣會與搜尋演算法分開運作。

>[!Note]
>如果您的解決方案並未正確地顯示在搜尋結果中，請透過 [Cloud Partner 入口網站](https://cloudpartner.azure.com/)的 [說明] 功能表建立支援票證。

GTM 支援也包括完整的自助服務範本程式庫、Web 內容、訓練和工具，可協助您進一步推廣您的清單與業務。
