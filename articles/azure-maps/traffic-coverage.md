# <a name="azure-maps-traffic-coverage"></a>Azure 地圖服務交通涵蓋範圍

Azure 地圖服務會以交通**流量**和**事件**的形式提供豐富的交通資訊。 此資料可以在地圖上以視覺形式呈現，或用來產生將行車條件納入考量的更聰明路線。

不過，Maps 沒有相同層級的資訊和精確度適用于所有國家或地區。 下表提供您可以從每個國家或地區要求何種流量資訊的相關資訊：
