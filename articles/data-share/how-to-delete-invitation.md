# <a name="how-to-delete-an-invitation-to-a-recipient-in-azure-data-share"></a>如何在 Azure Data Share 中刪除收件者的邀請

本文說明如何在收件者接受邀請之前先將其刪除。 這會讓他們無法建立共用訂用帳戶，而且永遠不會收到任何共用資料。 如果您刪除已建立共用訂用帳戶之取用者的邀請，刪除其邀請不會有任何影響。

## <a name="navigate-to-a-sent-data-share"></a>流覽至已傳送的資料共用

在 Azure Data Share 中，流覽至您已傳送的共用，然後選取 [ **邀請** ] 索引標籤。以下列出此資料共用的所有邀請收件者。 按一下您想要刪除其邀請的收件者旁邊的方塊，然後按一下 [ **刪除**]。
