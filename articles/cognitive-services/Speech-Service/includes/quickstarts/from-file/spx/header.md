在本快速入門中，您可以從命令列使用語音 CLI 來辨識音訊檔案中錄製的語音，並產生文字轉譯。 您可以輕鬆地使用語音 CLI 來執行常見的辨識工作，例如轉譯對話。 進行一次性設定之後，語音 CLI 可讓您使用麥克風或從檔案以批次指令碼，透過互動方式將音訊轉譯成文字。
